Shell 数组

bash支持一维数组（不支持多维数组），并且没有限定数组的大小，初始化时不需要定义数组大小。
数组元素的下标

  • 由 0 开始编号，应大于或等于0
  • 可以不使用连续的下标
  • 下标的范围没有限制
  • 获取数组中的元素要利用下标
