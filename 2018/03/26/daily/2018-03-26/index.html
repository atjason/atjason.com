0326 - 寺庙与互联网+

今天跟朋友聊天吃饭，说到一个有意思的话题：寺庙与互联网+

寺庙其实有很多痛点，举个例子：功德箱，也就是烧香时往里扔钱的那个箱子。一个很头大的问题：有人偷。比如，拿钩子进去钩、拿胶水进去粘、甚至直接把箱子搬走…

怎么办呢？最基础的，就是加装监控，但其实效果不佳。也有寺庙直接粘支付宝二维码（好吧…），但好像又少了扔钱（好吧，我总感觉是扔钱，不是捐钱）的仪式感。

怎么既解决收钱方便、不怕被偷，又不影响扔钱人的心情呢？这其实是个问题。具体的方案我就不展开说了，大家可以自己发挥。

这，仅仅是寺庙的一个痛点，其他还有很多（好吧，说实话，在听朋友介绍之前，我都不知道…）

另外，从信徒的角度，也有不少需求。比如，想求子，哪家寺庙灵验？（感觉很像想看病，哪家医院、哪位医生医术高明）

可见，无论是 B 端（寺庙）还是 C 端（信徒），这里面都有很多可做的事。尤其，这是个小众、未被发掘的蓝海，很值得玩味。
