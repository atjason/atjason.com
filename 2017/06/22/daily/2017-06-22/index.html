0622 - 如何坚守自己的价值观？

如果有人有着和自己迥异的价值观，且混得很好，且和自己是熟人，自己是不是要改变自己、向其学习？

  • 比如，常说的「学习好的比学习差的打工」，那到底是要好好学习，还是提前混人脉、攒经验？
  • 比如，如果有人贷款炒房，一个月赚的比自己一年都要，自己要不要也去炒？

如果自己遇到困难挫折，要不要从根本上调整方向？

  • 比如，自己辛苦做了产品、写了文章，却很少有人用、有人看，试了一段时间还是如何，还要继续吗？
  • 比如，有个强大的竞争者出现，自己是要迎难而上、还是尽快转向？

…

前方，总是会有各种诱惑、阻碍，该如何坚守自己？

我也不知道。

有点像是招聘和管理的关系：

  • 与其费尽心机管一个人，不如花精力招个不用管的人；
  • 与其想着如何抵御诱惑，不如想清楚自己到底要什么；
  • 到底要不要放弃？问问自己还想不想要。

向内寻找向外的力量。
