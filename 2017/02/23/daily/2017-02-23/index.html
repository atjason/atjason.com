每天写一点

0223 - 初为嘉宾

今天接受少数派的邀请，参加了 Matrix 群讨论，初为线上嘉宾。

主要讨论的是：大家是如何阅读、如何做笔记、如何看待做了笔记而不会回看、如何持久存储笔记、等等问题。

我觉得，在如今这个快消费时代，还有这么一群人能静下心来来，认真的看书，还做笔记，真是不容易。非常愿意跟这样的人交流。

在跟大家的聊天中，也了解到一些大家实际使用 Kindle 的方法、场景，对改进 Klib 提供了更多背景支持，谢谢大家的分享。

最大的遗憾就是，时间的限制，没能跟大家深入交流，深觉可惜。

今天的遗憾还有两处：

  • 没写代码
  • 没看完《从零开始做运营》
