0531 - 两眼论

围棋中有「两眼即活」理论，可以简单理解为：多一条路，就不会死。

对应到自己的生活中，发现可以找到很多映射：

领域AB
操作系统macOSWindows
编程语言SwiftPython
Python 虚拟环境pyenvvirtualenv
手机iPhone锤子
服务器阿里云Vultr
CDN七牛CloudFlare
通讯微信Telegram
………

换个角度讲，不在一颗树上吊死。多了解一个领域、多一个选择、多一种跨界，就多一种活下去的方式，就更不容易死掉。
