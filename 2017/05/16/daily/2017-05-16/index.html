0516 - Klib 玩跨界？

作为全新的 Kindle 标注、笔记管理工具，Klib 已不满足于 Kindle、玩起了跨界。快瞧瞧都玩出什么花了！

从 iBooks 中导入标注、笔记

作为 macOS 的原生应用，Klib 不支持 导入 iBooks 中的标注、笔记，有些说不过去。

使用 macOS Services 导入

聪明的你一定发现，这是书摘收集利器：在阅读时遇到心仪的句子，只需选择、按下「Command + K」快捷键，就立即收集到 Klib 中啦。

使用系统 Share 分享笔记

也就是说，有了读书笔记，你可以方便地发微博、Twitter，以及通过各种方式分享给朋友。

标记为章节

章节信息对于回顾整本书的笔记很有帮助，可以 Kindle 导出的笔记中并不包含，怎么办呢？一个小技巧供大家参考：

  • 阅读时在章节文字上添加标注
  • 导入 Klib 后，选择全部章节对应的标注，将其「标记为章节」
  • 在 Klib 中复制为 Markdown 时，自动为章节添加二级标题
