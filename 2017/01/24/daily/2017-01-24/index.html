0124 - iKindle: 用着很爽的 Kindle 笔记管理 App

iKindle 是什么？

iKindle 是管理 Kindle 中标注、笔记的工具，macOS 平台。

俗话说，「不动笔墨不读书」；在 Kindle 上看书时，也会积累很多笔记。如果任由这些笔记流失、或蒙灰，实在是太浪费精华了。怎么办呢？试试 iKindle 吧。

管理，从导入开始

将你的 Kindle 连接到 Mac 电脑上，选择 Kindle 目录，「开始导入」即可一键导入。

原生交互，未打开、已熟练

考虑了好几版交互，最后参考了系统原生应用「提醒事项」。非常契合 Kindle 笔记管理，也方便大家快速上手、避免学习新交互所浪费的时间。

体验在细节

仅仅实现功能，怎能让人满意？更贴心的在细节。

专注于当前正在阅读的书籍

爱读书的朋友，已读列表会越来越长。如果将所有书混在一个列表中，会失去专注。

将已读书籍放在专门的列表中、并折叠隐藏，一个小小的细节，重新找回注意力。

一条条回顾笔记

按下空格键，即可以快速查看当前笔记，给你熟悉的感觉。

一条条地回顾笔记，帮你深刻掌握读过的书籍。

以 Markdown 格式复制笔记

好东西要分享；以 Markdown 格式复制笔记，之后可以方便输出至博客等文章中。

其实，开发 iKindle 的动机，就是我在读完一本书后，会在博客中写读后感，而在 Kindle 中的标注则是其中重要的部分。于是，就需要一个工具，快速以 Markdown 的格式将笔记复制出来；这就是 iKindle 的由来。

全屏，带来酷爽体验

这感觉难以描述。

尤其是当你有很多书、书中有很多笔记时，全屏下查看笔记的舒爽，只待你自己体验。

键盘侠

效率工具，自然处处体验效率：

  • 空格键 快速预览
  • Command + I 查看简介
  • Tab 在书和笔记列表间切换
  • 方向键 查看相邻的预览或简介
  • J K 方向键，vimer，你懂的
