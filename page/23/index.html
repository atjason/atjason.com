0330 - 传统经验的失效

发表于 2021-03-30 | 分类于 每天写一点

什么鸡蛋是好的鸡蛋？我妈的标准之一，是蛋黄比较黄。

事实上，现代的农业，已经工业化了。

  • 你想要蛋黄比较黄？行，添加饲料 A
  • 你想要蛋黄比较红？行，添加饲料 B

在这样的情况下，传统的经验已经失效。而在新经验建立之前，事情就处于无序、混乱中。人类无法忍受混乱，也无法应对混乱，一定会慢慢的建立出新的标准。

动物也是类似的。比如，千万年来，狗在拉屎尿尿后，会用后瓜刨土，把屎尿盖住。现在，狗被养在了城市里，到处都是水泥柏油地面。可是，狗的习惯并没有改变，还在尝试刨坚硬的地面。它的后面，如果会思考的话，会想：爸妈怎么这么傻，这么硬的地面，刨它干什么？没办法，这种被写在基因里的东西，需要很多代才能改变。

0329 - 人到老年

发表于 2021-03-29 | 分类于 每天写一点

送娃去兴趣班，在外面等。边上一起等的，是位大爷。看样子，是从农村过来，帮儿子带孙子的。

大爷智能手机不太会用，边上的书也没兴趣，其他人个个低头玩手机，连个说话的人都没有；坐在那里，百无聊赖。

突然想，他活着的意义是什么呢？小区里还认识不少这样的老人，他们所有的时间，都在照顾小孩和家庭，完全没有自己的兴趣爱好。可以说，他们完全是在为别人而活。他们很高尚，也有点可悲。

不由得想到自己。等我老了，会不会也像他一样？会不会像现在的老人一样，不会用智能手机，站在路边也打不到车？

老而不死是为贼。从纯粹动物的角度，当个体没有活力之后，最好的办法，是将资源让渡给其他人。就像大象老了，会悄悄离开群体，独自前往大象墓地；使出最后的力量，用鼻子为自己清理出一块地方、刨一个坑，静卧在里面；再用土将自己埋上，慢慢地死去。可是，人毕竟不是动物，不可能实行社会达尔文主义。老人这个群体，是人类永远的软肋和包袱。

0328 - 我所拥有的，正在杀死我自己

发表于 2021-03-28 | 分类于 每天写一点

今天看到这个观点，感触很深。

举个别人的例子。比如，一个体制内的公务员，对自己的工作既满意、又不满意。满意的，自然是工作清闲、压力不大；不满意的，是晋升慢、收入不高。可是，对于换工作，却很犹豫。生怕万一新工作不好，自己又错失了现在的铁饭碗。

你看，阻止他做出改变的，正是他现在拥有的。哪怕他对自己拥有的，并不满意。

对应到自己身上，也是类似的。我现在能安心做交易，也是靠着之前开发的应用，其收入能勉强覆盖现在的生活费。问题是，这所谓的依靠，恰恰成了我不够努力的理由。相反，如果没有这份收入，相信我会破釜沉舟，更加努力。

要远行，就要轻装上阵、卸掉包袱。而包袱，既包括不好的、也包括好的。

0327 - 读史书

发表于 2021-03-27 | 分类于 每天写一点

最近接连读了两本史书：《一读就上瘾的中国史》和《大唐兴亡三百年》

合在一起看，还是有很多的不同的。

《一读就上瘾的中国史》这本书，先选出一些角度，然后从历史中，找出符合这些角度的人物和事件，进行串讲。

《大唐兴亡三百年》，则相对传统的，按照年份和皇帝，将唐朝历史按顺序讲一遍。讲的内容，更多的是政治、军事这些上层的东西。

不管是哪本书，其实讲的都是历史上有名有姓的人。而更多的普通人，则完全淹没在历史长河中。不客气的说，像你我这样的人，一辈子的时间，也只是不断的改变一些物质的形态而已（吃进去，拉出来），是不会留下名字或者只言片语的。

我读史书，很多时候，就像看抖音一样，是种消遣。不过，也多了一些可能没用的冷知识。比如，「曌」这个字的由来，读作「日月当空照」的 zhao，是武则天从民间寻得的一个当时并不存在的字。比如，武则天为了最高权力，先后杀了自己的丈夫、女儿、大约四个儿子（有些有争议，大概是这么回事）。
