0713 - 厉害不等于必胜

发表于 2023-07-13 | 分类于 每天写一点

狮子大多数的捕食行动都以失败告终。即便是草原之王，失败了也要饿肚子。

我们总有一个错觉：厉害的人做什么都能成功。其实不然。

更何况，我们觉得有些人厉害，可能是他们做成了某些事。而这件事中，可能运气和时运的比重比较大，个人能力可能并不是决定性因素。

0712 - 系统的确定性

发表于 2023-07-12 | 分类于 每天写一点

在不改变系统的情况下，单次结果改变，基本不会影响大局。

比如，一个穷人中大奖，再过一段时间，基本会回到原来的水平。

比如，一个不懂交易的人，在一次交易赚了大钱，重复很多次交易之后，这钱大概率会吐回去，甚至倒亏更多。

反过来说，后悔当初如果如何如何，我现在就会如何如何。大概率即使当初如何如何，现在该啥样还是啥样。看似当初只差一步，实际差得很远。

当然，不排除有人抓住机会，上了一个台阶，不再下来。要让自己成为这样的人、有能力抓住机会；或者说，让自己配得上这种机会。

0709 - Air001 发布

发表于 2023-07-09 | 分类于 每天写一点

Air001 芯片发布，入门硬件多了个选择，开心。

考虑到 Air001 支持 Arduino 开发，再次开始学习 Arduino。

了解到的一些点：

  • 支持 micropython 的方式，是提前在板子上安装 micropython 的运行环境，然后把脚本上传上去。考虑到 Air001 弱的性能，以为官方是否愿意适配，此路不通。
  • 支持 javascript 的方式，只是把板子当时瘦客户端，始终和电脑保持连接，根据电脑发出的指令进行响应。

Arduino 本身基于 C++，框架很简单的。学习路线，就是交叉看 examples and references，按接口及功能进行学习。

  • https://www.arduino.cc/reference/en/
  • https://docs.arduino.cc/built-in-examples/
  • https://www.arduino.cc/reference/en/libraries/

Arduino 的开源库不多且乱，很难选择。并且，很多疏于维护，已经半死不活。

同样的，npm 管理的库也是又多又乱，但至少有个下载统计，可以作为判断是否好坏不的一个标准。
