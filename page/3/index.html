1028 - 一人一扇窗

发表于 2021-10-28 | 分类于 每天写一点

每个人都是社会的节点。多和一个人连接，就连续了一片节点，多开一扇观察世界的窗户。

这样的连接，最好的自然是线下。尤其，如果有多年、长期的关系为基础，这样的连续更紧密。

现实的情况，大家都很忙，又受限于各式各样的原因，天南海北的人要见上一面，有时真不容易。而要用这样的方式认识一个新朋友，成本则更高。于是，网络成了很好的替代。

纪录片也是很好的一种形式。通过拍摄、采访得到素材，再经过一些剪辑、加工，辅以自己的理解，就形成一个很好的了解新事物的方式。纪录片不仅是自然地理，也有很多人文社会的东西。这样的纪录片对于避免闭塞，是非常有帮助的。比如，我最近在 B 站看的《伟大的制造》，推荐大家去瞧瞧。

1027 - 等机会与找机会

发表于 2021-10-27 | 分类于 每天写一点

比如，已经有一套交易模式，怎么执行呢？

一种方式，就是先找优秀的公司，然后等这些公司的股价走出自己认识的模式再下手。这是等机会。

另一种，是直接全市场套模式。符合模式的，再去研究行业基本面什么的，觉得差不多就上车。这是找机会。

以前我是找机会，因为觉得效率更高。时间长了，也遇到一些问题，比如这个过程人会长时间处于紧张状态，过滤做减法比较累，零散的研究行业企业不够深入。

现在，我想试试等机会。先深入研究几个好的行业，找出其中优秀标的，然后耐心等。这样操作，人应该会比较淡定，有耐心也更有信心。看似慢、效率低，但拉长时间看，可能会越做越扎实。

1026 - 费曼学习法

发表于 2021-10-26 | 分类于 每天写一点

运用费曼技巧，只需花上20分钟就能深入理解知识点；而且记忆深刻，难以遗忘。

费曼学习法可以简化为四个单词：Concept （概念）、Teach （教给别人）、Review （评价）、Simplify （简化）。

该技巧主要包含四步：

第一步：把它教给一个小孩子

想象一下，如果用最简单明了的方法，把它介绍给一个 8 岁的孩子，他的词汇量和注意力刚好能够理解基本概念和关系。

这样便迫使自己在更深层次上理解了该概念，并简化了观点之间的关系和联系。

第二步：回顾

在第一步中，你不可避免地会卡壳，忘记重要的点，不能解释，或者说不能将重要的概念联系起来。

这一反馈相当宝贵，因为你已经发现了自己知识的边缘；懂得自己能力的界限也是一种能力。

这是学习开始的地方。现在你知道自己在哪里卡住了，那么就回到原始材料，重新学习，直到你可以用基本的术语解释这一概念。

第三步：将语言条理化，简化

将不确定的点弄清楚的这一过程后，重新用简单的语言介绍一遍。

第四步（可选）：传授

检测知识最终的途径，是你能有能力把它传播给另一个人。如果你能找到这个人，试一试。

1025 - 像孩子一样不记仇

发表于 2021-10-25 | 分类于 每天写一点

孩子有个很明显的特点：不记仇。

比如，一起玩的小孩，可能因为玩具而吵闹，甚至会动手、说类似「再也不跟你玩了」之类的话。可事实上，可能要不了几分钟，就又开心的玩在了一起。搞得上来劝架、甚至因此起冲突的家长，一阵皇帝不急太监急的尴尬。

交易中也要这样，不要爱上你的仓位。比如交易过程做多，结果行情突破急跌。这时，很多人可能舍不得止损，眼看着亏损越来越大，最后砍在了最低点。要像小孩一样，跌了我反手做空啊。

当然，这里并不是说真的要做空，而是不要太把之前的仓位当回事。

  • 我和你打闹，那是刚才的事；我又和你一起玩，那是现在的事。
  • 我做多，那是刚才的事；我又做空，那是现在的事。

不让过去成为负担，随时轻松的切换。

1024 - 看书学育儿

发表于 2021-10-24 | 分类于 每天写一点

本周看了《陪孩子走过3-6岁敏感期》这本书，感觉挺好的。

书的结构设计得不错，读起来没有压力。基本是分成几个系列，每个系列有一些观点，每个观点先讲个案例，然后再引申些观点。每个章节都很短小，随时能停下来，也随时能继续阅读。

如何当父母，真是个大话题。生个娃，就自动成为父母了，好像根本就不用学习。之前，很多育儿方法都是从上一代传下来的；甚至，根本就是上祖辈带孙辈。然后，自己通过上一代的言传身教，和自己的摸索，慢慢学会当父母。

现在不一样了，新生代父母，普遍不认同自己父母的教育方式，都希望用更好的方式来教育。问题是，怎样才是更好的方式呢？只知道个大概。自己又忙，没时间持续、深入的学习，只能是碰到问题解决问题，小问题积累成大问题才去解决。现在很多都是利用搜索、公众号、社群，来解决孩子的各种问题。

这种父母如何成长为父母方式的转变、青黄不接，是孩子教育的一大问题。无论怎样，学这个过程是无法避免的，不管具体是什么形式。
