0629 - 图书馆是个好地方

发表于 2021-06-29 | 分类于 每天写一点

最近一直在蹭家边上的汉堡王干活，免费空调、环境也还行。

今天想换个地方，搜了下图书馆，发现不太远的地方，有个区图书馆，骑过去只要 10 分钟左右，不错。

关键，环境很好：边上一排排书架，很有感觉；大桌子，舒服；免费网络，虽然人多了不快；来看书的人都很自觉，听不到人说话，很安静。

哈哈，发现一个宝地，最近不用找共享工位了。

0628 - 手机与电动车的续航焦虑

发表于 2021-06-28 | 分类于 每天写一点

手机和电动车，都有续航焦虑，拿来简单比一比。

当年的诺基亚，充一次电用几天，没有电量焦虑。

现在的智能手机，基本一天一充，有电量焦虑。不过，也有解决办法：随处可充电，再加上自带或街边充电宝，基本能缓解焦虑。

电动车和诺基亚比较像，充一次电用几天，却有电量焦虑。

焦虑的主要原因，可能不是电池的单次续航，而是充电麻烦。

眼下，电池技术短期内突破无望。电动车的思路，应该是和现在的智能手机类似，让充电更方便、更快。

0625 - Excel 函数

发表于 2021-06-25 | 分类于 每天写一点

时不时，会用到 Excel 进行数据统计、图表做图。这时，就需要使用 Excel 函数。
