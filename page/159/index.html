0427 - 邻里问题很麻烦

发表于 2020-04-27 | 分类于 每天写一点

碰到一个不好相处的邻居，很麻烦。

今天跟朋友聊，他也跟我吐槽了奇葩邻居。也是采光权之类的问题，基本是我占你便宜可以、你的正常权益我也想试试的那种。

碰到这样的，软硬都不好折腾。用明面的、甚至是法律这样的办法，周期很长、成本高，且最终执行会打多少折扣也不好说。用阴的吧，又变成了比下限的操作，咱不擅长啊。

更麻烦的，还不能一走了之，还得抬头不见低头见。这种高频次、小难受累积在一起，真是要了亲命了。

碰到这样的，我是没什么好办法。在走完正常途径后，更可能的是苦练内功，降低对单点的依赖，增加选择余地，就不至于过于难受。

0426 - 思考宏观问题

发表于 2020-04-26 | 分类于 每天写一点

通常，慢慢的，我们的思维、视野会越来越狭窄，聚集自己身边的事。有时，哪怕是一颗树、买什么手机这样的小事，都会耗费大量的心神。

只把目光放在手边的小事，会让我们逐渐丧失思考宏观问题的意识和能力。

而一旦站得更高、看得更远、高屋建瓴，一些问题就不再是问题。

很遗憾，一旦我们试图站起来，懶、现实的压力等等，会立刻将我们拉回来，压在地上摩擦。慢慢的，也就习惯了，也就这样了。

0424 - 不够渴望美好

发表于 2020-04-24 | 分类于 每天写一点

感觉最近没什么进展，进而做事也感觉没劲。想了想，还是动力不足。

现在做交易，初衷自然是赚钱。而以赚钱为动力，除非确实视财如命；否则，一旦赚钱遇到挫折（这是必然的），立马感觉沮丧、萎顿。

我之前的想法，和绝大多数人一样俗：赚钱；有足够的钱之后，做自己想做的事。什么是自己想做的事呢？这个还算清楚：感受用户需求，思考解决方案，促成产品。落实成具体的，可以是我之前做的 iText、iPic、Klib 等一系列产品。

除此之外，我好像并没什么特别的物质或精神追求。物质上，大房子、豪车之类的，完全没感觉（可能是我并没有接触过吧😂），唯一比较有兴趣的，是数码类产品，比如高性能电脑。精神上，比较喜欢独处，比较兴奋的不多，比如看科幻小说。当然，这里并不是罗列，只是在落笔时想到的一些。

总的来说，感觉自己太封闭，接触到的美好太少，进而缺少对美好的追求。没有欲望，并不见得是好事；可能因此缺乏动力，死水一潭。
