0530 - 交易中的低频与情绪

发表于 2022-05-30 | 分类于 每天写一点

如果交易次数多，总会遇到见鬼的行情；明明好的信号，行情就是反向一根针所止损，那能怎么办？

关键一点在于，这样的事情会影响情绪。这样的情况出现几次，会影响情绪的稳定，会怀疑交易系统是否需要调整，甚至怀疑是否真的需要止损。

试问，当情绪出现这样那样的问题，当遇到一个信号不好、但事后看能走出大行情的信号，你还敢做吗？如果错过这样的信号，交易的整体预期由盈利转亏损，还能抗得住吗？

所以，低频很重要的一点：能保护情绪。我们都是主观的、有情绪的，不要假想自己有很强的情绪控制能力，不存在的，不要挑战人性。我们能做的，就是好好的呵护情绪，避免轻易的被打乱。

低频、交易次数少，意味着即便出现见鬼行情，也有足够的时间安抚情绪、恢复平静，不容易暴走。

0529 - 《反骗局中局》

发表于 2022-05-29 | 分类于 每天写一点

不知不觉，微信读书书架上放了很多书。准备过一遍清理一下，无意间打开了这本书：《反骗局中局》

没想到，这是本小说；

没想到，这本书很好看；

没想到，这本书巨长…

一不小心，这周花了很多时间刷书。

这本书也符合上面说的爽文套路：一出场，女主白富美，能力强，高冷，是男主上司。男主高帅穷，但能力更强、潜力更大。随着剧情推演，男主地位不断上升、光环不断加持。最后…噢，不，书太长了，不知道最后怎样…

0528 - 高帅穷 > 高帅富

发表于 2022-05-28 | 分类于 每天写一点

看过的爽文、小说、电视剧，大多有这个套路；高帅穷遇到白富美，之后变身高帅富，门当户对。

一般，女主一出场就自带光环，或是家境好、家族实力强大，或是已经高段位，或冷艳、或温柔，总之很美。

而男主一般比较惨，或是绝世老爹死了，或是因为善良而窘迫，或是曾经很牛但因为意外开始扮猪，或是表面痞子而心地善良。总之，一定是好人、潜力巨大。

之后，男主各种好运气、各种阴差阳错，每次反派的阴险行为，总能被男主化险为夷，甚至因祸得福、获得精进。

一番又爽又刺激的操作，男主不断进化，大大超越女主。大多时候，故事到这里就结束了。如果有后续，一般没女主什么事，会换条线继续。毕竟，已经超越女主了，再进化就不般配了。

0526 - 典型日内交易者

发表于 2022-05-26 | 分类于 每天写一点

盈透 (IB) 有个限制：如果账户金额 < 25000 美元，则 5 个工作日内最多有 3 次日内开平仓。

这一限制，目测是为了保护小交易者，毕竟日内交易难度大，新手又容易冲动交易。这一限制，可以避免情绪之下频繁交易，让爆仓来得更晚一些。

由于不知道这个限制（其实之前遇到过，时间太长忘记了），为了测试，手动进行了日内开平平仓，触发了限制。现在不想充值，只能等 5 个工作日再试了。
