0405 - 大处着眼，小处着手

发表于 2021-04-05 | 分类于 每天写一点

这个思想，在交易中很有用。也可理解为，宏观与微观。

盘前计划考虑大盘、关联品种，确定大方向。然后，在微观验证、寻找具体的交易机会。

比如，今天是盘整行情，则整体的策略，是在震荡区间上沿做空、下沿做多。如果当前是在震荡区间上沿，无论形态有多好，最好都不要做多。即使价格突破震荡区间，最好还是耐心等待回抽确认。

大方向对了，至少大亏的概率会小很多。

0404 - 炒单的节奏

发表于 2021-04-04 | 分类于 每天写一点

日内炒单，看似机会很多，但对盘口等细微的能力要求很高。

还有种表现形式，就是对节奏的把握。多空转换，可能就在几分钟之间。

  • 节奏踩对了，两边赚；
  • 踩错了，两边吃巴掌。

如果踩错了，最好老老实实休息一会。脑子清空了，再回到电脑前。先不要着急下单，观察一下。看出门道、看到自己熟悉的形态，再出手。

0403 - 交易如游戏

发表于 2021-04-03 | 分类于 每天写一点

把交易当成游戏，每次开平仓就是一局。这局没打好，想想如何改进。下一局去验证，改进策略有没有效。

但，不要把交易当成打怪。市场是大 BOSS，但不是被打到的那个，也是打不倒的。

市场更像是猫主子。无论它怎么高冷，不鸟你或甩你一张寂寞脸，你都要想办法把它撸爽了。

0402 - 商业中的抄作业

发表于 2021-04-02 | 分类于 每天写一点

这两天，小米官宣造车，雷军的一篇文章在朋友圈刷屏了。大概意思，造车是雷军的最后一次创业，放手一博。

评论中看到一个观点：小米这是在抄作业。

我当时第一反应是：当年，最开始做 Android 的是 HTC。可是，现在 HTC 早就退出手机业务了。

如果说是抄作业，商业中的抄太多了。且不说版权之类本就说不清楚的事，商业并不看重这个，而是看谁能笑到最后、是最后的赢家。至于成功者有没有原罪、有没有黑历史，大家会选择性遗忘。毕竟，商业里，笑贫不笑娼。

对于用户而言，谁的产品便宜好用，就用谁的。至于情怀、品牌忠诚度之类的东西，都是事后总结出来的，当作笑话或者调侃一下也就算了。

0401 - 豆类隔夜跳空

发表于 2021-04-01 | 分类于 每天写一点

今天把我惊了一下。

一直在做豆类。今天早上开盘，价格直接高开 5%；这在期货里，可以接近涨停的水平。怎么回事？第一反应，是不是换主力合约了。检查一下，并不是。有点蒙，还好不傻。这种看不懂的行情，还是不要去交易了。

接着去看新闻，才知道是美国农业部昨晚发了重磅报告，豆类种植面积不及预期，价格应声上涨。

事后老诸葛，虽然知道了原因。可是，吃一堑、长一智，下次不能再犯类似的错误了。举一反三，还是哪些比较重要的周期性报告呢？

去 大商所 找了下，找到了如下的报告：

  • 美国农业部，每年 3 月、6 月底， 会发布种植面积和种植意向报告。
    • 注意：历史上，此报告会对市场有巨大冲击，也是本次出事的报告。
  • 美国农业部，每季度月底（1、3、6、9）会发布谷物库存报告。
  • 美国农业部，每月上旬（10~13 号左右）会发布大豆、豆粕、豆油等供需报告。
  • 美国油籽加工协会，每月中旬发布豆粕油类压榨相关数据。

之后，为避免风险，在重要报告发布前，应该提前减仓、平仓。
