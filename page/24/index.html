0721 - 社群的一种结局

发表于 2021-07-21 | 分类于 每天写一点

做过群、论坛的，都有这种感觉：

  • 刚开始加入的，通常是很兴趣，也有一定能力，或者成长很快，慢慢成为社群的核心成员。
  • 社群总要长大，吸收越来越多的新成员。这些新成员中，必然有很多菜鸟、新手。
  • 新手的一些「白痴问题」、伸手党的习性、越来越多的灌水，都让原来的核心成员越来越不满。最后，愤然、戚然离开。
  • 「劣币驱逐良币」，群越来越小白、越来越水。最后，不了了之。

不论是什么形态，本质上都是人群。有人，就会分化，就不可能真的大一统。

怎么解决这个问题，我也不确定。能想到的是，这种「扁平」化的组织，是不行的。一定要是立体的、可进阶的、有上升通道的。这样，每个人总能和自己相同层次的人聚集在一起。

0720 - 河南大暴雨

发表于 2021-07-20 | 分类于 每天写一点

一次破记录的大暴雨，持续时间又很长。

郑州，作为省会城市，内涝很严重，也吸收了最多的关注。

现在，短视频之类的自媒体，比图文时代更加及时全面。不管官媒如何报道伟光正，你总能看到各种反应底层问题的报道。

比如，官媒说人民子弟兵勇敢冲在最前线。这固然是事实。然后，更多没报道的事实上，居民楼地下室被淹，完全没有人管；一颗翻倒的大树横在大路上，也没人去清理。

还有个点，在这样天灾面前，农村比城市有更强的抗灾能力。换个角度讲，城市居民依赖更多的公共服务，比如水、电、食物、交通、垃圾清运。农村虽然各项基础比较差，但要求也低，已有的条件，反倒能显现出更大的包容性。

在手机上看别人遭灾，和自己亲人手机失联，是完全不同的感受。真正受过灾的人，会激发出更多的同理心。

0719 - 回调验证心态

发表于 2021-07-19 | 分类于 每天写一点

交易中的回调，非常能检验交易的能力和心态。

首先，回调是不是在自己的计划、预期范围内。如果自己的计划完全没考虑到回调、对回调没有包容性，显然这样的计划是有问题的。

然后，如果计划中有回调的部分。可当回调真的出现时，自己又接受不了。心情开始变差，甚至开始胡乱交易，那计划还有什么意义。

当然，最好是小回调、小亏损。如果还做不到这一点、或者亏损时心态不好，那就先轻仓操作；至少不会大亏，不至于乱来。

0718 - 交易，简洁有力

发表于 2021-07-18 | 分类于 每天写一点

交易，两个极端更容易见效。

一种是简洁，考虑的因素不多，规则清晰明确，很容易执行。这种，适合个人使用。

一种是复杂，考虑尽可能多的因素，模拟现实金融环境，以期最真实的理解现实，做出最佳决策。这种，明显适合大机构使用。

大部分人，其实两边不靠。太简洁的东西，自己不信任，不相信这么简单的东西就能赚钱。于是，慢慢的加各种东西。遇到一种行情，规则一说不做，规则二说做。自己主观想做，就倾向于规则二。规则越多，其实和没有规则差不多。

明知做不到复杂，就往简单了做吧。
