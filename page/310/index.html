0401 - 4 月计划新鲜出炉

发表于 2017-04-01 | 分类于 每天写一点

一不小心就到 4 月份了，思考了下这个月的主题，记录一下：

  • Klib 新版发布
    • 回顾一下，3 月份只做了一件事：开发新版 Klib
    • 回头来看，节奏是有些慢。本打算在 3.28 发布的，结果由于审核、海外推广等原因，推迟至 4.5 发布。
    • 不过，好在对 Klib 目前的形态还比较满意：Klib 功能完备、交互简洁，还有多项地表最强、世界第一，对 Kindle 爱好者来说，确实是比较有用的工具；是拿得出手的产品。
  • 高流量需求
    • 自己目前做的产品，都是 小众需求 + 细分领域 的产品，喜欢的人很喜欢，但用户总量不大，推广起来也有些吃力。
    • 下一步，希望能做 大众需求 + 细分领域 的产品，即需求本身是时下热点，不过不是热点本身，而是围绕热点的周边服务、深度需求。也锻炼下自己对需求的把握能力，以及伺候更多用户的能力。
  • 品牌建设
    • 最近，在海外推广 Klib 时，遇到很大的瓶颈。究其原因，很大部分是因为自己没有响亮的品牌。相反，如果有 Omni 团队出品 这样的光环，会很容易吸引眼球。
    • 当然，自己的产品小众、影响力有限。而越是这样，越要聚集能量，组合推广资源，用最有效、最节约时间的方法，不断累积影响力；杜绝低效推广、浪费时间。
    • 具体的，从网站 SEO 优化做起，同时考虑域名、品牌形象、等等因素。
  • 思考、重新定位
    • 做独立开发 15 个月了，也到了该突破的时侯。经过上面的尝试后，思考下一步如何。

0331 - 运营，非一日之功

发表于 2017-03-31 | 分类于 每天写一点

今天 Klib 的海外推广依然不顺利，几乎没有有效收获。

其实想想，除了运营的技法，最主要的是自己没有品牌、没有影响力。而作为媒体、编辑，都希望通过报道有趣、有用、有价值、有影响力的产品，来增加自己的价值。所以，除了个人关系的熟络，更重要的是自己提供的东西，能帮助其增加价值。

而个人品牌的建立，并非一日之功。比如域名、品牌的选择，比如 Twitter、Facebook 的经营，比如坚持写博客，比如英文写作功底，等等，都是需要长期的支持、磨练，才能见效。

好消息是，一旦建立了良好的运营渠道、方法，但是很大的优势，因为竞品想要追上来，也非一日之功。

0330 - 运营：全面撒网、不如单点突破

发表于 2017-03-30 | 分类于 每天写一点

今天推广了一天 Klib，收效甚微，心累…

回头来看，就像投简历一样，随意的广撒网，不如盯住心仪的公司（媒体），重点突破。这样，保证有一家的可能性更大。并且，一旦在一家媒体上爆光了，在其他媒体上推广的难度就会小很多。

再者，时间有限，也不可能对大量媒体重点进攻，只能选择更对口的媒体。

换个角度，从媒体的角度，什么样的产品是他们愿意爆光的？从用户的角度，什么样的介绍方式是他们愿意接受的？从这些角度、而不是产品说明书的角度，好好的写文稿，然后再去敲门。

0329 - 运营，别急

发表于 2017-03-29 | 分类于 每天写一点

最近是有些急躁，总感觉运营有好多事情要做，却又感觉没做什么事、进展很慢。再加上 Klib 审核接连被拒，确实有些急躁了。

不过，反过来想，别总想着还有多少事没做，想想凡是做点推广的事，就有可能多一位用户。这样想想，压力就小些了。

目前，对 Klib 还是有些自信的，相信只要是需要管理 Kindle 标注的用户，都还是会喜欢 Klib 的。
