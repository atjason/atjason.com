1208 - 本能技能

发表于 2019-12-08 | 分类于 每天写一点

最近给小孩买了显微镜，就是有千倍左右、单目直筒的那种，我上学的时候很常见。也能看清楚一些好玩的东西，比如植物皮中的细胞、金鱼鳍中的血液流动等等。

我想说的是，十几年没碰过这东西，我竟然几乎能立即上手、甚至可以说是熟练，比如先低倍后高倍；物镜高倍时，先调到最下的位置，然后向上调；等等。

还有一些类似的技能，比如说骑自行车、游泳。这样的技能，可以说学会之后，想忘都忘不掉，像是把技能变成了本能。把你扔水里，你还能假装不会游泳？

与之相反，另外一些技能，却是稍不练习便会生疏、一生疏基本等于无用，比如编程。

1207 - 技能不嫌多啊

发表于 2019-12-07 | 分类于 每天写一点

最近梯子实在不稳，就有了找最快梯子的需求。

刚开始的想法是这样的。服务商提供订阅地址，通过这个地址，可以得到当前所有可用服务器地址。先用 ping 对其排序，然后挑选靠前的一些，直接使用其下载海外不可描述的文件，用这种方法进行实际测速，然后挑出最快的，并自动应用。

考虑到服务器是否可用、速度是否快是动态的，可以在后台定期运行这一小工具，总是用最快的，也提高了稳定性。

技术上核心一点在于：通过代码将 ss 服务转换为 socks 代理。我是打算基于 node 实现，可找了关于，最后发现官方已经 Deprecated 了。而 Python/Go 等版本，还活得挺好。哎，基于没有这些技能，这一想法只能搁浅了。

1205 - 享受闲暇的能力

发表于 2019-12-05 | 分类于 每天写一点

如果有一段时间，什么都不用干，你能享(忍)受多久？不拿手机、不看新闻、不刷抖音，就这么静静的呆着。你觉得是享受、还是煎熬？能坚持多久？

人有两大敌人：贫穷和无聊，分别是穷人和富人的敌人。

最近看叔本华的《人生的智慧》，他认为幸福来自内在，现在的我也这样认为。用忙碌来填补内心的空虚，是穷人的做法；用生理感观的刺激来对抗无聊，是富人的陷阱。

是享受还是忍受闲暇，是内心是否充盈的一种标志。

1204 - 自己判断价值

发表于 2019-12-04 | 分类于 每天写一点

50 元买了本书，随手在多抓鱼上一扫，回收价 5 元，瞬间觉得买亏了。

可，明显书的价值不应该这么算。书买来不是用于卖的，而是为了从中学到东西。至少能学到多少有价值的东西，这就难说了，且因人而异。

比如，同一本书，豆瓣上有人打 10 分、有人只给 1 分。如果不考虑作弊，那就是有人觉得很棒，有人觉得完全无用。

不过是多抓鱼回收价，还是豆瓣评分，还是别的什么评价，都是别人的，是别人根据对他是否有用而给出的评价。既然是别人的评价，也就只能参考。既然是参考，就不能因此替自己下判断。

说到买书，其实现在很多书微信读书上都有。即使不是会员，也可以免费阅读前面几章。与其看别人的评价，不如自己亲自读上几章，就知道是否值得继续阅读了。
