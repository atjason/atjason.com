0903 - 体重是玄学

发表于 2019-09-03 | 分类于 每天写一点

用小米体重称，和医院的体验结果相比，竟差了几斤。有时，挪个位置再称，也能差几斤。如果是 200 斤的胖子了就算了，关键是小孩的体重差这么多，就失去了监测的意义。

发朋友圈，也有朋友有类似问题。还有朋友说，医院的秤也不一定准…

妈呀，那到哪才能得到准确结果。科学发展到现在，一个小小的体重，竟成了玄学？

想起一件趣事，娃出生几个月，社区家庭医生上门体验，是用布兜兜住小孩，用那种有秤砣的秤秤的。不过想想也是，不然用什么呢？总不能随身整个电子秤吧，关键那东西还不准。

0902 - 指数曲线

发表于 2019-09-02 | 分类于 每天写一点

小婴儿的学说话就像指数曲线。很长一段时间几乎什么都不会说，突然某个时间节点后，好像什么都会说了；还动不动就迸出高级词汇，给人惊喜。

财富的积累也是如此。比如巴菲特的钱，绝大多数都是 60 岁之后积累的。

不过，这也会导致资源错配的问题。换句话说，即使回报是指数曲线，需求或者说成本可不是，甚至是反过来的。比如，项目在初期，往往是需要资源最多的，偏偏这个阶段却没有收入；而到了成熟期，即使不增加投入，收入也会持续很长时间。很多死掉的项目，就是没撑到指数曲线爆发的拐点。

这就可以引出很多东西，比如风投，比如不下牌桌。

0901 - Shrimpy 的社交策略

发表于 2019-09-01 | 分类于 每天写一点

昨天提到了币的基金，难点就是如何从上千种币中，挑选中合适的币、制定合适的比例，以构成基金。

对于这一点，Shrimpy 的方案是：社交。

  • 达人提供策略并获利，小白付费以获得策略，平台获取抽成。
  • 对于达人
    • 可根据自己的专业知识，构建自己的基金模型，甚至是策略。
    • 开放自己的策略，通过赢利的历史来吸引小白跟随。
    • 利益是：每位跟随者，每月获利 $4。如果有 1000 名跟随者，每月便是 $4000，比较可观了。
  • 对于小白
    • 可跟随达人的策略。所谓跟随，就是小白的账户会 1:1 复制达人的交易，达到完全一致的盈亏。
    • 代价是：每月 $8.99 的固定费用。
  • 对于平台
    • 一个小白同时只能跟随一位达人，所以平台每月可从每位用户身上获得 $4.99。

了解之后，我其实觉得这个模式挺好的，顺利的话，达人、小白、平台多赢，大家各取所需。

0831 - 定投币圈指数基金

发表于 2019-08-31 | 分类于 每天写一点

越来越多的钱涌入币圈，币圈也越来越正规，也越来越需要更丰富的金融产品，来满足高中低档不同资本的需求。币圈的金融产品，自然很多是照搬传统金融市场来的。指数基金，便是其中一个。

Shrimpy 是个自动交易以完成资本增长的服务。其核心策略就是组建币的指数，然后定期 Rebalance。

  • 所谓币的指数，说白了就是挑些币，然后按比例分配资产。
    • 之所以要挑币，主要是平抑不同币的波动，避免非系统性风险，即避免在某个币中阴沟里翻船。
    • 如何挑币？这就是考验功力了，换个话题再讲。
  • 所谓 Rebanlance，就是随着不同币的价格涨跌，其价值比例会偏离当初的设定。于是，卖掉一部分涨的、买入一部分跌的，平衡比例。而这部分，是该策略跑赢 HODL（买入不卖）的关键。

李笑来近期搞的 BOX 定投，核心也是币的指数基金。

不得不说，对于小白而言，不论是传统的股市，还是币圈，指数基金确实很稳妥，定投又是最适合的策略。想在币圈赚钱、又不想承担太多风险、且没太多时间研究的朋友，不妨了解下定投币的基金，或者是以这种思想来投资。
