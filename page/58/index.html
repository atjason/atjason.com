0601 - 一定被骂的政策

发表于 2021-06-01 | 分类于 每天写一点

To C 的产品，最容易被骂。因为有各种各样的用户，不可能每位用户的需求都能被满足。不满足，自然就开始骂娘。

政策是最大的 To C 产品。中国有 14 亿人，能全部满足才怪，不被骂才怪。

比如，今天最大的儿童节礼物：国家开放三胎。

其实，批评容易建设难。如果你是当局者，你能制定出怎样更好的政策，又权衡了哪些利弊。

0531 - 止损的边界效用

发表于 2021-05-31 | 分类于 每天写一点

有时，止损多一点点，就能有效抗住回撤，等到调头向上的黎明。
有时，止损再多，也抗不住大阴线快速向下打。

行情好时、自己有信心时，多一点止损，多一点包容。
行情不好、自己没信心时，要么不开仓，要么小止损试错。

止损，是交易的成本，要把它的效用发挥到最大化。

0530 - 放下太难

发表于 2021-05-30 | 分类于 每天写一点

刚开始交易时，多空都做，是个机会都做（又或者说，因为想做，所以随便找了个理由称之为机会）。总想着做的越多，赚的越多。

慢慢的，发现一直在亏损。一方面，机会不真的是机会。另一方面，就算是机会，自己也没能力抓住。

现在，转了一圈之后，退化（简化）到只做多、只做一种机会。看似机会少了很多、很可惜。可事实上，胜率高了，无谓的止损少了，累积越来的盈利多了。

不止交易，人生何尝不是如此。

年轻时，觉得只做一点事，简直不值一提。我要做很多事，很多很牛的事。

慢慢的，发现自己的时间有限、能力有限，肩上的责任却越来越重、压力越来越大。原先看不起的一点事，现在能做好，已是不错。

如果能重头来过，从一开始，就只做最后做的一件事，可以吗？多半，不行。就像七个半馒头的故事。虽说，是吃到最后半个感到饱了、吃不下了。可，如果没有前面的七个馒头铺垫，最后半个断然是吃不饱的。

换个说法。人生不可能直线上升，多半是螺旋式。走了一圈，发现又回到原来的位置。其实，此情非彼情、你也不是原来的你了，已经比之前有所进步。在这个新的基础上，再来一圈一圈的向上升。

0529 - 寄宿学校

发表于 2021-05-29 | 分类于 每天写一点

中学生的教育，大体有这两种分类：

  • 一种是，每天在家住，上学、放学跑来跑去。
  • 一种是，住在学校，只是每隔一段时间放学回家。

哪种好呢？应该说各有利弊。说一种，另一种也清楚了。

这其中的差异，主要是家长在孩子的教育中，起什么角色。对于大多数家长，在中学阶段，在辅导作业等直接教育层面，能起到的作用已经很有限了。无非也就是让孩子吃饱喝好。

再有，就是家长在人格、看到事物的观点角度等方面，对孩子进行耳濡目染的影响。可是，坦白讲，大多数家长，本身也是普通人，能不能在这方面教育孩子，还两说呢。

所以，从直接教育角度，大多数普通家长能起到的作用比较有限，和是否住校差别不大。而对于精英家长，差异还是有的，应该是住家好。

当然，家长能起的作用，肯定不仅仅是纯粹教育本身，还有亲情的培养。而这对于青少年的 心理健康，是非常重要的。尤其，前些天发生的中学生坠楼身亡事件，主角就是住校生。一些网络未考证的数据：

在一篇《青少年自杀行为原因分析》报告中提出，15~34岁青少年占自杀人数的20%，而且这个比例还在上升中。
卫福部公布的2020年公布的数据显示，青少年死亡率创24年新高，而自杀是青少年的第二大死因。

这些数据不一定准确，不能定量分析。但从定性的角度，应该是错不了。自杀代表问题已经很严重；而没有严重到自杀程序的心理问题，自然不少。要解决孩子的心理问题，家长无疑是最重要的。学校能做的，大体也就是在楼道里装防盗网，让学生跳不下去。

还有一点差异：培训班、兴趣班上不了了。这个利弊就不好说了。可能培训班对分数提升有帮助，也可能会摧残本就不多的对学习的兴趣。

家长会有一点担心：学生在学校里变坏了怎么办？一周回来一次，突然发现会抽烟了，家长能做什么？

在现在这个时间点，还有个很重要的区别：玩手机游戏。在学校，可能严格禁止带手机。而到了家里，家长通常不可能做到 100% 禁止孩子玩手机。我不是说一定不能玩，而是说孩子的自控力通常很差。一定手机成瘾，几乎没有好的办法戒除。

其实，对于学生来讲，很多回家了也是作业写到半夜，跟在学校也没多大差别。

我自己当年是寄宿学校，一周回家一次。我倾向于孩子住家；除非他自力能力很强，本人又有很强的意愿住学校。
