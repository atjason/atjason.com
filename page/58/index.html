0526 - 套路

发表于 2020-05-26 | 分类于 每天写一点

最近看了几本小说，发现小说是有套路的，或者说是写作的定式，就像是写文章的三段论。

交易中也是有定式的。就像是事先做一把某种形状的尺子，然后去市场里套；套中了就赚，不然就亏。当然，这种形状，是市场里经常会出现的形状。

看的套路多了、看透了某种套路，就很难再被打动。比如，你看某个乞丐很可怜，施舍了。可要是天天上班路上都看到这个人，你能不麻木吗。

简单的套路，脑补就够了，也就很难从中得到愉悦。怎么办呢？上新的套路、更复杂的套路。我们时不时觉得自己之前的某些举动很幼稚，就像是脱离的低级套路的趣味，要追求更高级的套路。

回头看看，就像是套娃，不都是套路吗。有点被绑架的意思，被推着寻找复杂套路带来的刺激。

就像吃饭，慢慢觉得不够有味，于是加更多种、更大量调料；等适应后，再换再加。有没有想过，不加任何调料，只吃食材本味，也很不错。

本文有点偏激，看看就好。

0525 - 长期价值

发表于 2020-05-25 | 分类于 每天写一点

我们总是太急了。尤其是赚钱的事，更是急上加急。

比如，学了交易方法，在先交易标的时，自然很想选择波动大、机会多的，比如数字货币。

但往往忽略的一个重要的特性：长期性。

比如，相比而言，股票市场有数百年的历史。可以预见的很长一段时间，都会稳定存在。而数字货币呢？本身成气候也就最近几年，未来几年、几十年会变成什么样、还有没有现在的波动性等等，这些都不好说。

交易虽然类似，但每个标的都有自己的门道、脾气。在一个标的里摸爬滚打久了，再切换到另一个，也是需要成本的。

基于以上考虑，在选择交易标的时，能不能忍住心里的小猫挠心，放弃短期的，去选择长期的呢？

孰优孰劣不是必然的；但，必须要将长期性考虑在内。

0524 - 养鸡

发表于 2020-05-24 | 分类于 每天写一点

在家呆了这么久，今天终于满足了养鸡的愿望。

其实，主要是给娃养的。鸡刚到家的时候，还是有怯生，站着不敢怎么跑。娃就用小手指，嘴里咿咿呀呀的说着鸡鸡鸡，追着鸡跑，挺好玩的。

还有个动机，就是娃每天浪费的饭太多了，扔了浪费，养鸡绝对养肥了。

不过，鸡还是带来的一系列麻烦。最大的就是邻居的猫会来偷吃，已经吃了我家好几条鱼，以及各种食物。小鸡要是这么放院里，不出 3 天，绝对进猫肚子里了。于是，就得给它们做个笼子，还得是很密的那种。找材料、制作，花了大半天。

哪天拿绳子拴着，去河边溜溜鸡。

0523 - 装空调

发表于 2020-05-23 | 分类于 每天写一点

天热了，就得装空调。

下单，第二天就送到家了。这听起来没什么是吧，要知道这可是农村哎。空调这么大件，还能有这样的效率，有点惊着我了。

收到货后，卖家会主动打电话；确认收货后，第二天就安排工人师傅上门安装。还是那句话，要知道这可是农村哎。这服务，有点规范，不像粘粘糊糊、人情社会的做法。在城市里呆的时间长了，更喜欢商业化的做法：我花钱、你办事、两相安。

一个原因，是空调比较容易标准化，更可能在全国范围内铺开。就像当年亚马逊起家，选的就是书这种很标准化的东西，更容易直插到底，打破已有的各种旧习俗。

对了，选的是米家的空调。为什么呢？便宜啊。还支持手机控制，挺方便的。

0522 - 太阳能好

发表于 2020-05-22 | 分类于 每天写一点

天越来越热，日照也越来越充足，就想着利用太阳能做点什么。

先是买了个太阳能路灯，装在大门口，晚上吃完饭，娃可以在门口跑一跑，消消食。还在院子里装一个，晚上亮堂堂的。关键不用布线，还省电，用着不心疼，没有心理负担，满足感就更高了。要知道，这样一个路灯，包邮才 33，真是不错。

又买了个太阳能灭蚊灯。其实原理还是紫外线之类的诱引，然后高压电死。关键还是电，太阳能就很合适了；白天晒着充电，傍晚蚊虫多时消杀。

还有一个号称最小的太阳能车。结构极其简单，就是太阳能电板 + 电机 + 车架。太阳一照就跑，跑到阴影下就站住，也挺好玩的。

太阳能真好玩。
