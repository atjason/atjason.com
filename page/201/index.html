0311 - 对比与印证

发表于 2021-03-11 | 分类于 每天写一点

比如，日常生活中的警示条，不会是单纯的红色或橙色，而会有至少两种颜色对比。不在于单一颜色有多醒目，而是高对比度醒目。

屏幕中也有类似的现象。OLED 屏幕的最高亮度，可能比 LCD 差很多。可是，OLED 的对比度非常高。或者说，黑色非常黑；在非常黑边上的白，哪怕亮度比 LCD 低，和非常黑一比较，就显得非常白、非常亮。

交易中，一些现象单独出现，可能并没什么意义。可如果关联因素同时出现，可能意义就比较突出。

比如，关键价位的突破，伴随着大成交量。这样的突破，则更可能是真突破。

比如，在连续上涨到高位，出现连续的大买单，却并不能把价格推高。则可能是遇到了强有力的卖压，也可能是庄家开始出货，上涨可能到头了。

0310 - 组织与个人

发表于 2021-03-10 | 分类于 每天写一点

组织有个好处，能推着个人前进。哪怕你想偷懒，也会有人拿着小鞭子，在后面抽着你的小屁股。

个人则不一样。只对自己负责，没有表面上的同事或领导的督促。如果没有坚定的毅力，很难对抗人性中的懒惰，很容易从放松到放纵。而且，由于缺乏必要的反馈，这样的下滑可能发生在不经意间。浑然不觉之时，已然落后别人一大截。

感觉，我就在这种可怕的下滑中…

0309 - 蓝牙翻书器

发表于 2021-03-09 | 分类于 每天写一点

大屏阅读器 + 蓝牙翻书器，配合起来用，很爽哎。

可以把阅读器固定在一个位置。可以是和视线水平的位置，然后自己站着；也可以放在桌子上，然后自己窝在沙发里。看完一页后，遥控器轻轻一按，即可完成翻页。阅读器不会像触摸时那样晃动再平衡，节约不少时间，以及脖子和眼睛的运动。

不过，缺点也是有的。毕竟，多了个东西，用的时候要在手边。如果做成磁吸的，不用时吸在阅读器上，倒是不错的。

0308 - 个人交易者的优势

发表于 2021-03-08 | 分类于 每天写一点

凡事，都要有相对优势，才有取胜的把握。

对于交易机构，他们的优势非常明显：资金雄厚、高手如云。

与之相比，个人交易者如何与之抗衡？又怎么能虎口夺食呢？

个人交易者最大的优势是：可以选择不交易。

对于机构，很难选择不交易。总不能跟投资说：现在的行情不好、或者说我找不到好的交易机会，所以我不交易，钱都闲着。这显然是投资者无法接受的。

而事实就是，交易市场不是每天都有机会。就算有机会，也有好坏之分，不是每个机会都要抓住。

个人交易者，就没有这个负担。想交易就交易、不想交易就不交易。关键的，可以只选择好的机会进行交易。

如果个人交易者放弃了这个优势，过度交易、胡乱交易，那可真是放下武器与军队开战啊。

0307 - 墨案 InkPad X 阅读器、微信读书与多看读书

发表于 2021-03-07 | 分类于 每天写一点

Klib 刚刚支持从微信读书导入笔记，对微信读书好感增加了不少。正好对一直使用的 Kindle PaperWhite 有些不满，觉得屏幕小了点。于是，产生了 换个大屏电纸书阅读器 的想法。于是，开启一系列 踩坑之旅。

首先，我的需求非常简单：能运行微信读书的大屏电纸书。对于大屏，13.3 太大了点，10 寸几乎刚刚好。在 10 寸这个系列中，也有一些可选的产品。最大牌的自然是 文石 Boox；可惜，价格也非常美丽。以及，这些产品大多有手写之类的功能。而这些加钱的附加功能，并不是我所需要的。我倒希望有没有这些功能的低配廉价版。最后，经电纸书专家直树桑的推荐，选了 墨案 InkPad X 这款阅读器。

京东下单，次日收到。对比可见，屏幕是 KPW 的 2 倍多点。

InkPad X 默认是多看阅读器。没办法，谁让人家是小米投资的企业。不过，还是良心的自带了微信读书。

打开 微信读书，发现 第一个坑：不支持邮箱登录，只支持微信扫码登录。微信读书 App 里，是可以扫码登录微信读书网页版的。可惜，这招也不行。咨询了微信读书的客服，目前还没回复。不过，不太抱希望。

算了，先忍了，就用微信登录。然后，又发现 另一个坑：不支持横屏、进而也不支持分栏。这就很尴尬了。人的眼球，横向运动是比较弱的，比较擅长上下运动。换成看书，就是比较适合读比较窄的内容，眼球不太需要左右看，更多的是向下移动。这种情况下，如果直接是 10 寸的屏幕竖着看，其实效率是不高的；不仅眼球要动，甚至脖子也要左右动才行。其实，微信读书 iPad 版是支持横屏分栏的，希望微信读书能早日适配大屏电纸书吧。

微信读书不行，转而试试自带的多看读书。其实，多看读书的阅读应用，可定制化是最高的。什么行高、段落间隔、左右上下空白、字体等，全是可以定制的。更惊喜的是，支持横屏分栏，这简直是最佳阅读体验。

不过，多看也有坑，商城的书太少了，远没有微信读书丰富。只能自己找 epub 版电子书（哎，这其实通常是盗版），然后上传到阅读器上看。另外，目前 Klib 在导入商城之外的书的笔记，还有点问题（研究了下，还是能解决的，近期有时间升级下 Klib）。还有个小问题，就是 阅读器和手机上的多看应用，在同步阅读进度时，有问题。是会提示跳转至更新的位置，但其实并不是最新位置。

哎，折腾了半天，目前还是没有完美的方案。最期待的，就是微信读书适配下，支持横屏分栏了。故事发展到现在，我其实有点想退货的。毕竟，花 1679 元买个电纸书，却没有带来明显的阅读体验的提升，反而引入了一些新问题。不过想想，我在这事上花的时间已经太多，时间也是成本，不想再折腾退货这种烦心事了，就这么用着吧。

还有一点，就是 蓝牙翻页器，其实就是模拟点击下一页。这样，看书时就不需要手去戳屏幕翻下一页了。这一点的体验提升还是很大的，因为手戳屏幕时，屏幕会晃动再平衡，这对读书是种干扰，而且是种频率很高的干扰。已经买了，还没到货。
