0826 - 快速决断力

发表于 2020-08-26 | 分类于 每天写一点

下楼，发现天可能下雨，要不要上楼拿伞？

如果你纠结了，甚至纠结后还有点纠结，很抱歉，你没有快速决断能力。

没有快速决断能力，就不要做日内交易。

比如，你本打算突破某个价位时买入。可价格突破的有点磨叽，甚至又有点回调。你纠结了，要不要买？买了会不会跌？要不要再等等看？

就在你纠结时，价格疯涨、气势如虹。你一拍大腿，那个后悔啊，我就说会涨嘛。然后，你又开始纠结，要不要追高，要不要再等等看……

哦，说的好像是我。

0825 - 自由恋爱 vs 包办婚姻

发表于 2020-08-25 | 分类于 每天写一点

结婚前，自由恋爱获胜，毕竟找到了自己真心相爱。

结婚后，5年、10年后，小孩、老人、各种困难矛盾暴露出来了。门不当户不对，不是问题的问题，也成了问题。不该发生的冲突，经常发生。

不听老人言，吃亏在婚后。这时候再看孰优孰劣，未可知。

当然，这里没有绝对论，只是以我仅有的认知，得出的概率性的结论。

年纪越大，阅历越多，越不会抱着一个观念不放。因为世界太大，未知太多。

0824 - 放过别人，放过自己

发表于 2020-08-24 | 分类于 每天写一点

淘宝上经常有这样的伎俩：商品的起步价很低，宝贝描述的功能很丰富。而在实际购买时，发现有很多型号。而功能丰富的，是最下面最贵的配置。最上面默认选择的最便宜的型号，功能是很差的。

这就是个坑，需要消费者很认真的去比较，到底乞丐版中，有没有宝贝描述里的那些美好功能。稍有不慎，就买到以为有某功能、实际却没有的型号。

这不，这次我就遇到了这个问题。有 2 个功能没有。其中 1 个，确实是型号的图片中有展示。注意，是图片中的细节，而不是图片中的文字描述。而另一个缺失的功能，完全没有任何提示。

也就是说，描述里说有此功能，选择型号时的介绍里也没说没有此功能。以一个正常人的思维来判定，这个功能自然该有。联系卖家，卖家就说，你选的乞丐版，就是没这个功能，完全不觉得自己有问题，反而说是我自己不看清楚、买错了型号。

我被激怒了。申请退款、举报卖家，操作起来。

从卖家的回复来看，应该经常面对此类问题；对于小二的话术，很有一套。就是往「买家选错型号，这个型号就是没这个功能」上引导。就卖家这个态度，我准备刚到底。

淘宝这个消费体验，实在是太差了。

从我个人的角度，这东西也就几十块，我在可以扔到垃圾桶，再去买个合适的。可就是咽不下这口气。东西不值钱，可我花进去的时间却不少；脑子里翻来覆去想这件事，也浪费了不少心力。而且，一旦决定刚，心绪上很难再软下来，就算是维护自己的面子、或是决定的正确性，也不能软。

只是，回头来看，这是非常非常不划算的。纯粹从理性的角度上来看，也许，不那么计较，放过别人，也是放过了自己。

0823 - 平衡

发表于 2020-08-23 | 分类于 每天写一点

接着昨天的说。

做开发的人，应该都有这种感觉：学新技术，学的时间长了，就没兴趣了、不想学了，至少没有刚接触时的新鲜劲。就像是一个杯子，装满水后，就没办法再装更多的水。

这时，如果用新技术做了个新产品，那就不一样了。随着产品研发的深入，对新技术的掌握也提出了更多的要求。发现了更多自己不会的点、更多需要解决的问题，就又重新有动力去学习了。

信息的接收和输出也是一样。每天刷手机，看各种新闻。看到最后，就想吐了，再好看的也看不进去，就算看抖音这种快节奏的东西也要快进。就是接收的东西太多了，已经装不下了。

这时，如果进行一些输出，就不一样了。比如，刷半天的新闻，就所看到的东西写篇文章，这就是种输出。写文章时，就需要反刍所接收到的信息，可以还需要多方查证，或者补充。输出之后，再去接收新的信息，就容易多了。

多输出，既是帮别人，也是帮自己。

0822 - 内求、外求

发表于 2020-08-22 | 分类于 每天写一点

交易，看起来是想将市场上的钱放自己口袋里，是外求。其实更多的是和自己内心争斗，是内求。交易的方法很容易学；甚至不太聪明的人，反而学得更好。可真到了交易实战中，理性这个看门人，往往压不住非理性在内院放火。搞定了内在，才有可能偶尔赚顺应市场的钱。

而如果一直是内求，则很可能变得狭隘、短视、自负、易怒、简单粗暴、以自我为中心、等等。总之就像内分泌一样，一旦失调，各种毛病。

作为平衡，就要对外输出。说是输出，其实还是为了换回自己内在无法生产的东西，就像国际贸易一样。比如，写东西给别人看、做出产品售卖，看似是为了赚钱或别的，但更重要的是收获其他的东西。比如，解决用户问题时带给自己的满足感；用户向自己表示感谢时，自己内心的喜悦；看到自己的产品频频出现在媒体上，自己内心的虚荣得到满足；等等。这些都是钱买不到，冥想也想不出来的。

内外兼修，自在平衡，方得始终。
