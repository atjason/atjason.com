0704 - 被迫补齐短板

发表于 2020-07-04 | 分类于 每天写一点

在社会上立足，主要还是靠自己的长处，为自己赚取足够的财富和社会地位。

但仅靠长处，并不是充分条件。对于自己的短处，也不得不进行补齐。

比如，再不喜欢处理人际关系的人，也需要在社会中和不同的人打交道。

拔高来看，在我看来，这其实是分工不深入的表现。从社会整体上看，这样的模式，并不能最大化发挥价值。就像人的知识传递方式一样（寿命只有 100 年，却需要近 20 年长身体和学知识），是整个人类的短板。

说到知识传递，突然想起手机电池。也是网上看到的，说最近的小孩觉得手机充电速度慢，说要是能直接换电池就好；两块电池替换，备用的在手机之外充电。

作为手机方面的老人，我笑了，多年前不就是这么干的吗；有点轮回的意味。不过，充电速度再怎么快，也还是不如换电池。不知道人类这种充电式的知识传递方式，将来会不会被换电池的方式取代。

0703 - 交易与赚钱

发表于 2020-07-03 | 分类于 每天写一点

比如玩德州扑克，当习惯了 10/20 的盲注，就很难再退回至 5/20。玩得越大，越难再回退小的。其实不止是扑克或者赌博或是其他事，都是一样道理。比如，画家画了一幅成名画之后，下一幅画什么呢？已经被先前所绑架。

交易是为了赚钱吗？也是，也不是。

自然是要赚钱，不然干嘛去交易。可，如果只是将赚钱作为目标或者说动力，更大可能是亏钱。因为交易中有赚有亏，就像花开花落。如果只以赚钱为标准，一旦遇到亏损，则很容易心态崩坏。心态一坏，不能遵守纪律，结果完全是未可知。模糊的确定是，赚赚赚，最后一把爆仓。

那交易是什么？交易是不断寻找、优化交易计划，然后坚决执行。盈利或者亏损，只是其结果，是用来优化策略以进行下一次交易的反馈。

交易就是交易，能把「我」从交易中摘出来，便入了一定境界。

0702 - 确定性

发表于 2020-07-02 | 分类于 每天写一点

不管是交易，还是生活，都在追求确定性。

市场，本身是混沌的，而混沌自然是无法交易的。硬要交易怎么办？就是硬在混沌中找秩序、找确定性。你「确定」接下来一段时间会上涨，于是做多。当然，你的这个「确定」可能是错的，于是亏损。可又能怎样？如果不假设「确定」，交易根本没法做，做就是赌博碰运气。

生活也是一样。比如说可怕的癌症，在做完检查到结果出来之前的等待阶段，是最难熬的；那种心理的煎熬，直让人崩溃。等结果真的出来了，哪怕是确诊了，心里也会有解脱的感觉；该治疗治疗、该干嘛干嘛。

从一个确定到下一个确定，交易和生活，就是这么向前推进。

0701 - 一花一世界

发表于 2020-07-01 | 分类于 每天写一点

让我们来看一朵花。

花怎么来的？树上结的。树的生长，又离不开光、水、风等元素，是和周围的动植物相互影响，才成了现在的样子。树是怎么来的？从另一颗树的种子来的。另一颗树又是怎么来的？

这样的问题，可以一直问下去。到最后，会发现，世界上所有事物相互影响，前因后果相互作用，才有了现在看到的一朵花。所以说，从一朵花中，可以看到整体世界。

佛说里的「空」，并不是没有的意思，而是「无自性」，说人话就是不能独自存在，需要和其他事物相互关联而存在。比如，「四大皆空」，可以简单理解为，「地、水、火、风」四大物质元素，相互调和依托，构成了整个世界。

我说的对吗？不一定。比如，「一花一世界」，也可以理解为，对于花中更小的元素，比如细菌，花就是它的世界，是它的地球。

需要对吗？不需要。佛，讲的是觉悟，是证道，是自己的事。并没有标准答案，而是自己向自身自省，去理解和感悟世界的真谛。

0630 - 少谈形而上

发表于 2020-06-30 | 分类于 每天写一点

少谈形而上，务实不务虚。

高屋建瓴这种事，当然好；只是可惜，我等俗人，大多无此境界。

比如，宇宙是有限的还是无限的，人性本善还是本恶，甜豆浆还是咸豆浆好喝，等等。这样的神问题，茶余饭后聊聊就好。
