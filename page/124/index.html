1109 - 储能的技术路线

发表于 2021-11-09 | 分类于 每天写一点

目前的储能方法，传统抽水储能依然占大头，大概 90%。不过，新兴的电化学储能发展很快，占比越来越高。

具体用的是什么呢？锂电池。为什么是锂电池，是它的技术最优、最适合吗？不见得。主要是电动车最近发展的太火了，太多的人才、资源投入这个领域，加速了锂的进化。核心的一点，成本大幅下跌。从最开始到现在，可以说下降了 90%。于是，锂电成了储能目前主要的选项。

而动力电池和储能用的电池，需求差异还是很大的。简单的说，车里的电池需要移动，就需要充电快、能量密度高（体积小、不那么重）。而电站用的储能电池，其实并没这方面的需求。电池大一点、重一点、充电慢一点，其实问题并不大；关键是要便宜。

等到锂电的进化空间、也就是性价比提升空间越来越小，大家会更有动力去研究其它的储能技术，比如钠电池、钒电池、氢能源等等。

1108 - 开放才有大格局

发表于 2021-11-08 | 分类于 每天写一点

现在满大街跑的是新能源车，都是锂电动的。

那新能源就只有锂这条路吗？明显不是。还有一条明显的技术路线：氢能源。

事实上，日本在这方面非常领先，世界上已经有很多氢能源车在路上跑了。

为什么目前看起来，氢能源输了呢？其中有一个很重要的原因：日本太领先了。关键是，只是自己跑得快、跑得远，手里握着绝大多数的专利。这种情况下，谁来陪你玩？陪你玩，结果钱全是你赚走了，那我图什么。

于是，大家被迫在其他赛道找机会，于是就找到了锂。正好在方面，专利垄断没那么严重，理论上大家都有机会。更多的人才、资源参与进来，使得锂这条赛道，迭代得非常快，也导致了目前的局面。

开放，才能把蛋糕做大，自己也才能真正的获取更大的利益。利他，才是真利己。

1107 - 行业知识树

发表于 2021-11-07 | 分类于 每天写一点

最近看了很多行业研报，对风光锂等行业有了树级别的认识。

之前看交易相关的公众号，除去惊悚的标题和灌水的内容，其中有价值的点并不多。并且，这些点比较零散，无助于形成系统的认识。

研报则不同。一些深度研报，其功能之一，就是分析这个行业完整的产业链、分别有哪些环节、又主要有哪些企业参与。有了这些信息，就有了一颗知识树。

有了这个基础，再去个公众号，就很容易对号入座。一方面，会发现这些公众号的内容比较浅，通常也没介绍多少新的东西。另一方面，还是有点用的，主要是介绍行业动向；像是鼻子，替自己捕捉变化。

尝到了这种方法的甜头，下周准备再研究几个行业。
