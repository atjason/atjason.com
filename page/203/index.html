1117 - 没人喜欢被胁迫的感觉

发表于 2018-11-17 | 分类于 每天写一点
  • 加这个功能我就付费
  • 没这个功能我就去用 XXX 了
  • …

其实，作为开发者，第一眼看到这样的要求，其实是不爽的。

其实，任何人，第一眼看到别人这种明里暗里的要挟，都不是爽的。

没人喜欢被胁迫。

只需坦诚地表达，等待对方的反应即可。过多的威胁加持，并不会让得到自己想要的结果的概率提高，而只是让人感觉幼稚。

1116 - 放弃一夜暴富的心态

发表于 2018-11-16 | 分类于 每天写一点

尤其，见识过、甚至经历过一夜暴富的人，要放弃这种心态、重回慢慢苦逼积累的心态，很难。

可，又有几人是一夜暴富？更多的人是一夜回到解放前。更靠谱的，不正是慢慢积累。这一过程必然是漫长的，只是期间会有些让人激动的小成果，激励着我们继续前行。

简单的话，忘记每年 100% 的增长，重新「看得起」每年 20% 的增长。能始终保持后者，结果也将是惊人的。

1115 - 带着问题睡觉

发表于 2018-11-15 | 分类于 每天写一点

睡觉前，脑子里想一个近期一直在想、却一直还没有满意答案的问题。

睡觉时，潜意识也会继续运转，说不定会有意想不到的点子。

据说，一些大人物们小憩时，手里会拿个球之类的物事。一旦球跌落、自己就惊醒，便记录下半睡半醒那一刻的想法。
据说，这种方法诞生了很多天才想法。

1114 - 仪式感

发表于 2018-11-14 | 分类于 每天写一点

最近几天，接连把自己的 App 升级了一遍；今天，更是提交了 3 款 App…

等正式下架并做简单运营推广后，正式切换到另一个项目中。

不只是做事，需要一点仪式感，需要在不同事情间有明确的切割、转换，这样能更好地投入到新的事中。

1113 - 有的聊

发表于 2018-11-13 | 分类于 每天写一点

跟别人聊天，是越聊越起劲，还是把天聊天？

除了说话及倾听技巧，很重要的，是自己有没有可以说的东西。

如果总是觉得很干瘪、除了自己的工作就没什么可说的，说明自己的生活真的太无趣了。

把生活过的，有的聊。
