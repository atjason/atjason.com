0217 - 折腾了 Paddle

发表于 2021-02-17 | 分类于 每天写一点

事情是这样的，国外一个媒体联系到我，想要推广 Klib 的时候有点返佣。

这个，虽然目前 Klib 支持序列号，但还不支持 Affiliate 和 Coupon。本来想算了，放弃这个机会。后来想想，要么试下吧。

功能上，主要是这几部分：

  • 国际支付，包括接受信用卡、PayPal 等方式付款，最后提现到银行卡。
  • 用户付款后，自动发送包括序列号的邮件。
  • Affiliate 联盟机制，以便统计推广者带来多少付费。
  • Coupon 优惠码机制，让推广者多一点吸引用户付费的动力。

比较了 Stripe、Paddle、Gumroad 等方式，包括功能支持、费率等因素，最后选择了 Paddle。其实是用了排除法：Stripe 不支持国内用户注册，Gumroad 有过随意封号的黑历史，直接不考虑了。

花了几个小时集成 Paddle，其实还是挺简单的。唯一的麻烦，是在创建 Affiliate 时，一定要有个自己网站的落地页。其实，我不太理解为什么一定要这么做。在我看来，直接在付费链接上加小尾巴，应该也是能实现的。不管怎样，我实现的落地页，是这样的：http://klib.me/pro/

序列号方面，只要上传一个文件，每行一个序列号即可。用户购买后，Paddle 会自动依次提取序列号并发送。需要提醒一点：Paddle 并不支持删除已经上传的序列号；所以，不要上传无效的序列号。如果已经上传了（比如像我），可以创建一个完全免费的 Coupon，然后用这个 Coupon 去消耗掉无效的序列号。

目前，由于还没有真实的付费，尚不清楚考虑到税之后实际收到钱的比例，以及提现到 PayPal 会不会再次收费。

总之，目前这个机制已经完全实现了。如果你有国外的软件推广渠道，欢迎联系我。具体的软件，推荐 Klib、iText、iPic。

0215 - 交易不是技术

发表于 2021-02-15 | 分类于 每天写一点

交易是不是艺术，我不好说；但交易一定不是技术活。

所谓技术，通常是线性的。所谓线性，就是投入产品成简单的正比。比如愚公移山，虽然很难，但只要世世代代坚持下去，总有移走的一天。

而艺术则不是这样的。比如画画，画几笔容易，但要画到能卖出价钱的画作，难。并且，并不是你坚持画，就一定能突破那个瓶颈。

交易更像是艺术。学习，一定能提高交易的水平；但，绝不能保证盈利。努力学习，最多到及格的水平。要成为优秀交易员，是需要天赋的。

接受自己不适合交易，不容易。退出交易，更难。

从老师的角度，当然希望自己的每一位学生都成才。可事实上，再好的老师，班里也有第一名和最后一名；考试成功，也不是每位学生的最佳出路。

0214 - 享受当下

发表于 2021-02-14 | 分类于 每天写一点

过年嘛，买了罐氦气、打个气球、系了绳子，娃拿了很开心，玩了很久，感觉可以在小区保持好几天的优越感。

可惜，好几天还没到，气球在娃的略暴力之下，绳断、飞向天空。在那一刻，去责怪孩子没看好气球、白白浪费还值点钱的氦气球吗？

不不不，当然不是。既然已经飞上天空，就指着气球对娃说：看，气球飞多高，我们看看气球能飞多高好不好？于是，娃更开心了，蹦蹦跳跳起来。

每个当下，用最好的方式对待。

0213 - 口腹之欲

发表于 2021-02-13 | 分类于 每天写一点

过年嘛，都闲下来了。不交易、不代码、不学习，整个人都松散下来了。

那干什么呢？无非是吃吃喝喝，看看电影视频。

这样的事，当时是舒服的，毕竟分泌多巴胺。可，实在是不持久；过了那一刻，爽的感觉立马消失。甚至，随着时间的流逝，开始有点空虚空洞的感觉。
