0128 - 房价最多跌多少

发表于 2021-01-28 | 分类于 每天写一点

房价仿佛永远在上涨。可是，有没有想过，如果房价跌了会怎样？跌多少会出大问题？

简单的说，取决于杠杆。

大部分人买房，都用到贷款，这其实就是杠杆。比如，常见的首付三成，贷款即是 70%。

  • 假如，房价下跌 30%；房子的价值，和银行货款是相同的，即原先的 70%。这时，不考虑住房学籍等因素，仅从钱的角度考虑，还不还贷款，是一样的。
  • 假如，房价下跌 30% 以上，房子价值，已经低于银行货款。换句话说，还房贷，还不如违约，让银行把房子收走。当然，参考上一条，房子还有居住、学籍、户口等附加属性。但，如果实在下跌太多，超出了这些附加价值，一样会出现大面积违约、不还贷款的情况。

一旦发生大面积不还房贷的情况，将引发雪崩。银行不得不收回房子，然后对房子进行拍卖。问题是，拍卖的房子越多，房价掉得越厉害；进而，不还房贷的人更多。这是一个加速的、螺旋向下的过程，就是雪崩，就会引发严重的经济危机，具体请参考 08 年经济危机，差不多是这么个意思。

简单的说，买房首付的比例，就是房价绝不能跌穿的底线。

0127 - 数星星、数月亮

发表于 2021-01-27 | 分类于 每天写一点

股市里，很多普通交易者，都希望能找到某一支好股票，将来大涨。

这当时是可能的；只是，太难了。就像是在满天繁星中，找最亮的那一颗。要知道，最亮的不一定最大，可能只是离得近而已。

相反，股指 ETF 就在那里，各行业 ETF 在那里，各行业龙头也在那里。为什么，要放着这些月亮不管，去寻找星星呢？绝大多数普通交易者，只是普通人，没那能力，找到最亮的星星，获利超额收益。相反，亏损的才是大多数。

认识到自己是普通人，数月亮。

0126 - 开始刷题

发表于 2021-01-26 | 分类于 每天写一点

我是喜欢编程的。

可惜，目前把大部分精力投到了交易上；又怕长期不写代码，手生了。怎么办呢？想了一下，开始刷题吧。

跟做编程教育的朋友简单了解了下，现在刷题大致有两个网站不错：洛谷、LeetCode

怎么选呢？我有一个比较大的限制：想使用 JavaScript 进行代码。又简单了解下，貌似两个网站都支持。不过，从界面上看，明显 LeetCode 支持的更好。再加上，LeetCode 支持英语；代码的同时，也保持对英语的熟悉。恩，就这么愉快的决定了。

刷的代码，放到 GitHub 上，既是存档，也当是对自己的激励：https://github.com/atjason/leetcode
