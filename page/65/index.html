0120 - 尊重交易

发表于 2022-01-20 | 分类于 每天写一点

之前，我一直觉得交易没有意义；只有交易的日子，回忆是一片空白。

这种想法是错误的。

交易，可能带来可观的收益。这些收益，可以使我的家人有基本的物质保障，可以让我有时间精力去创造软件产品。

交易，对我的生活是如此的重要，我要尊重交易、感恩于交易。

0119 - 丰田 All In 纯电动车

发表于 2022-01-19 | 分类于 每天写一点

之前，丰田在氢燃料车上投入巨大，也极为领先。

可是，因为太过领先，格局不够大，专利什么的都在自己手上，别的玩家没办法参与，大家转而投向纯电动车。在资本和人力的加持上，纯电动车辗压氢燃料车。

现在，丰田的转型，相当于一个中年富态男，放下身段，和一帮小年轻同台竞技，难。之前多年重金在内燃机车积累的优势，不仅可能不再是优势，反而可能是转型的包袱。有巨大开支、又需要向纯电动车投入巨额研发费用的同时，还需要传统车提供资金流。

一家企业，又或者具体到个人，如果在接受新事物、利用已有东西获益之间平稳，不容易。既要砸掉旧饭碗，又要在砸掉之前找到新饭碗。

0116 - 分析与计划

发表于 2022-01-16 | 分类于 每天写一点

分析与计划：

  • 交易前，分析是基础；
  • 交易时，计划是全部。

分析体系有共性，所有人都可以学习统一的标准。

交易体系是个性化的，每个人的性格、偏好、风险承受能力等不同，要基于这些，做适合自己的交易计划。

  • 分析体系是了解市场；
  • 交易体系是了解自己。
