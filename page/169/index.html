0710 - 不要因为忙而不学习

发表于 2020-07-10 | 分类于 每天写一点

如果你不学习的理由是太忙，那很可能你一定比例是在瞎忙。

说到我自己。如果我的理由是，最近的机会太好了，我要抓住机会多交易，趁这波多赚点。那很可能，我已经过度交易了。不仅抓不住机会，反而可能亏钱。

抽象来说，我们是在用旧知识解决新问题，也可以说是凭经验做事。但明显，旧知识或者说经验是会过时的、有缺陷的。只有不断打磨完善，才能更好的解决新问题。

另外就是心态。能静下心来学习，说明本身心态就不错。而学习的过程，会让心态更好。就像是，知道自己不知道，于是去学习；学习之后，能力圈扩大，接触到更多未知的东西，才发现有那么多自己不知道的东西。在这样的情况下，怎会不谦虚谨慎。

而谦虚谨慎，对交易来说太重要了。

0709 - 不要着急进场

发表于 2020-07-09 | 分类于 每天写一点

做交易总是会遇到这样的情况：听别人说、或者自己感觉发现了机会，机不可失，再不进去就来不及了。于是，立即开仓。然后被套，由小亏变大亏。

经验来看，上赶的不是买卖。一旦你着急，就落了下风。

一旦你也陷入了这种状态，离开交易界面，自言自语几遍：慢就是快；以后机会多的是。

更靠谱的状态是：就像猎人狩猎一样，事先找到猎物可能出现的区域，然后耐心等待。发现猎物后，不是立即开枪，而是先锁定。直到最佳射击机会出现，一击毙命。

0708 - 不冒风险，也是风险

发表于 2020-07-08 | 分类于 每天写一点

有农村，有个很典型的现象：有钱的，都是做生意的。

农民嘛，本职工作就是种地。把地种好，从地里刨食，这自然是风险最小的。

可问题是，收益也小。综合下来，投资回报比小。

如果做生意，前期就会有很大的投入。而且，产品做出来之后，销路好不好，能不能收回成本、进而赚钱，这都是风险很大的事。

可问题是，潜在的收益巨大。综合下来，投资回报比大。

中国人大部分比较保守，更愿意守着一亩三分地过日子；保守，也意味着风险厌恶。可，风险和机会是并存的，要想抓住机会，冒风险是必须的。没有基因，也只能逼自己一下。

激流勇进，不进则退。不冒风险，相当于自己保持不动。可自己不动，会被超越，被时代向后推。

0707 - 积极一点

发表于 2020-07-07 | 分类于 每天写一点

在一个交易群里，老师讲解了一个技术点。内容比较多，老师说看群友谁有时间整理一下。正好我对这个点感兴趣，就整理了 PDF 文件。

整理的同时，被动的去了解了其中的交易标的：天然气期货。之前，我是完全没了解过的，有了这次了解，再加上老师说的最近可以关注下，于是真觉得是个机会，就准备尝试做一把。

有趣的是，这几天又爆出巴菲特大举进入能源领域的新闻，算是上涨的佐证。

回到事情的起点，如果我不是自己主动的去整理，也就不会了解到这个交易标的，进而也不会研究最近的交易机会，极可能就错过了。

积极一点，时不时会有意外收获。

0706 - 股市与段子

发表于 2020-07-06 | 分类于 每天写一点

今天 A 股大涨，听到了很多段子。

比如，经典的大妈理论：大妈不动我不动。

比如，问：当办公室同事问我股票如何开户时，股市是不是到底了？答：不是，要等到他向你推荐股票。

段子说完了，还是要冷静一下；要看到看不见的，看见未来。眼下是情绪牛，肯定还会持续很长时间。可很长时间之后呢？接下来肯定会有类似融资炒股、借货炒股、卖房炒股、加杠杆炒股等投机行为。风一直吹，大家都开心；风一停，沙石先落，然后是一地鸡毛。

这一过程之后，必然伴随着财富的重新分配，乃至阶级的重新划分。坐着不动的人，会被坐火车的人超过；坐飞机的人，可能会坠机；总之，什么情况都会有。

即便金融有风险，没办法，现在已经被金融绑架了。再想老老实实干活、踏踏实实赚钱，也要学点金融知识、参与金融行为，不要被高维打击。
