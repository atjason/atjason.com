0210 - 重复犯相同错误

发表于 2021-02-10 | 分类于 每天写一点

人啊，总是在犯相同的错误。

虽说，吃一堑、长一智。但，也许要反过来理解；如果真的能做到这一点，也就没这句谚语了。就是因为不容易做不到，才发明了这句话，以鼓励大家。

比如，在辅导娃作业时，看到娃重复犯相同的错误，通常怒不可遏：你怎么这么笨、怎么不长点心、怎么这么没记性？这时，安慰下自己，人不都是这样吗？哈哈。

0209 - 花钱请人打扫

发表于 2021-02-09 | 分类于 每天写一点

这两天打扫卫生时，在想一个问题：如果花钱请人来打扫，会怎样？

我想了下，其实我还是愿意自己花时间打扫。

0207 - 试错

发表于 2021-02-07 | 分类于 每天写一点

比如交易，看到一个突破行情，想进去做多，又怕是假突破，被止损。怎么办？试错呗。先用小仓位做多，对了就加仓，错了就止损呗。既不怕错过行情，又不会大亏。

比如做事、做产品，想做，却又怕投入后没产出，投入打水漂。怎么办？精益创业呗。先用最小的代价，做了最小原型，投放到市场，让用户去试。然后，观察用户的反馈，再决定是继续加大投入，还是就此打住。

发现没，道理都是相同的。不确定的事太多，想要又怕，只能试错。试错的代码，是必须承担的。关键的，是成功与失败的概率和盈亏比。

0206 - 怎么劝别人

发表于 2021-02-06 | 分类于 每天写一点

我不知道怎么劝别人。

一种是，别人的问题，我知道怎么解决；这事放在我身上，我也有能力解决。那么，我通常也只能把我的解决办法告诉对方。但，这也并不是劝。

另一种，就是我自己也搞不定，就更不知道该怎么说了。尤其，通常别人的问题，都是依赖于我和对方之外的第三方，这就更加不是我所能控制的因素了。

求助的对方，通常，不仅需要解决办法，也需要情感上、心理上的安慰和支持。有了这些，他们自己内心安定了，所想出来的办法，可能更好。因为，毕竟他们置身事中，更了解全貌。
