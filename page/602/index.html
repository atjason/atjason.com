日本之旅 - Day 0

发表于 2015-06-18 | 分类于 旅行

9:30到大阪机场，例行的下飞机、摆渡到海关、出海关、拿行李。第一反应是确实能猜出不少日本字，因为和汉字很像，比如“到达”是“到着”。

过海关的时候，听到的日本人的吵架声，隔的比较远，像是日本人和海关工作人员吵起来了。这就打破了书上说的日本人非常讲究公共场合礼仪、不影响他人的印象。

入关时，包里还有个苹果和芒果，一直担心要不要扔掉，因为入境单上写着要保证不带水果。结果拿行李时，只要把单子给他们就好了，一秒钟通过。可能托运的箱子扫描过了，可是背包没有啊？看来严谨的日本人也有放水的时候。

下车时人生地不熟，要转两次电车，其中还要在地面走一段，最后连蒙带猜才找到了，不容易啊。

这个机场是建在海上的。出来坐电车要先过海，这个比较好玩。可惜是晚上，看不到外面的景色。

不太理解日本为什么还有这么多电线杆，在地下通过管道走线会不会更好

more

电车自助售票复杂到爆。电车不新，但还挺干净。低头看手机的明显没有国内多，有不少发呆的，也有看书的。关键报站只说日语，英语都没有，完全听不懂，只能看地图（后来换了一条线，有英语报站）。车厢和车厢间是不通的。座位是软席的（急行车，也就是普快，不是特急车）。还有女性专用车厢，不是为了女权，为了什么大家想下喽。

酒店房间真心不大，一个箱子放床边，就走不过去。卫生间，就是飞机的卫生间加上小孩用的浴缸。马桶可以喷水，人生首次体验，这感觉…

房间里有应急灯，平时一直插在床头，拔下来就亮，关都关不了，估计是地震训练出来的。

酒店有热水壶，贴心啊。比欧美好，只有免费冰块。房间里有小冰箱，可是里面空空如也，想喝水都不行，没中国人会做生意，还是不想给客人带来困惑？

床挺小，大约一米二，睡两个人，都有点不习惯了。

困了，累了，睡了

网站从Linode搬家到Amazon

发表于 2015-06-07 | 分类于 IT

StartFragment

为什么搬呢？

目前用的是Linode，对于我这种只放个博客的人来说，绝对是高大上。每月$10，不便宜。之前速度还挺快，现在感觉也一般般，明显不如之前，继续使用性价比不高

另外就是之前买的VPN快到期了，而且感觉变得不靠谱，经常断线。同时养VPS和VPN，感觉有些浪费，干脆合二为一吧

关于VPS的选择

先明确需求

  • 速度快；这是基本要求，宁可浪费钱也不能浪费生

  • 稳定；别过几天倒闭

  • 能换IP；为了架设VPN，万一IP被黑了，你懂的

到网上大致搜了一圈，选择真多啊。最后在DigitalOcean和Vultr中选择，简单测了速，当时的网络下DigitalOcean稍好些，就选择了它；另外当然还有免费一年的Amazon（吐槽下：亚马逊入华都入了N年了，还没全面开放）。也就是说，在DigitalOcean和Amazon中选择，二者都选择新加坡节点

另外，Lowendbox有专门对VPS进行介绍的文章，有时间可以去淘

more

速度测试

主要是两方面的测试：在VPS上下载东西、从VPS上下载东西。

注意：测速要看长性。简单的说，就是要下载比较大的文件，比如700M。因为有些VPS服务商会有机巧，刚开始下载时会很快，后来会越来越慢。如果只下载很小的文件，可能会被忽悠。

价格比较

Amazon免费一年、DigitalOcean每月$5，那就选Amazon呗。不过，一年后Amazon相同实例每年最便宜为$128，和Linode年付的价格差不多。不过也不一定，一年后降价也说不定，先用着吧，大不了到时花一天搬家

性能测试

这点要求不高，能跑WordPress就行。登录WordPress后台，随便写几篇测试文章，差不多顺畅就行。

博客搬家

这里就要说到AMH，国人写的主机面板，我用的依然是4.2的免费版本，在安装好后，搬家不超过10个命令，简直是不搬家都对不起这么方便的搬家啊

域名使用DNSPod进行管理，更改域名对应的IP很快就生效了。不过，域名的全线更新需要些时间，保险起见老VPS都多留几天

后记

时隔一年又来折腾后台，还是挺感兴趣的，但没有以前那么起劲，毕竟过了折腾的季节。现在更重要的是多多的博客上积累，学到东西、个人成长才是最重要的，花点钱搞个稳定的VPS，节约下来的时间要比花的钱划算得多。

2015年第9本书《远程工作》

发表于 2015-06-05 | 分类于 读书笔记

看过这本书，都不想去公司上班了。

远程工作，也就是说大家不用专门去一个叫“办公室”的地方上班；相反，可以选择在家、咖啡馆、图书馆等地方办公。毕竟，对于大部分智力工作者，一台电脑、无线网络就够了。

远程工作，有诸多好处，比如不受打扰、可以专心工作，比呆在一个地方更容易产生灵感，更容易聚集优秀的人才，不用浪费通勤的时间，可以更好得平衡工作和生活，等等。

当然，也有很多问题和风险，比如缺少情感交流，复杂的问题面对面沟通更具效率，不适合自制力差的人，需要改变管理方式、等等。

总的来说，个人比较喜欢，也认为这是趋势，至少以后不会像现在这么小众。就像在PC时代，人们无法想像、也不能接受在手机上买东西，但现在在手机上买东西可能比在电脑上还多。而且毕竟以后个性化会越来越突出，大家越来越希望有自由、丰富的生活方式，自然也包含工作方式。

期待早日能体验这种听起来就很爽的工作方式。

最后：

这本书出自很喜欢的37signals团队（现已全线更名为Basecamp），英文名为REMOTE，国内还没有上市，是在简书上看到别人的翻译，并推送到Kindle上看的。
