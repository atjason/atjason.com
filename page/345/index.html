0103 - 每接触一个新概念，就像打开一扇窗

发表于 2016-01-03 | 分类于 每天写一点

概念，是我们认知世界的基础。我们不断学习和掌握新的概念，并将其压在脚底 而成为概念地层；进而以此为基础，继续去接触和学习新的概念，不断循环。

喜欢刚接触新概念时的感觉。感觉一切都是新的有，有无限的可能，让人感觉很有活力。

0102 - 远程工作

发表于 2016-01-02 | 分类于 每天写一点

找一个晚上可以看到星星的地方，远程工作。不用通勤，无需忍耐堵车、或雾霾。春暖花开。

仔细想了下，如果一年都在这里，可能不现实，因为还要考虑将来的医疗、子女教育等问题。但如果只是每年找个地方远程工作几个月，应该还是可以的。除了远程工作，还可以有机会亲近大自然，钓钓鱼、爬爬山、跑跑步、骑骑车、种种菜什么的，都是挺好的。

恩，争取未来几年内可以有一次这样的生活体验。

0101 - 勇敢地迈出第一步

发表于 2016-01-02 | 分类于 每天写一点

凡事开头难，这包括实际做事层面，也包含心理层面。

最近在旅游，有一段路交通不便，其实是说公共交通不便，还是有很多来往的私家车，于是就想要搭车。可是从来没搭过车啊，心里没底，也很不好意思在路边伸手搭车。当鼓足勇气、伸手搭车时，很意外地一次成功。车主是位年轻人，也算是本地人吧，正好带朋友来这里旅游，也正好和我们的目的地相同。总之就是很顺利。

在做事之前，设想了各种可能性，也包含失败后的退路。可踏出第一步后，也许会发现事情并没有想象中的麻烦，去做就好了。

1231 - 要想富，先修路

发表于 2016-01-02 | 分类于 每天写一点

最近，杭州到丽水的高铁刚刚开通，于是去丽水及周边玩了下。另外，当地政府为了刺激旅游，还推出凭高铁票免门票的优惠活动，明显看出增加了游客人数。

丽水这个地区多山，这点从来时的路上就看到了，等看到了山顶，更是看得清楚。如果没有便利的交通，大家很难到达这里。没有人，也就没什么可能性，包括旅游，或者其它经济形式。

最重要的是把人连接起来。这种连接，除了虚拟连接，比如互联网、电话，实体连接也非常重要，比如物流、交通。基础设施的完备，给上层的游戏带来了无限的可能性。

1230 - 劣势，也可变为优势

发表于 2015-12-30 | 分类于 每天写一点

最近在设计东西。由于自己没有什么设计功底，做出来的东西都很简单。可正是由于这种设计上的简单，有意无意的让产品变得简单，算是因祸得福吧。

所以，也并没有绝对的劣势，就是看把心思和能力花在什么地方。为自己的能力找到适当的发挥渠道，就可以扬长避短、变劣势为优势。
