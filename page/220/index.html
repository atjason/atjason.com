0408 - 春天，共享单车的春天

发表于 2017-04-08 | 分类于 每天写一点

今天骑车出门，发现杭州共享单车已经挺多了。即使是在高峰期，路边还是能看到不少单车。尤其，路过黄龙体育中心时（当时有演唱会），原本停着杭州「小红车」的地方，已经没有车了，而共享单车还有。

在我看到的单车中，感觉「Hello Bike」是做工最好、看着最顺眼了。

不过，好骑不好骑我就不知道了，因为可悲的是，我试了「Hello Bike」和「小鸣单车」，都失败了：一个总说验证码失败而无法登录，一个押金已经扣款了、App 内还显示没有押金。看来大家 为了「快」，太「糙」了。

0406 - 为什么有些人连提问都不会？

发表于 2017-04-06 | 分类于 每天写一点

比较典型的，会遇到用户这样的反馈：

xx 失败，怎么办？

你让我怎么办？

最基本的，你总要描述下 做了什么操作、遇到什么结果、有什么提示、等等，这样我才能知道发生了什么、问题可能出在哪，才有可能帮到你。

真的很难理解，为什么连基本的提问都不会，是饭来张口习惯了吗？总要在别人不断的追问下，才能把问题描述清楚，真真是自己理解、表达能力的缺失。

另外一种陋习就是，上来先问：在不？凭空多了一次回复和等待的时间；况且，我得看你说什么事、然后才能决定在不在呀？想起 Google 的视频 App，接收方能看到发起方当前的视频，这样就能大致判断出对方视频的意图，对于是否接受视频是很有帮助的。

0405 - 睡后收入

发表于 2017-04-05 | 分类于 每天写一点

好吧，今天又是套牌车…

「睡后收入」，也即「被动收入」的另一种说法，比如：

  • 在睡觉的时候，银行里的钱在增加
  • 在旅游的时候，银行里的钱在增加
  • 陪家人的时候，银行里的钱在增加
  • 在…的时候，银行里的钱在增加
  • …

说白了，就是能 构建一种能自动运行、无人值守、无需客服、自动盈利的商业模式，这怕是自由职业者的终极目标吧。
