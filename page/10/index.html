0204 - 停下来，想一想

发表于 2021-02-04 | 分类于 每天写一点

我发现，这个小技巧，对我很有用。

比如，之前独立开发产品时，尤其是项目早期和中期，有很多事情要做，千头万绪。这时候，很容易陷入其中。好像什么都要做，却又不知道先做什么。

这时，我会停下来了，稍微想个 7、8 件事。然后，从中挑选中最想做、或者最必要做的前 3 件事。挑出来之后，就不再想任何其他的事，把这 3 件事做空再说。

  • 一方面，这可以让我集中注意力对付这几件事，不受其他事的影响。
  • 另一方面，随意这 3 件事的完成，对整个局面，很可能会有更清晰的认识，更有利于接下来的安排。

现在做交易，也是类似的。当行情走到一定程度，或者刚刚平仓后，我会停下来，从更高的视角，审视接下来的行情会怎么走。至少做正反两方面的推演，当然也包括我最希望看到的行情是什么；以及，不同行情下该如何应对。

然后，把这些判断显式的记下来。可以用纸笔，也可以在电脑上。总之，让它们脱离大脑，成为一个实体的存在。

然后，随着行情的演进，新的行情和自己之前的判断进行比对。比如，恰好和自己最期望的行情匹配，勇敢的按计划操作即可。行情是很随意的，如果没有事先的预演作为对照，自己很容易陷入行情的无序波动中。

当然，这样的预演，不必太长太久，那样通常会走样，和事实差距太大而变得没有意义。只需要理出最可能发生的 123 即可。如果自己的推演都没有命中，停下来，再想一次。

0203 - 不要羡慕

发表于 2021-02-03 | 分类于 每天写一点

比如，我很长时间没开发出自己的软件产品。当看到别的独立开发者，又出了新产品；或者是，已有的产品，又被 App Store 推荐，或有其他成就时，难免心生艳羡。

比如，我还在学习交易，还是亏损中。当看到别的学员，已经达到稳定盈利，甚至盈利颇丰时，难免心生艳羡。

羡慕是种很不好的情绪。这是病，得治。

既然自己选择了新的路，就不要对过去有太多依恋。随着时间的推移，对于过去的事，总是习惯性记得它的好，忽悠其不好的地方。

对于现在的路，必然会有困难。遇到困难，就退缩，想缩回原来的安乐窝，想象原来的窝的多美好。

以上，都是人之常情；只是，也是人之弊病。

反过来，如果自己已经有一定成就；或者哪怕没有，但知道自己走在正确的路上，有这样的成就或心态，就不会羡慕别人。因为自己知道，别人有好的，我也有别人没有的好东西。或者，即便我现在还没有好东西，那也只是时候未到；假以时日，该来的总会来。有这样的心态，何必羡慕，又何需羡慕。

既往不恋，勇往直前。

0202 - 赌徒谬误

发表于 2021-02-02 | 分类于 每天写一点

赌徒谬误：

  • 已经跌了这么久了，总该涨了；
  • 已经涨了这么久了，总该跌了。

殊不知：

  • 强者恒强；
  • 烂泥扶不上墙。

说得学术点：

  • 之前与之后发生的事，是独立的；
  • 之后发生的事，不会受之前发生的事的影响。

交易中：

  • 之前，我恐高，就怕买了被套在山顶上。
  • 现在，偏偏喜欢新高，向上海阔天空。

0201 - 看问题的角度

发表于 2021-02-01 | 分类于 每天写一点

看动物世界，狮子抓斑马。我妈不会夸狮子多厉害：即有抓斑马的能力，还有足够的耐心、良好的策略。相反，她会说：这些斑马怎么这么笨啊，还不团队；要是它们围成一圈，腿朝外，狮子来了就踢，肯定不会被吃掉。

你看，相同事物，不同视角，便会有截然不同的观点。

进而，你听到一个观点时，别直接认为这就是事物本身。一方面，这只是其中一个视角。另一方面，很自然的，每个人在表达时，都会有意无意突出对自己有利的、跳过对自己不利的。

最好的办法，是听控辩双方都陈诉一遍，基本你就知道是怎么回事了。

0131 - 互相成就

发表于 2021-01-31 | 分类于 每天写一点

有了娃之后，家庭生活几乎全部围绕着他们进行，时间也花在了他们身上。

相应的，留给自己的时间就少了。为家庭付出更多时间的一方，尤其如此。

而人，是需要有自己的独立空间、自由时间的。没有了这些，就会感觉失去了自我，好像总是在为别人而活；哪怕这个别人，是自己的子女。

如果，能时不时的从这种状态中脱离出来，完全搁下家庭琐事，享受一段独处时光，对心态的平衡，非常有帮助。经过这样的独处，既离自己的内心更近；回来之后，也更愿意在家庭、亲人身上付出。

要做到这一点，就需要家庭成员之间，互相成就。这周，我来带娃，你想干什么干什么去；下周，再交换过来。

互相分担，彼此成就。
