0529 - 卖涨持跌

发表于 2023-05-29 | 分类于 每天写一点

之前为释放资金给量化交易用，把最快最早从之前下跌走出来、回本或小幅盈利的股票清仓，比如英伟达、拼多多，留下一堆破烂，比如B站、蔚来。

结果，卖掉的大涨，持仓的大跌，像蔚来持仓已经亏损80%。

卖涨持跌，多符合人性，多惨痛的教训。这些垃圾股不准备清仓了，留着给自己提醒。

0528 - 纠结

发表于 2023-05-28 | 分类于 每天写一点

工厂呆这几天，一个很大的感受是：纠结。

比如，一个产品有划痕，影响颜值和外观，是否要正常包装发货，还是挑出来当次品？

从产品和销售的角度，当然希望挑出来。既可以保持产品好的印象，还能减少售后。

问题是，产品不可能完美，小批量生产的也不可能有非常稳定的质量。如果高标准去挑，废品率和成本会高得吓人，会这产品价格失去竞争力，甚至根本卖不出去。

你说纠结不纠结。其中分寸，实在不好拿捏。具体到一个个产品，纠结来纠结去。左看看，还是发吧，差不多可以的；右看看，要不还是算了，万一碰到较真的用户就麻烦了。

见得多了，经历的多了，自然变得谦逊宽容，因为知道各个角色的不容易。

0527 - 工厂呆着

发表于 2023-05-27 | 分类于 每天写一点

为了新产品备货，在工厂呆了 4 天，跟进生产、包装。

在生产线呆久了，发现跟在办公室的思维会差很多。

办公室离生产线远，思考方式更倾向于理想化。而在工厂，则不可避免地考虑是否容易生产、良率能否提高。

大部分产品，不可能像 iPhone 这样，有天量支撑，几乎可以不计成本去研发。即使有巨量研发投入，分摊到每个手机上也就很少了。回归到现实，只能做各种取舍和折衷。

0526 - NMOS 管与 PMOS 管

发表于 2023-05-26 | 分类于 每天写一点

当下方左侧输入低电平时，右侧 NMOS 管 Vgs = 0，不导通；进而，上方没有分压电路，PMOS 管的 Vg == Vs == 12V，Vsg = 0，不导通，无输出。

当下方左侧输入高电平时，右侧 NMOS 管 Vgs > 0，导通；进而，上方有分压电路，PMOS 管的 Vg < Vs，也即 Vsg > 0，导通，输出 12V。
