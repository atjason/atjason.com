0922 - 照单办事

发表于 2021-09-22 | 分类于 每天写一点

清单是个好东西。

在动作未成型前，想办法制作检查清单，先做什么、再做什么、什么情况下做什么。这样的方法，可以帮助自己完善思路，考虑不同的情况。

动作成型后，这样的清单可以降低大脑负担，像傻子一样执行就可以了。

0920 - 车的界限在哪

发表于 2021-09-20 | 分类于 每天写一点

在农村、小县城，有很多四个轮子的电动车，号称老年代步车，但车子的外观内饰，已经和传统的汽车差不多了。

那么，到底这样的车算不算机动车、该不该上牌、机动车的界限到底在哪？这种高级复杂的问题，摆在了执法者面前。尤其当机动车怕被拍而老老实实守规矩时，不上牌的小电动车横冲直撞，这种冲突就会越来越严重。

写到这里，我又想起来，人和动物的区别该如何定义？上学时，这个问题的典型答案，是人会使用工具。但事实上，灵长类动物也会用树枝去洞里掏蚂蚁，那这个树枝，算不算工具呢？

0919 - 精英社会的异义

发表于 2021-09-19 | 分类于 每天写一点

现在是精英社会，所有人追求的都是更好、更多。比如，赚更多钱、享受更多。

在这样的环境下，看到好人好事、有人一辈子精研一件并不赚钱但有意义的事，就给人很异样的感觉，就像是个异类。

这样的人，对我们也是个警醒；警醒我们以为的自然而然，是否真的自然。

0918 - 要用就用好的

发表于 2021-09-18 | 分类于 每天写一点

最近看的一本书，作者说：咱们都是用脑子的，别喝乱七八糟的酒，把脑子喝坏了。结果，作者只喝茅台。

关键在于，时间就这么一次，喝了这个，就不能喝那个。更珍贵的，是人生在此时此刻，只有喝一次的机会。要对得起这珍贵的唯一，要用上好的。

其实，我也有类似的想法。如果有条件，还是要用上好的。这不，iPhone 13 发布，给老婆安排上。

说起茅台，想起另一个故事。为什么茅台这么有名，是它比别的白酒好很多吗？其实不然。你说，和它隔壁的酒厂，用着相同的水、经历相同的环境变化，怎么可能会天差地别。

核心的原因，在于红军长征时，在这里盘桓很久。部队喝酒，你懂的。后来，共产党成了执政党，周恩来在为招待外宾时，定了茅台为国宴之酒。于是，茅台火了。也于是，周总理被尊为「国酒之父」
