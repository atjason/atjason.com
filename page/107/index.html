1015 - 要不要买点比特币？

发表于 2017-10-15 | 分类于 每天写一点

一位八杆子跟 IT 打不着的朋友，跟我打听怎么买比特币，让我感觉这个东西好像变得很真切了。

这让做程序员、本该对技术很敏感、却一直没有买过比特币的自己，好好考虑一下了。

另外，这两天在听得到王煜全的专栏，也增加了买币的理由。

还有，就像朋友之前说的，政府是很难管住自己手去印钱；而人民币又不是像美元一样是国际通用货币，央行印钱就直接等于从老百姓口袋里抢钱。

再有，手上有些美元，放着也是放着，不如整点币玩玩，毕竟今年比特币已经涨了 5 倍了，让人心痒痒。

一个技术点没有想明白：比特币的无限可分割属性，是否抹杀了其稀缺性？

1011 - 找了位好帮手

发表于 2017-10-11 | 分类于 每天写一点

今天，勾搭上一位对运营感兴趣的朋友。之后，我做产品，有人帮我吆喝了。

之前应该说过了，还是想再说下这个观点：国内的个人开发者、小团队，对于产品的运营有普遍需求：

  • 自己时间少
  • 不擅长做运营

类似这样的产品还是很多的，非常需要专业运营力量介入，比如：

  • 从运营的角度，产品处于早期，稍微优点运营，效果就很明显
  • 如果只是基础的运营，通用的部分很多，一个走通了，很容易套到别的产品中

不过，问题也很明显：

  • 运营必须产品的销售、收入等核心数据。而这一点，只会开放给非常信任的合作者
  • 运营需要时间才能见效，产品所有者不一定有耐心
  • 运营很难分成。产品销量增加了，到底是运营的功劳，不是因为产品本身变得更好用了？

目前，我并没有接触到有专门做运营的团队，跟这位朋友也刚认识没几天，面也没见过，摸着石头过河吧，希望有好的结果，期待。
