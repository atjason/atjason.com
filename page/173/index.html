0422 - 做长期正确的事

发表于 2021-04-22 | 分类于 每天写一点

又是句正确的废话。怎么理解呢？举个反例吧。

比如，交易中，时常想赌一把。

  • 明明在下跌，却想抄个底。
  • 明明持仓是亏损的，却想抗到回本。

这样的事，哪怕短期、单次能获利，长期来看是没有意义的。能重复做、反复盈利的交易，才是值得做的交易。做其他事，也是类似道理。

0421 - 专注一种交易方法

发表于 2021-04-21 | 分类于 每天写一点

不同的方法，对于同一种的行情的处理方式，是不同的。进而，其盈亏结果也是不同的。

拿炒单和波段来举例。比如，在 4328 做多，价格上涨至 4336 后回落至 4330。

  • 如果是炒单，应该已经平仓，能赚几跳的钱。
  • 而做波段，为了抓可能的波段，在建仓初期，止损是放在保本甚至是亏损位置的。对于这样的走势，不仅不赚钱、甚至可能亏钱。

这时，做波段的人，容易羡慕炒单的：还是落袋为安好。

这样的思路是不对的，既看到已经发生的，还要想想没有发生的情况。比如，价格如果继续反弹，炒单不一定能找到机会重新开仓、或者很难，进而不容易抓住完整的波段。而做波段，则可以抓住这波上涨的大部分，盈利更多。

另外，毕竟波段消耗的精力小。如果单一品种没什么机会，可以同时关注多个品种；这点，是炒单无法做到的。

因此，每种交易方法都有其优缺点。不要当自己的方法遇到命门时，就羡慕别的方法。而是找到自己真正认同的、适合自己的方法；好好了解它，尤其是它的命门；然后，好好利用它，找到适合它的行情，反复使用它来赚钱。

0420 - 取巧不行就守拙

发表于 2021-04-20 | 分类于 每天写一点

一直以来，在交易这条路上，总想用高效的方式学习：用最短的时间、最小的成本，取得最大的进步。

目前来看，是失败的，还在稳定亏损中。

我需要好好反思下自己的做法，想想破解之道。

不过，在有进一步思路之前，先用笨办法：

  • 每天收盘后，用上帝视角，重新盯一遍盘。
  • 把去年一年的某品种的 K 线，一根根复盘。

0419 - 积极努力应对

发表于 2021-04-19 | 分类于 每天写一点

最近几天状态不好，交易不顺，生活上的烦心事不少。

怎么办呢？想了想，只能是积极努力应对。

当然，这不容易。本身，遇到挫折，状态已然不好。这样的情况下，还要积极、还要努力，实在是勉力。

可，不这样，又能怎样。任凭状态低迷，虚度光阴，就这么耗着？

首先，承认失败。做的不好，就是自己能力不济。不怪他人、不怪环境。坦然接受失败，才可能从失败中走出来。

然后，内心平静。不论接下来做什么、怎么做，这都是起点。既然，解决问题还得靠自己。那么，需要让自己有好的心态。

再者，向前看。眼下的困难，确实很难。那么，不如在未来找动力。

  • 想想自己的欲望，不想被满足吗？
  • 想想自己的内心，不想过得更加充实吗？
  • 想想自己的家人，不想让他们过得更好吗？
  • 想想自己的责任，让责任赋予你力量。
  • 想想生命的美好，让自己不舍得虚度。

你前进一分，它便退却一分。走出第一步，就能走好下面的路。

0418 - 大病自由

发表于 2021-04-18 | 分类于 每天写一点

说财务自由、时间自由，太虚了。

就说你能不能大病自由。

几万、几十万的手术，做不做得起？
几个月的治疗康复期，完全脱产不工作，抗不抗得住？

什么，你还年轻？好，父母大病，你看不看得起？

看似中产，大病返贫。
