0731 - 不够专注

发表于 2019-07-31 | 分类于 每天写一点

这里说的专注，是指在做某件事时，心无旁骛、全情投入、效率最高。

在家的呆的时间长了，做事总是被打断，好像已经习惯了低效地做事，这显然是不好的。

其实，越是这样，越是需要瞬间投入的能力。前一秒还在哄娃，下一秒就要高效代码，要的就是这种切换的能力。另外需要的就是，无法坐在电脑前时思考；一旦在电脑前，立即执行、实现刚才的思考。

0730 - 美股定投标的选择

发表于 2019-07-30 | 分类于 每天写一点

大体上，想选一个指数和一个普通股进行定投。

指数方面，主要是 ETF，比较了 SPY/IVV/VOO/VTI/QQQ/FDN，最后选择 VOO。主要是稳定，费率低（0.03%）。其实，SPY/IVV/VOO 三者，不管是包含的成份股，还是历史收益，都非常相似，选谁都差不多。

其实，本来想选择 FDN 的，因为看中其收益高。不过，其波动也比较大，不同时间进场，差异很大。想着，既然是投指数基金，还是求稳大于求收益。高收益的事，留给其他标的去完成吧。

普通股方面，本来是想选一个非科技类股，当是对冲，比如可口可乐。不过想想实在是不了解，还是选科技股吧。候选的有苹果、亚马逊、微软、奈飞。苹果最近表现让人提不起兴趣；亚马逊股价太高，不适合小散定投；微软股份一直稳步增长，没有前二者那样的震荡，回测结果也比较好。恰好最近看了《刷新》这本书，那干脆就 微软 吧。

《定投改变命运》

李笑来的一本书，还未出版，开源在 Github 上。读完，最大的收获是，提醒自己好好研究下定投。

当然，如果你不信李笑来，看不上他在书中推荐了、他构建的 BOX，可以根据其思路，自己寻找适合定投的标的。比如，可以在 A 股、港股、美股中，寻找合适的股票，并制定定投策略。

0728 - 金融危机还有救吗？

发表于 2019-07-28 | 分类于 每天写一点

这两天在看《纸牌大厦》这本书时，一个印象较深的观点是：资本主义经济危机的一个底层原因是，生产过剩和有效消费不足。大量东西生产出来后，

  • 工人（泛指非资本家之外的人）无力消费，即想买也没钱买；
  • 资本家无欲消费，即他们依然想钱生钱，而不是把钱用来买自己生产出来的、原准备卖给工人以赚钱的东西。

一旦商品滞销，资金无法回笼，层层影响后，就引发了金融危机。

由于这个根本性问题，不可能在资本主义制度下自动被解决，那怎么才能解决呢？毕竟，经济需要循环，而资本天然有集聚的属性，会打破这一循环。全书看下来，挺悲观的，没太多办法。似乎眼下最有效的是建立国家垄断资本主义。国家通过垄断的利润，收回市场中多余的资本，补贴给底层，以增加有效需求，维持经济运转。如果单纯的经济方式无法解决问题，就只能诉诸政治；即通过政权变更、暴力等方式，建立不一样的政治制度，如社会主义。

0727 - 快速入门的能力

发表于 2019-07-27 | 分类于 每天写一点

比如，大家都不希望自己的钱被通货膨胀稀释，或者想增加被动收入，就需要了解投资理财。可怎么学呢？

如果是自己学，可能不知道从哪下手，不知道要学什么，或者学习周期很长。这时，焦虑就产生了。怎么缓解呢？这时，知识付费就来了。「免费是最贵的」、「学会投资自己」，这样的口号，就出现在一些付费的投资理财课程中。好像，这些课程可以帮你在最短时间内打通任督二脉；好像，你买了，就代表你会了。

当然，我不是说这样的课程一无是处。只是，确实鱼龙混杂。尤其，投资理财，入门容易精通难、块钱容易赚钱难，这是很难的事。但凡难，通常意味的痛苦。没有痛苦的思考、没有痛苦的亏损，只是像看抖音一样看教程，像聊天一样打卡讨论，很难有真正的收获。

我们通常说的学习能力，很重要的表现就是快速进入一个新的领域，即速成。能够明确目标是什么、知道需要掌握哪些知识、知道知识的获取渠道、能通过吃苦来掌握关键知识、通过实践达到目标。这一系列的技能是通常的，通吃各种新的领域。一旦有这样的能力和意识，进入一个领域，只是时间和兴趣的问题，焦虑也会大大减少。
