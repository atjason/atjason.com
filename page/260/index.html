0917 - 兴趣与成就感

发表于 2020-09-17 | 分类于 每天写一点

有时，真的很难分清兴趣与成就感。

一件事做成了，很开心。到底是自己确实喜欢这件事，还是成就感带来的喜悦，其实很难说。

反过来，如果做事时遇到挫折，感觉不想继续了。很难说，有这种感觉，是因为不喜欢这件事，还是纯粹因为遇到困难了。

一种是精明的人，始终想找自己最喜欢的事做。
一种是勤快的人，始终想把现在手上的事做好。

感觉勤快的人，比精明的人，更容易获得满足感。

0916 - 技术可行性分析

发表于 2020-09-16 | 分类于 每天写一点

如果用软件开发来类比交易，感觉自己目前还处于技术可行性分析。

这个阶段是开放性的，有很多可选择的技术。到底哪个好、哪个适合自己，目前尚未可知。唯一的办法，就是一个个去了解、一个个去试。在试的过程中，会有各方面的综合感觉。试的多了、深入了，适合自己的感觉就会慢慢涌现。

又有点像拼一幅残图，或者是一般的那种拼图。刚开始拼的时候，碎片很多，不知道从何拼起。可以先找到相对明确的点，比如拼图时先找到四角。拼着拼着，当拼好的块越来越多、剩下的块越来越少，就越来越好拼了。

选择是幸福的，也是痛苦的。选择太多，不知道选哪个好，也不知道该不该在当前的方向下重注。如果实在受不了这种痛，就根据目前已经的条件，选出当下最适合的。然后不管是不是最好的、不管三七二十一，死命研究，把这个方法做好。一通百通，之后再去看其他方法，就容易了。而此时，也更了解自己的性情，知道哪个适合自己。

关键的，要勤快，不要懒，不要等着好东西自己上门。

0915 - 国际汇款真麻烦

发表于 2020-09-15 | 分类于 每天写一点

这两天有笔可能的国际汇款要收，真处理起来，才发现国际汇款真麻烦啊。

最直接的，是汇款到银行卡。而实际上，这并不直接，而且很复杂。比如，你的银行卡在不在 SWIFT 等国际清算俱乐部中，汇款有 ACH 等多种方式，汇款过程中要准确填写银行 SWIFT 编码，等等。除了这一系列问题，还要接受并不菲的手续费。还有个小问题：汇款可能要等好几工作日才能到账。

再有，就是用 PayPal，这个渠道是比较常用的。相对于银行汇款，这确实简单了很多。不过，PayPal 只是简化了收款的步骤；收款之后，还是需要提款到银行卡这一步骤。不过，如果你经常用 PayPal 进行网络支付的话，其实也可以不提现。关键一点，PayPal，每笔收 4.4% + $0.3 左右固定的手续费，这着实不低。要知道，现在银行理财一年期也就差不多 4.4%，很多线下生意一个也就十几个点的毛利。

当然，还有比较潮的数字货币。不过，这就需要双方有极大的信任，不担心一方付款后，另一方跑路。关键还有一点，从币到法币，是需要特殊手法的。而且，如果金额较大，是很可能被管控的。

说了一堆，最后选择的是 TransferWise；其实，它和 PayPal 非常像。最大的好处是：手续费便宜，折合下来，大概不到 1%。交易还没进行，希望能一切顺利吧。

习惯了国内支付宝微信转账，瞬间免费到账。碰到这么昂贵落后的国际汇款，还真是不太适应。

0914 - 学习与火把

发表于 2020-09-14 | 分类于 每天写一点

交易市场是混沌的，也就没有绝对的正确。

前人们也是要在这混沌的市场立足的，因为从各个方向去摸索，也都各有所成；各家之言，自成体系。

对于我们这种新进的个体，踏进这混沌的、迷雾环绕的黑暗森林，该如何是好呢？最直接的，就是向前人学习。可前人那么多，该向谁学习呢？

接下来就是我最想说的：广泛学习。各家之言，就像一个个照亮黑暗森林的火把，共同照亮整个森林，或者说一片森林。即使不去某个被火把照亮的地方，至少你知道这个地方有什么，心中不会再恐惧。将来某天，如果真的需要，至少脑子里知道有这么个东西。

知道了这么多，最后还是要做减法，形成自己的交易方法，回归到某个点。甚至，只需要一个方法，一招鲜、吃遍天，就可以赚取足够的收益。

当然，要是有高人引路，就更好了，这就可遇不可求了。

0913 - 困、累、倦、没劲

发表于 2020-09-13 | 分类于 每天写一点

有时，我们觉得很困，透支的是精神，很想睡一觉。

有时，我们觉得很累，透支的是体力，很想休息一下。

有时，我们觉得疲倦，又有点厌倦。不想再做这件事，想换个事做。

有时，我们觉得没劲，干什么都提不起劲，觉得没意思，不想干。

负面状态有很多种，还可以是复合型的。我们又不可能完全消灭这些状态，只能是当它们出现时，积极合理的去应对。

确实是要「积极合理」。比如，觉得困，就去睡觉啊；如果只是躺在床上刷手机，非但不能得到休息，还会像水龙头漏水一样，慢慢的损耗精力。

很多时候，完成所带来的成就感，最能冲刷消极状态。可怎么在消极状态中完成所做的事，来冲刷当下的消极感，真是个鸡生蛋、蛋生鸡的事。也难怪，有那么多该休息却不休息的人在咬牙坚持，也许就是为了完成时那一刻的畅快。
