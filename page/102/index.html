1229 - 手机丢了怎么办

发表于 2019-12-29 | 分类于 每天写一点

到年底了，小偷也要回家过年；时间紧迫，抓紧时间再捞一把。这不，我家人的手机丢了，从被摸到发现，前后不过几分钟。当时人头攒动，想找到难如登天，毕竟不是外国人。

按道理说，现在手机越来越难丢了，因为大家对手机太依赖了，干啥都用手机、随时都拿在手上用，被偷的难度，要远大于钱包。可没办法，现在很多人出门都不带钱包，或者带了也没多少钱。就算难，小偷也只能硬着头皮上。

手机丢了，最想要的，当然是找回来；其次是避免二次损失，比如小偷盗刷支付宝；再次是不能让小偷得利，把手机变成砖，让小偷卖不出去。

找回来，很多手机都有找回功用。可一般太耗时，通常只能在另一手机上登录网页版找回功能，然后定位、锁定、让手机发声。这太需要快了，哪怕只是慢一分钟，都让这一切变得没有意义：扒手早就没影了。一关机，就完全没戏了。

防止二次损失，这个主要靠预防。一般来说，手机都有解锁密码，指纹什么的，让这一步变得很难。再加上微信支付宝，一般还有二次身份验证。还有一点，便是防止隐私数据泄漏。这一点，一般手机都支持远程擦除手机数据。不过，这需要平时经常备份手机数据，不然自己也永久丢失。

让手机变砖而卖不出去，所我了解，应该 iPhone 是做的最好的。替别人解锁 iPhone，操作上麻烦，风险也很高，很容易被苹果定位。

网上找到的一些丢手机后要做的事：

  • 致电运营商挂失手机号
  • 致电银行冻结手机网银
  • 挂失支付宝
  • 更改微信密码
  • 到手机运营商处补办手机卡
  • 修改各种账户的密码

1228 - 有感而晚发

发表于 2019-12-28 | 分类于 每天写一点

不论微博还是朋友圈，抖音还是快手，都是随手记录生活点滴，想到什么就说什么。

这样的表达固然快，但总缺少些深度。当下看起来很感动，过后可能都不愿再看。

有些东西，还是需要蕴酿。被触动想要表达，如果能给表达以时间，表达前多思考、多求证、多推敲，哪怕只是多一点点，出来的东西，也更有生命力。

人会自然地维护自己所表达的东西，哪怕只是自己仓促之前表达的东西，哪怕自己隐约也不觉得完美。既然是自己说的，就算是错的，含着泪、厚着脸皮也要去维护。可能你不承认，可这是潜意识层面就有的东西，摆脱不了。

想要摆脱，最好是晚表态、不要犯低级错误。就像当年万州公交车坠江一事，刚开始时、事情还未清楚时，大家纷纷指责私家车女司机；监控爆光后，公知纷纷删文章，好不热闹。

1227 - 自制力不足

发表于 2019-12-27 | 分类于 每天写一点

最近明显感觉自己的自制力不足。

当自己对事情感兴趣、知道怎么做、有明确可预期的结果时，自己会比较有动力，也可以很投入地把事情做完。

而一旦兴趣不足、不太确定该怎么做、遇到困难时，很容易气馁。这时，通常会想做点别的事调剂一下，或者干脆什么都不做。问题在于，一旦把注意力叉开，很难再切换回来。

相对应的，看一些年纪大的人做事，感觉就比较平。一方面，做了这么多年了，很难说事情本身能带来什么激情。另一方面，却能很自然的坚持做下去。虽说干活时的效率肯定不如年轻人，但没有年轻人明显的波谷，一段时间合计，并不一定比较年轻人出活少。

因为兴趣或其他原因，开始做一件事；
而一旦开始做一件事，最好像机器人一样，不喜不悲，干完为止。
