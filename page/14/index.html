1222 - 二刷《遥远的救世主》

发表于 2023-12-22 | 分类于 每天写一点

这一次刷，主要有几个比较深的感受。

1、丁的每一步，都是很简单、很直白的，并没有什么神来之笔。可一步步累加起来，却成了无可抵挡的必然。

换个说法，可以说是阳谋。即使你事先知道我要出什么牌、即使打明牌，依然打不过。

2、另一个感受，这种做法，只有王庙村是赢家。或者说，蛋糕并没有做大，只是把别人辛苦打下的江山，放到王庙村的口袋里。

在当时的环境下，对外人家说我们卖恶意低价、反倾销惩罚，对内我们依然用小农经济的思维，无限向下卷。从中国整体来考虑，确实是坏事。

3、芮的洒脱，尤其是最后的自戕，让人在精神上无比羡慕。当生则生、当死则死，还能更纯粹、更自在吗。

至于为什么这本书及改编的电视《天道》被半下架，可能因为这种深层次意识形态的探讨，容易擦枪走火，触碰到了敏感的神经。

1221 - 商业模式

发表于 2023-12-21 | 分类于 每天写一点

商业模式实际是一个链状的体系，一个好的商业模式创新，至少要想明白如下 5 个核心问题：

  1. 你准备提供的产品（或者服务），解决什么需求？
    换而言之，你的目标用户群体是谁？刚需，还是改善性的？高频需求，还是低频？

  2. 这个需求的市场是否足够大？
    如果是个小市场，那就不是创业，是糊口；

  3. 你的模式，会动谁的奶酪？
    换句话说，你的潜在敌人，块头有多大？这决定了你创业的摩擦系数与阻力，也决定了你最终能走多远；

  4. 你的模式，是否不容易被复制，被抄袭？

  5. 盈利模式——这是最、最、最重要的。
    简单说，你的商业模式，怎么赚到钱？任何不考虑赚钱的商业模式，都是耍流氓。但如果你耍流氓（当然吧，不能违法）就能解决赚钱问题，我不会在意你是流氓还是君子。

1220 - 国内期货

发表于 2023-12-20 | 分类于 每天写一点

准备再把国内期货再捡起来，用钱、时间换取经验。

毕竟，期货可以做两个方向、交易是在白天、资金进入无障碍，是比较合适的标的。

注意到螺纹钢有信号，准备交易的。可是由于账户太久没用，刚从休眠状态激活，当天不能交易，所以没开仓，错过了一波。

第二天做了豆一，目前盈利 10 个点；持仓过周末，看看市场会给一巴掌还是一颗糖。

不过，给什么也没关系了，不以结果论英雄，不以盈亏论好坏。只要坚持期望值为正的一致性交易策略，根据实际交易经验，在有必要时调整优化策略，也就可以了。

真正的困难，不是交易策略，而是交易过程中的心理活动，想大赚、怕错过、怕卖飞、想死抗、受不了浮盈的大幅波动、等等。这些才是接下来交易中的真正考验。

1219 - 美股货币基金

发表于 2023-12-19 | 分类于 每天写一点

空仓的时候，钱放着也是放着，考虑买些货币基金，赚点零花钱。说是零花钱，考虑到今天通胀等因素，一年也有 6% 的收益。

一些在美股市场中受欢迎的货币市场基金（MMF）：

  • MINT: 由太平洋投资管理公司管理的主动型管理货基，主要投资于1年以下投资级别的全球企业债。它是规模最大、流动性最高的场内货基，过去12个月的分红率达到1.41%，并且致力于将久期控制在一年以内​​。
  • SHV: 安信（iShare）出品的被动型跟踪货基，专注于1年以下的美国国债。它跟踪ICE美国短期国债指数，提供较低的费率，虽然回报率不及一些主动管理的基金​​。
  • NEAR: 同样是安信出品的主动型管理货基，与MINT直接竞争。这只基金投资于到期期限为3年以内的投资级别全球企业债，同时致力于将久期控制在一年以内​​。
  • BIL: SPDR出品的被动型货基，投资于1到3个月期限的美国0息国库券。这个基金主要持有短期国债，因此回报率较低​​。
  • GSY: Guggenheim管理的主动型货基，专注于全球各类短期债务。这只基金的过去3年总回报率超过了MINT，同时它持有大量的商业票据和机构逆回购，以及一部分现金​​。
  • RAVI: 一只主动管理的货基，专注于超短期全球投资级别的公开与非公开债券​​。
  • PVI: PowerShares出品的被动型货基，专门投资于浮动利率的美国市政债。它持有的市政债主要是超长期的，但由于利率浮动，有效久期仅为一周​​。
