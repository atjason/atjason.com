0522 - 太阳能好

发表于 2020-05-22 | 分类于 每天写一点

天越来越热，日照也越来越充足，就想着利用太阳能做点什么。

先是买了个太阳能路灯，装在大门口，晚上吃完饭，娃可以在门口跑一跑，消消食。还在院子里装一个，晚上亮堂堂的。关键不用布线，还省电，用着不心疼，没有心理负担，满足感就更高了。要知道，这样一个路灯，包邮才 33，真是不错。

又买了个太阳能灭蚊灯。其实原理还是紫外线之类的诱引，然后高压电死。关键还是电，太阳能就很合适了；白天晒着充电，傍晚蚊虫多时消杀。

还有一个号称最小的太阳能车。结构极其简单，就是太阳能电板 + 电机 + 车架。太阳一照就跑，跑到阴影下就站住，也挺好玩的。

太阳能真好玩。

0519 - 钱之外的判断标准

发表于 2020-05-19 | 分类于 每天写一点

今年种的一点油菜收割了，最后榨成菜籽油，也就 8 升左右。换成钱，也就 100 元左右。

这就引出一个问题：种地到底值不值？

如果从钱的角度，肯定是不值的。就这么点毛收入，减去种子化肥等硬性成本，再减去人工，肯定是不划算的。

可换个角度呢？比如对我妈而言，种地能让她感到充实。只要自己劳作了，不论多少，总会有收获。这种有稳定预期的事，让人感觉很踏实。这种踏实感，要比呆在家里好得多。以及，在大自然环境里劳动下，吹吹风、听听鸟鸣，也是很惬意的。

在操场上随机的奔跑、转向，肯定像傻子一样，至少是不自然。可如果有个足球在脚下呢？油菜对于耕作，也是类似的。足球比赛，并不是为了抢那个球；种地，也不完全为了最后的果实。

不要总钻在钱眼里，想想别的。

0518 - 黄金有价

发表于 2020-05-18 | 分类于 每天写一点

金价涨涨跌跌，不过好像没有那么坚挺、没有发挥出避险金融资产的特性，和股票等资产的走势比较相关。

就我二流的感觉，黄金的价值在相对降低。

所谓价值，本质是预期、共识。

当越来越多、尤其是年轻人不觉得黄金有价值、持有黄金不酷，黄金也就不那么值钱了。

相反，啥都不是的 BTC 倒是值钱了，因为人们相信它有价值。

当然，这些变化是缓慢发生的，是渐变。等老一辈死光了，就成突变了。
