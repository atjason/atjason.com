0416 - 不经历、无体验

发表于 2017-04-16 | 分类于 每天写一点

今天坐朋友的车出门，发现 有车以后，明显距离感不同。我没买车，会觉得去城市的另一端，很远；而有车的朋友，就会觉得还好，开车也不用花太久。

这就是体验、感受上的差别。没有车，是很难体会这种心理上的细微差别。

另外，在跟朋友吹牛时，他提到的一个幼儿园监控系统的案例，也清晰的表示：有些商业模式、背景里的逻辑，是坐在办公室无论如何也想不到的；一定要深入真正参与到项目的实际，尤其是了解整个利益链条上各个环节，才能真正理解项目的要害、被采购的关键。

0415 - 输出倒逼输入

发表于 2017-04-15 | 分类于 每天写一点

这几天在了解小密圈，心里在想：如果自己开个「独立开发者」相关的小密圈，怎么才能让自己有持续的输出、持续的产生对别人有价值的信息？

目前来看，自己尚未准备好。不过，如果只是抖存货，那肯定有耗完的一天。更重要的是，自己要能持续地学习，不断获取新的信息、刷新自己的认知。并且，要能了解圈友实际的情况，能结合自己的经验、对方的实际情况，给出自己的建议。

从信息的角度，自己不太可能造出全新的、唯一的、没有别人知道的信息；更可能的是，根据信息不对称，交流自己知道、而圈友不知道的信息。从圈友的角度，从别的地方也能获取信息，但是从我这里获取的时间成本更低、更准确。这应该就是小密圈能运转起来的真正动力。

0414 - 高风险、高收益

发表于 2017-04-14 | 分类于 每天写一点

如果有以下 2 个选择，你会选择哪个？

  1. 一年有 100% 的机会赚 10 万；
  2. 一年有 20% 的机会赚 100 万；

以前，我会选择 1，保险起见；

现在，我想试试 2; 从概率的角度，一年收入 20 万。

是时候尝试下高风险、高收益的选择。

是什么选择？我还不知道，在思考；希望年底能交出满意的答卷。

0413 - 多花钱

发表于 2017-04-13 | 分类于 每天写一点

节俭是美德；仅此而已。

感觉最近花钱花太少了。虽说，这样可以省点小钱，却失去更多。要知道，很多实际的体验，不花钱是永远感受不到的。比如；

  • 不交押金，就无法骑单车；
  • 不坐专车，就无法体会和快车的区别；
  • 不买 App，就无法理解购买者心态上的变化；
  • …

多花钱，不是为了花钱，而是收获更多体验。更宝贵的，能发现更多的机会，发财的机会。

0412 - 怎么找到 100+ 个需求？

发表于 2017-04-12 | 分类于 每天写一点

这两天在面对这个问题，还真难到我了。

先问问自己

关键的，自己平时要多看、多体验、多想、多思考，这样才能积累一些候选的需求，关键时候才能拿出来救场。

功夫在平时，还是要靠平时积累；临时抱佛脚，质量不高。

再问问用户

毕竟，自己能想到的点子还是有限的，不如直接到用户那问。

这里发现个技巧：愿意花钱的，才是真需求。比如，国内有类似猪八戒、58 这种发布需求的网站，绝对是真的有需求才来找供应商的，可以到上面找找灵感。另外，国外也有类似的地方。

当然，这些都是已经成熟的需求，很可能已经有解决方案了，自己再做就是重复了。所以，更多的是找灵感。

还有个地方，就是用户吐槽的地方。比如别家 App 的评论区、论坛、等等。已有产品可能有自己的包袱，很难解决用户的某些问题；而新产品轻装上阵，就好办得多。。旧产品的槽点，可能就是新产品的卖点。

别人也有好建议

每个人的知识领域、经验有限，多和别人头脑风暴，交换思想、杂交出新想法，会有意想不到的收获。

周五结束前，出 10+ 个基本靠谱的产品胚子。
