0427 - 创造力与复制力

发表于 2022-04-27 | 分类于 每天写一点

很明显的，虽然大家明面上说，要创造创新，但一到实际，就是参考 xx、模仿 yy、照着 zz，把一堆竞品拿过来分析，希望通过自己的排列组合，整出一个更好的，更不要说复刻抄袭。

乔布斯电影里有个细节，他把索尼卡带随身听扔到垃圾桶，然后创造出 iPod。

不是说复制力不重要，太重要了。这个世界就是普通人构成，用的是普通的产品，过着普通的生活。这当然没有错，甚至是世界的大部分事实。比如，我们改革开放，就是来料加工这种复制模式做起来的。

问题是，一方面，要知道自己是复制而不是创新，不要既做又立，要表里一致、言行一致。做复制的事，就要有低端的自觉，不要眼馋别人赚创新的钱，不要因为别人卡自己脖子就气得不行。另一方面，要知道世界是由创新驱动升级的。想要卡位赚爽钱，就要愿意承担失败破产的风险，去启动那不靠谱的创新。

0425 - 提高阈值很重要

发表于 2022-04-25 | 分类于 每天写一点

娃练习篮球跳绳，之前一直被压制在一个很低的数值。某一天，不知道因为什么，一下子突破了之前的数值。

这时，非常关键，一定要想尽办法，让他在突破后，尽量把数值提到非常高。比如，很用力的鼓励激励表扬。

一旦有了这次高记录，之后的练习就容易多了，因为他会很自然的将之前的最高标准，作为下次训练的目标，甚至是起点。

交易也是一样。一旦有过一次大盈利，之后就很容易把大盈利作为交易目标，就不会在意不好的机会，而只等好机会。

总之，抓住关键机会，提高关键阈值，非常重要。

0424 - 关于连续大回撤

发表于 2022-04-24 | 分类于 每天写一点

连续回撤本身是很正常的，任何策略都会遇到这种情况。

在对一个策略投入资本和时间之前，就要对历史有充分的回测。对最多连续回撤次数、最大回撤幅度，有准确的认识，并确认连续大回撤真的发生时，自己能否接受。进而，还要考虑更糟的情况。

至于是否降仓，其实降仓对策略不见得是对的，主要是照顾自己的心理。

0423 - 高铁公平吗

发表于 2022-04-23 | 分类于 每天写一点

建设高铁是财政出钱、是大家的钱，可高铁只通大城市，那些同样出钱的小城市和农村，不就亏了吗？

资源的聚集，造成了很多隐形的不公平。

怎么才能公平呢？

根本的方法，是人可以自由流动。农村和小城市的人，可以无门槛享受大城市的公共服务，比如学校。

懒惰的方法，是高铁受益的城市，向其他地方进行财政转移支付。
