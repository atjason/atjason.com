0530 - 相互扶持，结伙同行

发表于 2018-05-30 | 分类于 每天写一点

一个人，总是孤单；时间长了，总想找人说说话。

一位朋友说过：「我想找人一起干活，不是为了他能帮我什么，而是我想说话的时候，有人可以说话」

确实，很多时候，自己想，很累。跟人说话时，自己脑子同时想，好像就很容易想出新点子、新思路；这很神奇，我也时不时遇到过。

更重要的，总是一个人干，是不会有什么出息的。要办大事，还是要找牛人、抱大腿。

0529 - 银行的心机，总是比我们深

发表于 2018-05-29 | 分类于 每天写一点

之前被招行百般多次骚扰，推销其 e 招贷；出现一些原因，办了这个业务。

今天准备还款时，发现被「借款提前结清了」，一脸蒙逼，啥意思？

搜索了一下，大概明白了：我借了一年，目前刚好是半年。银行要提前收回借款，并同时下半年的利息照收不误。这不是坑吗？

于是，给银行打电话，得到的解释是：这笔钱只能用于个人消费，不能用于投资等业务。问题是，我根本没去投资啊？思来想去，最可能的是：我用这张卡买了币，于是被 zf 盯了，进而采取了这个措施。

同时，业务员轻车熟路地说：「我也想最大化地解决您的问题，多余的利息不收了，您看行吗？」果然，应该挺多人有类似问题。我也不想难为同样是打工的业务员，此事就此了解。

感叹：银(国)行(家)对金额的把控，实在是有些强到可怕。

0528 - 留点时间思考

发表于 2018-05-28 | 分类于 每天写一点

总是很忙，不一定是好事。没有大块时间代码时，心里也别急，正好用这个时间想想：

  • 具体层面，下一步做什么。
  • 如果能实现预期目标，下一步做什么。
  • 如果实在无法实现，又要怎么做。
  • 等等。

缺少时间做具体的事情，正好给了我们思考的契机，不至于 too busy to think.

0527 - 生活不会自己变好

发表于 2018-05-27 | 分类于 每天写一点

家务事真是头大。

不过，要知道，生活不会自己变好。要解决问题，还是要花时间、花心思，去想、去找解决办法、去试。

我们习惯于工作，也擅长自己的工作，能在自己的工作中找到安全感、满足感。而同时，我们几乎都不擅长于家务事的处理。于是，遇到问题、一有时间，宁愿加班，也不愿回家、或解决家里的问题。

但，越忽视，问题可能越积越大，越来越难处理。至少要拿出对待工作的认真，花时间，才能慢慢改善，并一直维持在好的状态。

0526 - 改善工作环境

发表于 2018-05-26 | 分类于 每天写一点

今天有 2 件事值得说下：

  • 简单了解使用 node 输出纯文本的表格。有了这项技术，可以延缓项目中对前端的需求，因为对于表格，哪怕是纯文本的，基本的展示也足够了。
  • 用了 BetterTouchTool；其实之前有用的，后来其改为收费后，就没再用了。现在又用上了，主要是配置快捷键。
    • 比如，我现在配置的 Command + Shift + 鼠标左击后，就可以在 VSCode 中返回上次编辑的位置，这还是非常方便的。尤其是在搭配键鼠时，右手不用离开鼠标，即可快速返回。

改善工作环境、改进工作能力，都是性价比非常高的事。
