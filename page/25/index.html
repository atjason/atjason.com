0822 - iPaste 更新粘贴方式

发表于 2022-08-22 | 分类于 每天写一点

之前，iPaste 是通过 System Events 脚本，来粘贴到目标程序，会有 0.x 秒的延迟。并且，需要一系列授权，一些用户总是遇到问题。

现在，iPaste 直接模拟 Command+V 快捷键进行粘贴，不再使用脚本，速度更快，更不容易出问题。

推荐更新；macOS App Store，搜索 iPaste 即可。

0821 - 养儿防老

发表于 2022-08-21 | 分类于 每天写一点

电影《反贪风暴5》中有这么个情节，一个老人硬拉两个儿子，陪他在养老院和一堆同住的老人吃饭。过程中，反复强调自己有两个儿子，还当众从口袋里拿出一把现金，抽出一部分给一个儿子，并说：你给的钱太多了，我花不完。

这里透露了两个情况：我有两个儿子、并且他们都管我，我钱很多花不完。

养老院也是个小社会，有家人管、有钱，这是最硬的两个标志。

反过来说，如果一个人非常有钱，但家人基本不管，那他很可能在养老院被欺负，因为被欺负了也不会有人帮他出头。被欺负的方式就太多了，比如护工暗地里的区别对待、甚至是虐待。

想想有些悲哀阴暗。但没办法，现实就是这样。

0820 - 蒸汽清洗机

发表于 2022-08-20 | 分类于 每天写一点

家里油烟机、空调要清洗，找人上门吧，有点折腾，花钱也不会少。

简单了解下，整了个蒸汽清洗机。虽说没卖家秀上那么摧枯拉朽般神奇，也确实是有用的。尤其是冬天，反差之下，高温蒸汽的清洁交易是很不错的。

从纯效率上来讲，自己花很多时间动手打扫卫生，不一定划算。但家不一样，投入的时间精力，不能仅用钱、用效率来衡量。

0818 - 交易如同麻雀啄食

发表于 2022-08-18 | 分类于 每天写一点

一种思想，不一定对。

麻雀每次只叼取一点点，然后迅速飞走。就这样它们不停地飞来飞去叼取面包，而每次仅获取一点点。它们用一百次的努力去获得鸽子一次便可获得的面包，但是你很难射中它们，因为它们飞得太快。
‌
交易也是一样，不能贪图一次赚个大的，不能总想着毕其功于一役，交易是长期的事。对市场保持敬畏，贪念就会少一点。
