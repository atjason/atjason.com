0124 - 最大化存量价值

发表于 2019-01-24 | 分类于 每天写一点

存量可能是包袱。经济学里的沉默成本，讲的也是在决策时，不应该被已经发生的成本所束缚，而应该仅关注当下做决定。

而从另一个角度，我们都是存量的结果，我们的肉体、思想、之前所经历的一切，共同构成了此刻的我们。完全抛弃存量，只考虑当下，是不现实的。

应该抛弃的，不是存量，而是存量思维，或者说避免被存量绑架。比如，之前我们一直在做一件事，突然发现了另一个天大的机会，是固执地做之前的那件事，还是全身心投入到新的机会？

应该利用的，是存量的价值。存量也是变量，就像是围棋，之前下的一子，可能是为了攻。而因为局势的变化，这个子再用作攻已不现实，却可能是守的关键一环。已经下的子，就像是存量，不能改变或抛弃，而它的作用却可以是不同的。我们每下一子，都要最大化利用之前下的子，最大化存量价值。

0123 - 用户系统是核心资产

发表于 2019-01-23 | 分类于 每天写一点

今天出个事：之前，用户可以使用微信登录头条；今天，突然不行了。

我自己测试，是新用户无法通过微信登录；之前已经用过微信登录头条的老用户，依然是可以登录了。可以说，微信还不至于那么黑。

万一微信连老用户也禁止了，对头条意味着什么呢？很简单：之前通过微信登录的用户，全部没有了；用户根本没有机会找回自己的账户。虽然，这些账户和数据都在头条数据库中，但头条根本没有办法将海量账户与真实用户对应起来。

对于微信，这绝对是有失商业道德的做法。今后，还有谁敢用微信作为登录账户呢？

对于头条及其他平台，敲响了严重的警钟：用户系统是核心资产，绝对承担不起风险，绝对不能纯粹依赖第三方，哪怕这个第三方声称自己多么可依赖。

我也开始真正理解，为什么几乎所有网站在使用微信、微博等第三方账户登录后，还是要绑定手机、邮箱。除了留下钩子方便日后骚扰用户，原来更怕的是被第三方掐断喉咙。

0122 - 纯学习还是不行

发表于 2019-01-22 | 分类于 每天写一点

这两天在纯学习 ES6，教材是阮一峰写的《ECMAScript 6 入门》

收获自然是有的，只是感觉纯学习的时间长了，效率自然低，经常书看的头昏脑胀，有些艰涩的部分有意无意就跳过了。明天把剩下的看完，还是再来一波实战吧。

实践和学习，还是得交替进行。做项目时，可以不管用什么方法，能出结果就行。而项目告一段落，就要正规化项目中非正规化的实现。这样，在实践中刻意练习，在学习时思考之前的实现是否可以优化，不断的在学习和实践中交替，更容易精进。

0121 - 每天听本书

发表于 2019-01-21 | 分类于 每天写一点

最近，在得到听了好多本书。为什么？因为要喂娃喂奶，喂完后还要拍嗝，这期间没法看书、操作电脑，手持手机也不方便。时间上，差不多 20 分钟。于是，非常适合听书：不用碰手机，时间也差不多刚刚好。

虽然，直接看干货，会消化不良。可以，消化不良总比营养不良好吧；没深入理解，也比不知道的好。而且，如果确实对某本书比较感兴趣，在听完后，也可以再找书来看。

记得有人问薛兆丰，在听完你的《北大经济学》课后，是不是要找曼昆之类的经济学书，完整系统地学习一遍。薛兆丰就问：曼昆的书，你看完了吗？那人挠了挠头，说只看了前几章，就没再坚持下去。这就很明显了，不管书再如何成体系、再如何完整，看不完，也是白搭。看书和听书也是一样道理，看书再好，没时间看，也是白搭。

小插曲：目前是手机 + 有线耳机在听。也有想买个无线耳机，可 AirPods 据说很快就要出二代，于是买个便宜的华为荣耀 FlyPods 青春版过渡一下。结果，这耳机在配合 iPhone 时，即使手机交易量调最小，耳机音量也依然巨大；配合 Android 手机则没有问题。算了，先有线耳机吧，毕竟自带的耳机戴着还舒服。不过度纠结外在，不买椟还珠；把主要的心思和精力，花在主要的事上。

0120 - 卖课

发表于 2019-01-20 | 分类于 每天写一点

最近看朋友圈，有人买了极客时间的一门前端课程：重学前端，正好最近想学下前端，于是就习了。买完后，App 会生成一张分享图片，我也就分享到朋友圈了。

其实，我也知道如果别人通过我的分享买课，我会有提成。可我没想到，提成着实不低。原价 98，活动期间我是 68 买的。分成是 24，按原价算也是 20%，着实不低，这是让我吃惊的。号外：通过我的分享买的朋友，我也一一发了红包。

舍不得孩子套不着狼。只有舍得让利，别人才愿意牺牲或者说消耗自己的信誉，去帮你推广。确实，有位朋友就说，就是看我推荐才买的。虽然我根本谈不上推荐，只是说了下我买了。
