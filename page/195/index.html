1121 - 新冠疫情带来的量变与质变

发表于 2020-11-21 | 分类于 每天写一点

新冠疫情，无疑给我们生活的方方面面，带来的巨大的变化。比如出门不戴口罩，比不带身份证更麻烦；感觉自己就像恐怖分子一样，很多公共场所都进不去。

今天听到一则新闻，说是中国造船行业接到的订单量，比之前上涨了很多，在全球也占到很大比重。在世界造船三强日本、韩国、中国中，只有中国受疫情影响最小，相对产能也更高、更稳定，所以接到了更多的订单。此消彼长，另外两家自然就少了。这只是量变。

如果时间拉长，这样的影响就会累积扩大；说不定，就会从量变到质变。因为，其实到了一定程度，竞争很激烈，各家的实力其实已经差别不大。一些关键技术早一点点诞生，就能在关键环节卡位；一步领先、步步领先。

我感觉，我才开始理解「新冠疫情，会给我们的生活带来巨大变化」这句话。

1120 - 用了就回不去的产品

发表于 2020-11-20 | 分类于 每天写一点

你用过哪些，用了就回不去的产品呢？

比如，视网膜屏幕。看过之后，就不想再看颗粒感十足的普通屏幕了。

比如，双十一买了无线充电器；从那之后这些天，我从没有用有线充过电。

比如，用了无线耳机之后，就受不了戴上有线耳机后，耳机线与衣服摩擦带来的异物感。

基本上，都是由奢入俭难。一旦感观接触并习惯了更好的东西，大脑感到兴奋的阈值就瞬间提高了。之后再接触原来的东西，就很难达到阈值并触发兴奋感与满足感。

1119 - 交易，游戏，年轻人

发表于 2020-11-19 | 分类于 每天写一点

日内高频交易，玩多了就很像游戏。

比如，即时反馈。游戏里，杀掉一个怪，有什么奖励，立马就兑现。交易也是一样，是赚是亏，当下立刻见分晓。交易里的账户余额，和游戏里的血值很像；对了立刻增加，错了马上减少，没了就出局。

比如，快速反应。游戏就不用说了，尤其竞技类游戏，速度甚至比策略更重要，拼的就是手速。高频交易也是一样，很多交易员的鼠标操作速度，并不比游戏竞技者差；鼠标下单后，立即移至止损或止盈的目标价位，随时准备点击。

这么说下来，是不是觉得很适合当下的年轻人；在游戏里浸淫多年，非常习惯于眼疾手快、数字的跳动等等。只是，涉及金额，和虚假的血值经验值金币等不同，很容易让人不淡定。反过来说，如果能把交易看成游戏，把余额看成游戏里的血值，减小账户波动对心态的冲击，会不会交易的表现更好呢。

1118 - 与情绪共存

发表于 2020-11-18 | 分类于 每天写一点

人是无法战胜情绪。

在交易中，贪婪、恐惧这样的情绪，反而出现，不断折磨着每位交易者。想要变成纯粹理性、没有贪婪恐惧情绪的人，是不可能的；只能与情绪共存。

「它强我弱、它弱我强」，当感受到自己已经处于贪婪、恐惧的情绪时，减小仓位、减少交易、甚至完全不交易、退出市场。等情绪过去了、感觉很轻松自然时，才正常交易、放大仓位。

1117 - 复活后能力加倍

发表于 2020-11-17 | 分类于 每天写一点

动漫里经常有这样的情节设定：一个角色死而复活后，能力会加倍。

其实想想，还是有点道理。

对应到交易，如果一个人亏损过，大亏，难受的抱着痛哭、晚上睡不着觉、不知道未来的路该怎么走。这样的人，深深的认识到市场的可怕，会从心里敬畏市场。如果他再次爬起来进入市场，一定会小心谨慎，避免再次失败。有这样的心态在，更容易盈利。

投资界也有这样的说法，选择项目时，会看发起人有没有过创业失败的经历。有这样的经历的人，会更加小心谨慎，不会盲目算大，更容易成功。
