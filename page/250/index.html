0519 - 钱之外的判断标准

发表于 2020-05-19 | 分类于 每天写一点

今年种的一点油菜收割了，最后榨成菜籽油，也就 8 升左右。换成钱，也就 100 元左右。

这就引出一个问题：种地到底值不值？

如果从钱的角度，肯定是不值的。就这么点毛收入，减去种子化肥等硬性成本，再减去人工，肯定是不划算的。

可换个角度呢？比如对我妈而言，种地能让她感到充实。只要自己劳作了，不论多少，总会有收获。这种有稳定预期的事，让人感觉很踏实。这种踏实感，要比呆在家里好得多。以及，在大自然环境里劳动下，吹吹风、听听鸟鸣，也是很惬意的。

在操场上随机的奔跑、转向，肯定像傻子一样，至少是不自然。可如果有个足球在脚下呢？油菜对于耕作，也是类似的。足球比赛，并不是为了抢那个球；种地，也不完全为了最后的果实。

不要总钻在钱眼里，想想别的。

0518 - 黄金有价

发表于 2020-05-18 | 分类于 每天写一点

金价涨涨跌跌，不过好像没有那么坚挺、没有发挥出避险金融资产的特性，和股票等资产的走势比较相关。

就我二流的感觉，黄金的价值在相对降低。

所谓价值，本质是预期、共识。

当越来越多、尤其是年轻人不觉得黄金有价值、持有黄金不酷，黄金也就不那么值钱了。

相反，啥都不是的 BTC 倒是值钱了，因为人们相信它有价值。

当然，这些变化是缓慢发生的，是渐变。等老一辈死光了，就成突变了。

0517 - 情绪免疫力

发表于 2020-05-17 | 分类于 每天写一点

反过来，别有玻璃心。

情绪对一个人的影响，很大，可以大过做事所需要的能力。

现实中，有太多糟心的事。老人的身体、小孩的作业、手机被偷…数不胜数。

如果但凡遇到点糟心的事，什么都不想干、几天都缓不过来，那完了，啥也别干了，就忙着糟心了。

必须给脆弱的小心情，加个厚脸皮。别把事当事、该干啥干啥。

有些实操的做法。比如，在做该做的事之前，先把糟心的事仔仔细细在脑子里过一遍，想想最坏的结果、所有可能的解决办法。等你把这些都想一遍后，再去做事，潜意识里就不会没完没了地想那件糟心的事。越压抑不想，就越会去想。主动想了，反而会压抑之后不自主的去想。

绝对实力越强、生活和事业的支点越多，越不容易被影响。

0516 - 想法与能力匹配

发表于 2020-05-16 | 分类于 每天写一点

我们想要的东西很多、很美好，但要和自己的能力匹配。不然，很尴尬。

想想前些年中国的低调：

  • 炸了我们的大使馆，忍；
  • 南海害死我们的飞行员，忍；
  • …

是我们不想站起来、硬气一把吗？显然，我们缺少站起来的实力、硬气的资格。

现在呢？这样的事还会发生吗？

不会。

为什么呢？自然是我们的实力增强了。

具体的个人身上，也是一样。没有实力时，别人欺负你，也就欺负了。最佳的办法，就是忍，然后低头发展。等你强大到当初欺负你的人不得不低头来求你时，不是很爽吗？

0515 - 量化交易虚幻的自信

发表于 2020-05-15 | 分类于 每天写一点

之前想过很多策略，也都用代码实现了。不出意外，全都失败。

现在回想，当时有一种诡异的成就感、满足感、自信。

复盘时，为什么会沉迷在那种状态呢？

最主要的，还是想制造一个自动赚钱的机器，这个确实太诱人了。

再有，就是这个过程中，考虑了很多场景，写了很多代码。有了这些付出，想着应该会有回报。而在交易这个领域，达到门槛之前，付出不会有回报，学费倒是不少。

不知不觉中，很多时间就过去了。要从这种状态中抽离出来，切换到累人的主观交易，并不容易；这是自己目前准备做。机器并不会放弃，退居人机配合。
