0620 - 把难题放一放

发表于 2016-06-20 | 分类于 每天写一点

前几天解决一个难题，折腾了很久都没搞定。没办法，只能放一放。

今天又来搞，比较顺利就解决了。

之后，碰到难题，如果折腾了很久都没搞定、又不是立即要解决了，就先放一放。】等过些时间，换了脑子后，再来解决，说不定就有新的办法、就能顺利解决了。

0619 - 别迷失在非必要条件中

发表于 2016-06-19 | 分类于 每天写一点

今天注册微信公众号，想用一个独立的邮件注册，就去注册 Gmail 账户。结果 Gmail 注册不顺利，花了 1 个多小时都没搞定，还让心情很差。最后还是随便用个邮箱注册了。

其实，现在想想，一个恰当的邮件并不是注册微信公众号的必要条件，更重要的是微信号、是注册本身。不管什么邮箱，先注册了再说。

事情上，这种事情经常发生。这种所谓的完美主义，却让我们远离目标，在不重要的分支上越走越远，消耗精力、心情。

以后要特别警惕，不管是在分支上走得顺与不顺，都要经常提醒自己：我的目标是什么、现在做的事是实现目标最重要的条件吗？

0616 - 不能做到哪算哪

发表于 2016-06-16 | 分类于 每天写一点

做事，还是需要目标感、方向感。

做到哪算哪，在不出问题时还好，一出问题就比较麻烦。比如，目标一拖在拖、持续工作导致低效率、等等。

更严重的，一直在忙碌的状态，还会掩盖上面的问题。

做事时可以有一定自由发挥空间，但不能离目标太远、太偏离方向。
