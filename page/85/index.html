0816 - 意志力不可靠

发表于 2020-08-16 | 分类于 每天写一点

「我一定要做成某件事」、「我一定要改掉某个问题」，基本上，以这样方式描述的，最后都是做不成的。

人的意志力，其实少的可怜、脆弱的可怜，实在是不足以依靠。

能依靠的，是习惯、是简单的规则、是有明确反馈与奖惩机制的系统。

比如交易，再怎么强化要止损的意志，都不如先列个交易禁忌清单，然后在每次交易前读一遍，来得可靠。

0815 - 听书与抖音

发表于 2020-08-15 | 分类于 每天写一点

听书，大体是有人读完一本书后，根据自己的理解浓缩整理，然后用有声的方式读出来，大概 20 分钟。

  • 好的方面，不听不知道，听了会对更多的书有一定的了解。
  • 坏处是，一旦知道了结局，可能就不愿意再打开书细细品读。即使阅读，读到精彩、反转处时，能感受到的冲击和喜悦，也会少很多。

类似的，抖音上有很多拆解电影的。也是用大概 20 分钟时间，把电影关键情节剪辑，尤其是堆出所有反转情节。

其结果和书类似，既享受了短时间高强度的刺激，却也错过了细细品味所带来的更绵密悠长享受。

不听不知道，听了不想看，不知道是好是坏。

0814 - 景区农家乐

发表于 2020-08-14 | 分类于 每天写一点

旅游住的农家乐，生意还是很好的。工作日，几乎都停满了；要是周末，估计最少要提前一周预订。在这样的情况下，感觉主人家的服务态度并不怎么好，明显也没有提高的动力啊，反正就这样也是客满。

不过，我个人觉得这样的生活挺无聊的。每天都生活在相同的地方，做着相同的事，看着相同的风景；唯一不同的，是面对不同的游客而已。

对年纪大的人来说还好，可对于下一代就不那么友好了。比如，小孩上学、上兴趣班，很是麻烦；看个电影、逛个街，也要花挺长时间下山。所以，看到的主人家，基本都是年纪比较大的，我几乎没看到本地的小孩。当然，老人家也会有困难，比如看病。

可问题是，10 年、20 年之后呢？现在这批人会老去，年轻人会有多少愿意从城市回来接班呢？当然，并不是所有人都对大城市感兴趣，可城市毕竟还是有很明显的聚集效应；更大可能，是在城市混不下去，才会回农家乐接父母的班吧。

0813 - 买车与租车

发表于 2020-08-13 | 分类于 每天写一点

一直在买车与租车之间纠结。

买车，直接的困难是要摇号。如果是买电动车，不需要摇号，却有充电的麻烦。最关键的，车买来后，还会引入一系列麻烦事，比如停车、处理可能的车上的异响异味等问题、可能的刮擦追尾等事故、年检等等，想想都头大。

租车、打车，可以避免以上所有问题，不过也有新的麻烦。比如，高峰期不容易打车，去景点或郊区旅游，去时容易回来难，司机开车水平不确定等等。

这两天去附近的城市旅游，全部是打车，其中主要是顺风车，整体还算顺利。

目前的打算是，佛系摇号，摇到了就买辆油车过度下，几年后换辆能自动驾驶的电动车。

0812 - 限定条件是好事

发表于 2020-08-12 | 分类于 每天写一点

想当然的说，做事时，资源无限、不设条件，是最好的，能做出最大的成绩。

实际上，不见得。

比如交易，什么限制都没有，可能会乱来。如果限定了一天最多只能做一笔交易，那自然会小心谨慎、优中选优、宁缺毋滥，反而可能会避免各种骚操作、避免亏损。不亏，就已经是赚了。

资源有限、条件限定，这样的约束是永恒的。不要抱怨、不要抵抗，试着把约束变成避免浪费的帮手，戴着镣铐跳舞。
