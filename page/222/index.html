0119 - 不要过于期待某件事情、某样东西

发表于 2019-01-19 | 分类于 每天写一点

当即将得到一个好的东西，就会不自觉地期待。这个期待的过程，会使得我们不自觉地夸大它的好处、压抑它的坏处。一旦得到之后，却可能因为期望值与现实间的差距而心生不满，至少是满足感降低。

除了东西，事情也是一样。

从另一个角度，我们期待新的东西，是因为已有的东西有些地方让我们不满意，我们迫切希望好的东西来改变这一局面。不过，事实上，任何事都有两面，好的东西也是有隐性的坏处，或者至少是有代价的。d而目前的东西，可能也没那么不堪。稍加优化，还是能有很大改善，幸福指数大增。

再跳出来，我们总是需要不断地通过新的、好的东西来帮助我们改善生活。不过，要清楚，这些外在的东西，只是「帮助」我们。真正是好是坏，还是要看我们自己的内心。不要因为这些具体的东西，模糊了虚无的内在。

0118 - 系统性学习

发表于 2019-01-18 | 分类于 每天写一点

面向 Google 的编程，总是遇到什么问题，就想办法解决什么问题。好处是快，坏处是长期来看效益低。

一方面，可能在遇到相同的问题时，由于没有积累，需要重复地去搜索解决。另一方面，由于缺乏系统性，不能深入地掌握技术，只能在浅层划水，也解决不了什么深刻的、复杂的问题。

系统性学习，就是在掌握一部分知识后，就试图为知识建树、建网，对于自己已经掌握的知识，想办法固化；同时，标记自己尚未掌握的盲区。随着学习的推进，不断固化更多的知识，也标记更多的未知。在这样重复的过程中，不断地螺旋式上升，扩大自己的知识版图。

0117 - 立志高远

发表于 2019-01-17 | 分类于 每天写一点

这两天在看张宏杰写的《曾国藩传》，其中介绍，曾国藩在很早的时候，就立下了做圣人的人生目标。有了这个目标，他可以更明晰自己将来要走的路；在遇到困难时，有毅力坚持下去；在需要做选择时，也不会被眼下的、短期的东西也蒙蔽，可以从更长远的角度，来反推出目前应该做的事。

就像是学开车时教练说的：要把车开直，不是要关注手和方向盘，而是要目光看远处，这样都会下意识地调整左右方向，保证车走直线。

反观自己，感觉最近很盲目，东一榔头西一棒子，逮什么搞什么，没有计划性和目的性。几年后回看自己现在做的事，会觉得很可惜，就像现在回望几年前的自己一样。

当然，现在马上定下人生目标，可能还写不好。不过也要把这个问题放心里，时不时拿出来问一问自己。

今年的目标：技术补课；夯实基础，为之后的持续前进打下基础。
