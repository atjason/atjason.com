1030 - 小事开始

发表于 2020-10-30 | 分类于 每天写一点

当面临一件复杂的事，千头万绪，不知道从哪下手，怎么办？

不要站着不动干想。动起手来，先做一件小事。

做事时，会让大脑处于更沉浸的状态。不那么急躁，又依然浸入事情之中，脑子里依然在默默的想着。等手上的事做好了，心里也会慢慢有一些成型的片断。以点带线带面，能慢慢的把事情串起来。而且，已经做好的事，会成为一块石子、一个台阶，成为下一步的基础。

1029 - 专注是幸福的

发表于 2020-10-29 | 分类于 每天写一点

看牛人同时做很多事，又都做的很好，觉得他们很牛。

其实，我自己内心并不怎么羡慕。目前的我，只想把一件事做好；做好这件事，再去做其他事也不迟。具体的说，这件事就是交易。

想到自己找到了合适的事，不用去寻找，也不用在众多选择中纠结，觉得自己身处幸福之路。

我其实不太擅长做选择，进而也就不喜欢，就像在外面吃饭我不喜欢点菜。既然不用点菜都已经上桌了，专心享用便是，又何必再去眼馋别人桌上的美味。

1028 - 发散还是收敛

发表于 2020-10-28 | 分类于 每天写一点

是多挖几口井，还是一口井挖到底，这是个老话题。

可，之所以是个老话题，是因为它很重要，值得反复说，却也不会有最终解。

时间上，年轻时可以多尝试，试错成本低。等年纪大了，再切换赛道，成本也跟着大了。

经验上，如果已经找到了自己喜欢、又值得深挖的井，又何必费神再去开新的坑呢。

以为已经找到了答案，可能只是目前这一阶段的答案。比如，号称我们这代人能活百岁，那现在喜欢做的事，50 年后还喜欢吗？

所以，这更像是个螺旋的问题。时常又会遇到这个问题，而回答问题的你，已与当初不同。

1027 - 小事大时间

发表于 2020-10-27 | 分类于 每天写一点

有这么个说法，我们在考虑小事时花的时间，要比大事要多，至少是不成比例的。

比如，淘宝上买个几十块钱的东西，总要货比三家、看看评论吧。一番折腾下来，时间可花去了不少。而在花几万甚至更多钱，决定买哪支股票、什么时候买时，所花费的时间，绝对不是几十块钱东西的几十、几百倍。中午吃什么这样的世纪难题，更是如此。

这其实跟思维的习惯有关。在考虑鸡毛蒜皮的小事时，整个事情我们比较了解，插得上话，事情做越来比较容易，自己就愿意去做。而在思考大事时，通常是比较累、比较纠结、比较痛苦的，人就不愿意在这样的状态停留太久，迫不及待想摆脱这种状态，于是随便做了决定。

这还是和思维停留在舒适圈有关。总在舒适圈里思考，会有虚假的成就感，就是我一直在思考、一直在做决策啊。其实，都是些鸡毛蒜皮的决策。应该减少做这些决策所花费的时间，随便就好；而把节约出来的，用在让自己感到不舒服、却很重要的决策上来。
