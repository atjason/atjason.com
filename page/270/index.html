0605 - 爱好与专业

发表于 2019-06-05 | 分类于 每天写一点

经常听到这样的观点：别拿自己的爱好去和别人的专业 PK。

我本身也是认同这个观点的；毕竟，人家吃饭的家伙，每天都在操练，是有长期优势的。如果只是一腔热情冲进去做，基本也只能做别人的陪练或韭菜。

可，有时候总是不甘心；毕竟是喜欢的东西，如果还没 PK 就先放弃，总归是不舍的；哪怕知道最后会是输，也想试一试。

爱好和事业能够重叠固然好；如果不能，我还是想给爱好一点时间。

0604 - 必然要做的，不抱怨

发表于 2019-06-04 | 分类于 每天写一点

这种事情时不时会发生在自己身上，我想也很常见：并不想做某件事，但基于各种原因，又不得不做。于是，做吧。而做的过程中，各种不爽、乱发脾气、心情很差，总之很糟糕。

事后来看，都说是不得不做，抱怨除了降低效率、让自己心情变差之外，并无意义。不如尽快处理掉，释放出自己的时间和心情。

0603 - 挑最好的

发表于 2019-06-03 | 分类于 每天写一点

今天去书店，门口看到一个小孩吵着要买一本书，奶奶说：楼上全是小孩子的书，你先看一遍，再挑最喜欢的吧。

这就很有意思了。我们一直以来都是受到这种教育：挑最好的。但实际上，这很难操作，挑是有前提和成本的。比如，楼上有那么多书，小孩子不可能全部看一遍。再举个极端的例子，找异性结婚，最好的当然是把全世界所有潜在对象过一遍，然后再挑，这一听就知道不靠谱。

触手可及的，可能就是最好的。即使不是最好的，也可能是最值得选的。

0602 - 控制变量

发表于 2019-06-02 | 分类于 每天写一点

最近几天不确定的事太多，导致有点焦虑：要不要做量化、选什么标的、怎么开发策略、用什么语言实现、要不要学 Python 等等。这么多变量混在一起，直接混沌了。

人脑是无法并行同时处理这么多问题的；只能是一种思想占优，其他的辅助。而真的面对这么多问题时，只是选择性忽略，或者给予基本的假定，等等。总之，是 把不重要的变量固定住，以减少变量；然后 专心对付最主要的变量，也就是主要的问题。

0601 - 吃到肚里，才是好的，吗？

发表于 2019-06-01 | 分类于 每天写一点

比如，在家里自己做的饭，吃到最好还剩一点，怎么办？如果家里有老人，多半是努力吃到肚子里。就算是由饱到撑，也不能浪费粮食。

原因上，基本是老人是人匮乏时代过来的，一辈子节俭怪了，见不得浪费。而现在的年轻人，可能更注重健康，更别说天天嘴巴上说减肥的人，扔掉也不容易有什么。而再年轻的小孩，估计连眼都不眨一下。

环境，就像重力一样，牢牢把人吸在地表。而人的能动性，就像是能在地上蹦哒几下而已。能挣脱引力的人，太少太少。
