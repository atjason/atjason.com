0624 - iTips: 极简笔记本

发表于 2017-06-24 | 分类于 每天写一点

今天和一位荷兰的用户+朋友聊了他的一个想法，我取名叫 iTips

核心诉求：轻量化的常用信息备忘录。

工作、生活中，我们常会碰到这样一些 碎片信息：

  • 公司发票抬头
  • 客服的常用语
  • 回复邮件时不同的签名
  • ⌘ 这种常用、却不容易打字的特殊字符
  • 常用的 Emoji 表情
  • 某个视频看到第几分钟
  • 最近工作常用的一些用语
  • 代码片断
  • 常用的一些命令
    • gc master; git merge dev; git pull; git push; gc dev; git merge master;
  • …

如果记录在 Todo 类应用中，感觉怪怪的，一不小心就会被完成/删除。
如果记录在笔记应用中，搜索起来麻烦，且还需要选择、复制，才能在目标应用中粘贴。

能不能有一个轻量的应用，专门记录这些轻量的、碎片化的信息，需要时可以快速获取/复制？能，这就是 iTips

目前只是点子，也只画了原型。不过应该会挖这个坑，因为至少是个很好的练手 macOS + iOS 的项目。

关于这一个点子，你有什么想说的？欢迎跟我聊聊。
