0220 - 别用赶路的心态学习

发表于 2019-02-20 | 分类于 每天写一点

学习和旅游，其实挺像的。

比较现代人的旅游，基本是下车拍照、上车睡觉。炫耀的是去过哪哪哪，有朋友圈为证。可如果你问些旅游地稍微有深度的问题，基本就 GG 了。

学习也是类似，「我会 XXX」「我学过 XXX」，很多也都是朋友圈晒照级别的。真要用 XXX 解决某个问题、做出一个产品，基本就 GG 了。

花一年时间走遍世界，可能不如在一个地方生活几个月。
A/B/C/D/E/F/G 都知道一些，可能不如好好琢磨其中一个。

当然，浮光掠影未尝不可，只是一定要和深度游结合。就像是先把地图看一遍，做完攻略，然后重点体验某些地方。

学习也是一样，应该保持开放的心态，什么都接触、什么都知道一些，培养全局观；但一定要在某些领域，有自己擅长的、拿得出手的东西。

学习的目的，是提高解决问题的能力。

0219 - 怎么才能成为独立开发者？

发表于 2019-02-19 | 分类于 每天写一点

时不时会被问类似的问题：怎么才能成为独立开发者？这条路有哪些坑？要注意什么？等等。

我目前的理解，最主要的是：

  • 对内：知道自己想要什么，并有足够的自制力去学习、去实现
  • 对外：很长时间不成功，生活也不至于受很大的影响

当然，还有很多外围的因素：

  • 家人的理解和支持，或者说是不是能扛得住周围人对你这种状态不理解？
  • 自己是不是能够适应一个人呆很长的时间？
  • 等等

Anyway，独立开发只是一种状态、一种选择、一种方式；方式与方式之间，并没有高下之分。合不合适，自己知道。

0218 - 体己才能悦人

发表于 2019-02-18 | 分类于 每天写一点

比如一位妈妈，既要工作，又要带娃，还要处理各种家中琐事，很容易脾气暴躁心绪不宁，事情更加不容易处理好，心情和脾气变得更差，陷入恶心循环。

如果每天能有半小时自由时间，或冥想，或逛街，或刷剧，总之做自己想做的，或者干脆什么都不做，会让自己的状态不至于越来越差，甚至能越来越好，开启正向飞轮。

体恤自己，才有能力取悦别人。

0217 - 恢复背单词的习惯

发表于 2019-02-17 | 分类于 每天写一点

最近在学习 Vue，看的是英文的官方文档，也是为了英语不丢下。期间，遇到一些不认识的单词，自然会去查。可查了之下，只是当时明白了，之后自然是忘记了。

对开发者而言，英语是非常重要的。比如获取国外最新的技术，比如伺候国外用户，等等。具体的说，主要的是阅读和书写的能力，听力和口语相对要求不高。而阻碍阅读和书写的，主要还是单词。

于是，问题就变成了 如何掌握更多的英文单词。

我并没有打算背托福、雅思之类成型的词典，而只是记自己日常遇到了（已经够多了…），所以就需要自己构建生词本。工具上，我还是选用的自己之前用的 Anki；Chrome 浏览器结合 Online Dictionary Helper 插件，划词时就可以将重启加入生词本。

目前的计划是：每天学习 10+ 新单词，一年就是 3000+；如果计划能充分执行，一年后应该可以无障碍阅读大部分技术文档了。
