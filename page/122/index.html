0819 - 通透地掌握一门技术

发表于 2019-08-19 | 分类于 每天写一点

感觉我还没有通透地掌握一门技术的感觉。

比如英语，前后学了有 20 年了吧，可今天看一篇文章，感觉有一半的单词不认识…

对于交易，无疑国外是开创者、领先者，直接从英文世界里获取信息，无疑是高效、准确和必要的。进而，英语阅读能力则显得非常重要。

0818 - 行情好时赚钱，行情差时积累

发表于 2019-08-18 | 分类于 每天写一点

在币圈有这么句话：牛市赚钱、熊市赚币。

对于玩短线策略，基本是靠行情吃饭。行情好时，能赚多赚点；行情不好时，鸣金收兵、挂靴封印，多积累，填补之前遗漏的点，学习新的东西，扩展知识面，以备再战。

尤其在行情差时，不能恋战。否则，既浪费时间精力，收获不大；又可能在行情来时反被套。不如无仓一身轻。

0816 - 说说养老

发表于 2019-08-16 | 分类于 每天写一点

或多或少、或早或晚、或别人或自己，我们都会遇到养老问题，今天来聊聊。

养儿防老

最开始，我们的文化是「养儿防老」。老人年纪大了，多个子女共同承担，每个人的压力都会小些、效果好些。

可，记得我小时侯，电视上时不时看到，几个儿女纷纷把「没用」的老人住家外推，这种现象在农村绝不罕见。即便不是普遍现象，也足以说明这种方式有很大的风险。

尤其，计划生育让这千年习俗消失了。此路不通。

国家养老

接着，迎来了「国家养老」；即年轻时交养老保险，等退休了，按月领取养老金。看起来一直都挺美好的。

可，养老保险本质上是 先交再返 的操作，且不说通货膨胀、货币贬值，

  • 如果交 100 返 90，这是扯淡；
  • 如果交 100 返 100，我干嘛不自己存？
  • 如果交 100 返 110 或者更多，这才有「保险」的意义。

普通保险，能在商业上成立，靠的是概率，是 个人交的少、赔的多；整体交的多、赔的少，即意外很少发生。一旦发生意外，由大量投保人的钱来支付理赔。

而对于养老保险，概率实在是太高，尤其现在大家寿命越来越长，理赔几乎是必然。那逻辑上成立，靠什么？说的难听点：死得早。

比如，交了一辈子养老保险，在退休的前一天出车祸、脑溢血死了，那之前所有交的保险怎么办？遗产？显然，是给别人理赔。这种概率高了，模式也玩得下去。可，尴尬的是，大家死的越来越不早了…

当然，逻辑上成立还有一条：国家财政补贴。可，国家用来补贴的钱哪里来？羊毛出在羊身上，还是年轻人的税。与其交税，为什么不自己存起来？

当然，我不否认，国家能把民众的力量（钱）集中起来办大事，也确实做了很多好事。比如农村的 新农合，每年交几十、几百元，大病时能报销几千、几万，挺好的了（这里混淆了养老和医疗，可 对很多老人而言，养老问题，主要就是医疗问题）。可，既然养老保险是全国范围的事，就得从整体看，不能只看局部。

比较健康的模式，是退休时领的钱，是年轻时交的钱，国家再贴一点。而事实上，你周围见到的老人领的退休钱，是现在的年轻人交的钱。事实上，现在的年轻人交的钱，已经不够现在的老人领，这也是之前新闻上报道的有的省份养老保险基金亏空。

问题是，等现在的年轻人老了，去领谁的钱？自然是未来的年轻人。可，这不是明显的拆东墙、补西墙吗？很有点 资金盘、庞式 的味道。尤其现在老龄化已经不是趋势，而是事实。老人和年轻人的比例越来越高，接盘侠快不够了（于是，放开二胎了…）

再说点医疗。去医院看病，医生说，有两种药，能报销的药，治疗周期长、副作用大、痛苦多，自费的药各种好，你选哪种？保险在这个时侯，最多保障的是 生命最基本的尊严。至于幸福，还是得自费。

其实，「养老保险」更应该叫「养老税」，这也是为什么是强制的。说起来是国家强制企业为你缴纳养老保险，是为你着想；其实是国家强制你纳税。养老保险，有可能，保障的是国家税收，用以避免社会上出现大范围的老无所依的极端事件，维护稳定。

养老靠自己

如果养儿防老和国家养老都不可靠，那靠什么呢？靠自己，靠「积累」和「保险」。

其实国家也希望你能自己解决养老的事，别给国家添麻烦。比如之前媒体上不断吹风的「以房养老」，不过你现在应该很少看到这种新闻了，因为走不通啊。情感上，显得子女不孝顺，难以接受。利益上，子女们盼望继承的房产，被国家收走了，难以接受。更别说，没房的是大多数，尤其是农村。

养老问题，核心就一个字：钱。年轻时多攒钱，能解决大部分养老问题。更准确的说，是资产，能不断增值的资产，而不完全是存在银行里不断贬值的存款。

保险，主要是指 商业保险，也是要有的。自己的积累主要是解决日常基础性需求，商业保险主要是解决意外。富如马云，他买保险吗？我想是买的；他不买，公司和股民都不答应。

可，你不可能在媒体上大面积见到这种思想。一旦这种一切靠自己的邪恶思想蔓延开来，国家怎么收养老税；岂不是和万恶的资本主义无异，又如何体现制度优越、特色主义。

确实，本文调调比较阴暗。没办法，谁让我是广大人民，而非公仆或优势阶级。

突然感觉小号不保…

0815 - 三两句说清楚

发表于 2019-08-15 | 分类于 每天写一点

如果你正在做的事，不能用三两句说清楚，要么是你还没想清楚，要么是事情太复杂，超出了你目前的能力范围。

平时很容易做这样的测试，比如在和朋友聊时，用最简短的话，说清楚你在做的事。最好是和真人面对面时的聊天，而不是自言自语，或者看着镜子。因为看到真人时，镜像神经元等神奇的未知因素，会带来不可知的效果。
