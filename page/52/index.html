0326 - 寺庙与互联网+

发表于 2018-03-26 | 分类于 每天写一点

今天跟朋友聊天吃饭，说到一个有意思的话题：寺庙与互联网+

寺庙其实有很多痛点，举个例子：功德箱，也就是烧香时往里扔钱的那个箱子。一个很头大的问题：有人偷。比如，拿钩子进去钩、拿胶水进去粘、甚至直接把箱子搬走…

怎么办呢？最基础的，就是加装监控，但其实效果不佳。也有寺庙直接粘支付宝二维码（好吧…），但好像又少了扔钱（好吧，我总感觉是扔钱，不是捐钱）的仪式感。

怎么既解决收钱方便、不怕被偷，又不影响扔钱人的心情呢？这其实是个问题。具体的方案我就不展开说了，大家可以自己发挥。

这，仅仅是寺庙的一个痛点，其他还有很多（好吧，说实话，在听朋友介绍之前，我都不知道…）

另外，从信徒的角度，也有不少需求。比如，想求子，哪家寺庙灵验？（感觉很像想看病，哪家医院、哪位医生医术高明）

可见，无论是 B 端（寺庙）还是 C 端（信徒），这里面都有很多可做的事。尤其，这是个小众、未被发掘的蓝海，很值得玩味。

0324 - 切回主战场

发表于 2018-03-24 | 分类于 每天写一点

今天，手上的项目告一段落，终于可以切回改动自己的 App 了。

这些天真是难受，总是被用户报问题，说白了就两个问题：

  • 腾讯 OCR 服务并发太差了，时不时出现无法识别的情况，严重影响 iText 用户体验。
  • 自己国外的服务器莫名被那个了，无法访问。进而导致 iPic 无法上传图片至微博图床。

庆幸的是，还好有 iPaste，可以快速回复这些完全类似的问题，不然总复制粘贴，更加痛苦。

0323 - 做自己能掌控的事

发表于 2018-03-23 | 分类于 每天写一点

不喜欢跟着别人走的感觉。

做事，就要做自己能掌控的事。就算不能掌控，也至少有一定的影响力，而不只是别人想法的实施者。

要掌控，一方面要有想法，既自己知道要做什么。也要有能力，既把它做出来；或是把会做事的人聚集起来，把它做出来。
