0121 - 锚定效应害人

发表于 2021-01-21 | 分类于 每天写一点

在交易里，锚定效应是非常害人的。

比如，股价前期最低到 100，现在跌到了 105，感觉差不多了，于是入手。谁知，市场不言底。买了之后，股价狂泻。

比如，之前自己是 60 买的，后来跌至了 50，又反弹至 58。自己对 60 这个成本价印象太深了。反正现在就差 2 点，等反弹到成本价，我再平仓吧。结果，价格又继续大跌，失去解套机会。

当然，锚定效应也有好的一面。只是很可惜的，好的一面是：反着用。在交易市场里，亏钱的是大多数。既然想赚钱，关键时刻，就要跟大多数反着来。大众会被锚在什么地方，你在这个地方，就可以反向操作。

这说得有点大了。往小了说，就是自己不要把自己锚进去了。比如，开仓后，忘掉成本价。加仓减仓还是平仓，完全只根据市场的走向决定，与成本价是多少、赚了多少、亏了多少，没有关系；自己不应该陷在这个没有关系的羁绊中，用心听取市场给的信号，据此而动。

0120 - 小聪明与大智慧

发表于 2021-01-20 | 分类于 每天写一点

交易世界里，有很多小聪明与大智慧的选择。

比如，猜顶摸底，就是小聪明。蒙对了，能小赚一点。可一旦做错了，却会大亏。更关键的，方向做反，会错过一大波行情。

这时的大智慧，是顺着趋势做。涨到高位，依然敢做多；跌到低位，依然敢做空。

总在小聪明里打转，既浪费时间精力，反复小止损也可能累加为大止损。并且，因为多次止损，而丧失交易信心。总的来说，就是小聪明不可取，要追求大智慧。

0119 - 无谓的细节

发表于 2021-01-19 | 分类于 每天写一点

最近国家要求返乡人员需持有 7 日内核酸证明。

这样的政策，当然有很多漏洞、诸多问题。但意思很明确，傻子都能看明白：过年就在原地呆着，别乱跑。在中心思想这么明确的前提下，还去纠结细节，要么是实在想回家而遇到困难，要么就是糊涂、小聪明、有事没事就想辩几句。也许就算辩赢了，他还是不回家，何必呢。

一眼望穿、直击重点，这是种能力；不要陷在不必要的细节之争中。

0118 - 每天都有人发财

发表于 2021-01-18 | 分类于 每天写一点

有人彩票中奖，有人交易大赚，有人做生意接到大单…有无数多，每天都有人发财。

这些事情，听听就好，跟你有什么关系呢？最多是碗鸡血。

关键的，可别把自己带入了，把自己想成了主人公。抛下自己手上的西瓜，去抓别人手里的玉米，就想赚个快钱。

哪有那么容易。也许，你进场，就是去接盘，就是当韭菜的。

最多修正下自己的方向，还是要低头走自己的路。
