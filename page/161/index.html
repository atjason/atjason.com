0621 - 捕捉与等待机会

发表于 2021-06-21 | 分类于 每天写一点

以动物捕猎来举例。

比如老鹰，一直在天上盘旋。看到猎物，一个俯冲。这叫捕捉机会。
比如鳄鱼，长期潜伏在水里。等猎物来喝水时，一个猛扑。这叫等待机会。

在交易的世界里，绝大多数人，每天都在市场里捕捉机会。兢兢业业的盯盘，生怕错过机会。这样的机会，都是小机会。当然，不排除累积小盈利成大盈利。只是，一方面性价比未免不高。另一方面，对身体、反应速度、执行力、反人性止损等多方面，有相当高的要求，并不是普通人能做到的。

更好的方式，其实是半年不开张、开张吃半年式的模式，像鳄鱼一样。平时没有大机会，死等。机会来了，迅猛扑上去。坚决咬住猎物，直到彻底拿下。

0620 - 印证

发表于 2021-06-20 | 分类于 每天写一点

如果自己思考得到的东西，和牛人的认知相同，或者多位牛人说的相匹配，那这个东西，大概率是正确的，也是自己更愿意相信和坚持的。

交易是混沌的世界，观点无数多，又无从验证。只能从逻辑的角度推理，从实际的结果验证。有多方面的印证，观点更可信。

0619 - 政策的另一面

发表于 2021-06-19 | 分类于 每天写一点

最近有几个跟三胎相关的话题，放一起看，比较有意思。

大背景是大家不愿生，当家的很着急，那咱就想办法吧。

你不是说，生孩子就没法上班了吗？好，我可以规定企业必须延长产假、产假必须发工资、还有哺乳假什么的，全配上。怎么样，保护妇女了吧？其结果会是怎样呢，忍不起我躲得起，企业更加不愿意招女员工，女性更难就业。

你不是说，课外培训负担太重吗？好，我成立专门的部门，严禁课外辅导。如果培训机构全关门了会怎样？富人请私教，穷人瞎玩，其差异可想而知。

你不是说，高考只看分数不合理吗？好，我可以评测各方面特长。问题是，你有钱有时间去培训孩子的特长吗？你有放得下钢琴的大房子吗？每年暑假能去不同国家的夏令营吗？

你不是说，带孩子太累吗？要是你只管生，有个社会机构专业负责养育小孩，多好。问题是，到那时候，都基因技术了，你还有资格生吗？甚至，你还会被生出来吗？

从大的层面系统性解决问题，极难极难。反过来，不管什么环境、哪个国家，都有混得好的、混得差的。与其抱怨环境、期待问题被解决，不如鸡自己，争做当下环境的强者。

0618 - 家庭的耳濡目染

发表于 2021-06-18 | 分类于 每天写一点

我妈在家里，聊天时经常说的内容，是什么菜多少钱、哪个菜场便宜、明后天超市搞活动、等等。你一听就知道，都是些家长里短的事情。

如果是让她带孙子，平时说的话多半跳不出这个框架。正所谓耳濡目染，她能给孩子提供的教育，也差不多是这个框架内的。当然，除了知识性的，还有比较人品、性格等层面的影响。

我其实想象不出，如果是高干、身家过亿等等这些跟我完全不在一个层面的家庭，他们的家庭范围是怎样的，会给孩子怎样的熏陶。结合 24 365 N 这样的时间量，其结果又会如何的天差地别。

0617 - 又做了个小工具

发表于 2021-06-17 | 分类于 每天写一点

最近，开发了个自用的小工具，用于辅助交易，感觉挺好。

一方面，开发过程用到了我大部分的技能，差不多算是拿出浑身解数。挑战成功后，自然是比较愉悦的。

另一方面，这个工具能在每天的交易中，切切实实的提供帮助。尤其，在使用的过程中，不断根据自己的使用习惯进行微调和优化。这种感觉，也是比较愉悦的。

愉悦就对了；哪有苦大愁深的状态，能赚到钱的。
