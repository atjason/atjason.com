1031 - 《天堂旅行团》

发表于 2021-10-31 | 分类于 每天写一点

惭愧，本周阅读时间不长，周末刷本小说增加时长吧。

《天堂旅行团》是之前就放书架里的，看了开头，感觉还行，就继续扫完了。

书中基本的矛盾是：一个受尽生活的折腾，选择放弃要自杀；一个小孩得了脑癌，却选择想要活下去。这样的思想碰撞，会让人好奇在一起会发生什么，是小孩激发活下去的勇气吗？

我觉得书中最神奇的转折，是最后主人公发现，小孩并不真的想要缠着他、让他带着去看演唱会或什么的，而是担心他自杀；在确认他不会自杀之前，不放心离开。看似是主人公在帮助身体弱小的小孩，实则是小孩在帮助心理弱小的他。

1030 - 父母的心理健康

发表于 2021-10-30 | 分类于 每天写一点

父母年纪大了，身体开始出一些大大小小的问题。除了身体上的，还会有些心理上的问题。

尤其是一种无力感。毕竟父母年纪大了，基本没能力去赚什么钱，学东西也比较困难。当自己遇到病痛时，会很惧怕，非常依赖于子女，而自身就非常缺乏安全感。

在和父母相处时，除了照顾身体之外，还要兼顾心理。包括日常的聊天、讨论病情，都要注意些。说者无心、听者有意，有可能不经意的一句话，会让父母不高兴好几天。

1029 - 止损要及时

发表于 2021-10-29 | 分类于 每天写一点

这周又出现了这个问题：有一天不能及时盯盘，持仓的一个股票快速下跌，比较预设的止损位还多。由于跌得比较多，舍不得下手止损了。同时，还会不断给自己暗示：这标的不错，再拿一拿吧。

人性实在是经不起考验。那么，就不要考验人性。开仓后，就设置好自动止损单。这样，不论是否能及时盯盘，都能及时止损；不要把小浮亏拖成大的，到了那时候，几乎没人能下得去手。

1028 - 一人一扇窗

发表于 2021-10-28 | 分类于 每天写一点

每个人都是社会的节点。多和一个人连接，就连续了一片节点，多开一扇观察世界的窗户。

这样的连接，最好的自然是线下。尤其，如果有多年、长期的关系为基础，这样的连续更紧密。

现实的情况，大家都很忙，又受限于各式各样的原因，天南海北的人要见上一面，有时真不容易。而要用这样的方式认识一个新朋友，成本则更高。于是，网络成了很好的替代。

纪录片也是很好的一种形式。通过拍摄、采访得到素材，再经过一些剪辑、加工，辅以自己的理解，就形成一个很好的了解新事物的方式。纪录片不仅是自然地理，也有很多人文社会的东西。这样的纪录片对于避免闭塞，是非常有帮助的。比如，我最近在 B 站看的《伟大的制造》，推荐大家去瞧瞧。

1027 - 等机会与找机会

发表于 2021-10-27 | 分类于 每天写一点

比如，已经有一套交易模式，怎么执行呢？

一种方式，就是先找优秀的公司，然后等这些公司的股价走出自己认识的模式再下手。这是等机会。

另一种，是直接全市场套模式。符合模式的，再去研究行业基本面什么的，觉得差不多就上车。这是找机会。

以前我是找机会，因为觉得效率更高。时间长了，也遇到一些问题，比如这个过程人会长时间处于紧张状态，过滤做减法比较累，零散的研究行业企业不够深入。

现在，我想试试等机会。先深入研究几个好的行业，找出其中优秀标的，然后耐心等。这样操作，人应该会比较淡定，有耐心也更有信心。看似慢、效率低，但拉长时间看，可能会越做越扎实。
