0611 - 信任的建立

发表于 2023-06-11 | 分类于 每天写一点

在处理用户有问题的硬件时，有这么两种选择：

  • 如果让用户先把有问题的退回来，修复之后再寄回去，用户觉得耽误时间，也担心我们收到后处理不及时、甚至不寄回去。
  • 如果我们先把好的寄过去，用户有可能收到后不寄回有问题的。

这其实就是个建立信任的过程。很多「凭啥」这类问题，也是如此。

0610 - 重新开始考虑美股交易

发表于 2023-06-10 | 分类于 每天写一点

暂定的计划是：

  • 在各大指数选择靠前的股票形成股票池
  • 粗筛一遍，去掉明显下跌、涨跌幅度很小的
  • 每周选出大概的股票，主要是首次突破回调，大幅上涨后回调形成小形态
  • 每周过一遍周选股票池。如果有机会，人工加条件单，之后考虑加入机器人

0607 - 蓝牙连接问题

发表于 2023-06-07 | 分类于 每天写一点

智能产品发布后，遇到了多种蓝牙连接问题。

比如，用户手机开启了蓝牙，却没有允许微信使用蓝牙，导致无法连接；

比如，安卓需要开启定位，才能使用蓝牙。理由是蓝牙可能间接定位；

并且，硬件问题，不只是软件设置就能解决。即使是软件问题，用户对手机的熟悉程度不同，定位和解决问题的难易程度也不同。
