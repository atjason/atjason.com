0306 - 交易的风格

发表于 2021-03-06 | 分类于 每天写一点

交易也有自己的风格。

类比足球，有种风格比较常见：防守反击。

这方面，意大利是最知名的。

  • 你来进攻，我防守很强，就不让你进球。
  • 而当你懈怠时、有漏洞时，我就趁机进攻，进球得分。此消彼长，最后我就赢了。

对应到交易中：

  • 行情不好的，我就及时止损，不大亏，抗住市场的攻击。
  • 一旦行情有机会，我就开仓、加仓、重仓，打平亏损，有所盈利。

当然，把市场比作攻击方，不是太合适。市场永远是正确的、不可战胜的。我们只能站在市场的一边，顺势而为。

0305 - 交易要克制

发表于 2021-03-05 | 分类于 每天写一点

交易一定要克制，耐心等到好的机会再开仓。

交易一些差的机会，看似是想赚更多，其实是消耗了大量的时间和精力。

尤其，当大行情来临时，如果持有的是不利仓位，很难把思想扭转过来，去反向建仓；甚至，会不止损，造成大亏。而这时候，如果没有仓位，则更可能抓住这样大的机会。

精力和金钱，是你的子弹。把有限的子弹瞄准西瓜，而不是芝麻。

0304 - 交易员的情感输出

发表于 2021-03-04 | 分类于 每天写一点

交易是孤独的。

长期的交易，大赚大亏、天人交战，内心积压了大量情感，是需要释放的。所以，你看尤其是中国的股民，特别喜欢跟人说股票的事。不论是交易的方法、赚钱亏损的经历等等，一聊个开始，就停不下来。

而，越是好的交易员，可交流的人越少。毕竟，人群是金字塔分层的；越优秀的人，数量越少。而高手平时又都很忙，也确实没太多闲散时间聊天吹牛。

越是优秀的交易员，越是需要在交易之外，找到情感输出的点。这可以是自己的兴趣爱好、美满的家庭、公益事业等等。

0303 - 为耐心付费

发表于 2021-03-03 | 分类于 每天写一点

符合规则的交易机会是极少的，需要像动物狩猎一样，需要极大的耐心等待。

问题是，人不是机器，是有感情的。这样漫长的等待，会让交易索然无味，也就坚持不下去。

这时，需要一些感觉驱动的、冲动型的交易。代价是可能亏损，收获是保持对交易的兴趣和积极性。

当然，这样的交易，适可而止，切不可重仓，更不能上瘾。

0302 - 《我们台湾这些年》

发表于 2021-03-02 | 分类于 每天写一点

一位台湾人，写的从 49 年至今的台湾演进。

在看这本书的过程中，我一直在找这个问题的答案：在从党统到党争的政治环境下，台湾是怎么完成经济起飞的？

2020 年，台湾的 GDP 在全球排名 22，是香港的近 2 倍。历史上更高，好像最高排第 14，亚洲四小龙。

大致的答案，台湾做了世界代工厂。当时，不论美国造、还是苏联造，都是台湾造。只是，现在这个世界工厂的位置，慢慢被中国大陆和东南亚一些国家取代。

书中有 1/3 以上的篇幅，是政治相关的；1/3 是民生。其中，很多是以作者的视角来描写，通过周围发生的事，来反映当时的大事件。毕竟，时代的微尘，落在个人身上便是大山，更别说时代的大事了。

政治是个很敏感的话题。我只说一点，台湾政治的开头，其实和大陆很像，党政、将国姓。只是后来，二者分化到不同的路线。邓和小蒋，都尝试中兴。只是，一个成功、一个失败。好在，小蒋做出了下界总统不姓蒋的改变，对台湾人是万幸。

这套书有两本。第一本我算是精读，第二本就是跳着看的了。
