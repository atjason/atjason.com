1124 - 多和不同的人接触

1123 - 就怕流氓有文化

发表于 2018-11-23 | 分类于 每天写一点

做产品，「被致敬」是很常见的。可总会碰到很恶劣的行径，让人很恶心。

怎么办呢？

我现在能想到的办法有这么几个：

  • 法律手段
    • 如通过著作权、版权、商标等方式保护自己。
    • 但实际上，对于小的产品，这点基本没用。
  • 技术门槛
    • 简单的说，就是做的东西有一定的技术难度，别人不容易模仿。
    • 对于个人、小团队，这点也很难。而且，像网站这样的产品，代码几乎是半公开的，很难防护。
  • 品牌 & 口碑
    • 这个就需要长期积累了。
    • 比如，已经开发了一款深受用户好评的产品，再推出另外一款，大家就更容易接受。

1122 - 开始小程序开发

发表于 2018-11-22 | 分类于 每天写一点

看了几天官方文档后，今天开始写小程序的代码了。

作为练手项目，打算做个杭州公交查询的工具。核心功能是：一眼看到上下班的公交车还有几站到站。

之前收窄技术面至 js 相关，而小程序主体正是 js，刚好。不过，我对前端开发不太了解，这部分进展估计会比较慢。

另外一个比较坑的是，实时公交信息基本是没有 API 的，暂时打算逆向一个 App 的接口，这个做法商业上不太合适，我得再考虑考虑。

对了，小程序叫什么呢？杭州公交车。

1121 - 为时已晚？

发表于 2018-11-21 | 分类于 每天写一点

时不时会有这样的顾虑：现在才开始学这个，是不是太晚了？

晚不晚，是相对的。相对于什么呢？通常是相对于年轻人，相对于已经很牛的人。确实，跟这样的人存在竞争。但实际上，真正竞争的，还是自己。自己是否真的喜欢、有兴趣，能否坚持，能否做的好。这些因素，比自己假想的对手，更重要。

晚不晚，其实还是该不该的问题。该不该，其实就是：除此之外，是否有更好的选择。如果有，就做更好的；如果没有，就做现在的。如果不清楚，抱歉，问你自己吧。
