1129 - 启动效应

发表于 2019-11-29 | 分类于 每天写一点

请加一个字，组成一个词：

  • 运动__
  • 游泳__

我想，绝大多数人想到的都是：运动场、游泳池。这就是启动效应的一种表现。

再比如，当你看到「年老」这个词，或者接触到相关概念时，会让你在接下来走路时，不由自主地放慢脚步。

启动效应，就好像触发了一个开关、启动了一个程序。接下来发生的事，就好像是这个程序的体现或延伸。

1128 - 简单的努力没有用

发表于 2019-11-28 | 分类于 每天写一点

很多事情，都较高的阈值；年纪越大，所接触到的事越是如此。

简单的努力，如果没达到阈值，之前所有的努力，全是白费。

小孩子有一点点进步、学会一点点东西，就会被各种肯定，以及奖励。而对于成人，必须有长足的进步、有很大的进展，才会有所回报。和别人做的一样、甚至好一点点，很多时候，都是不够的。

有自己的兴趣，能不费力就能进入注意力集中的高效状态（文雅点说叫心流），并在此基础上持续的努力（是真的努力，不是做做样子），才「有可能」有满意的回报。

最近的我，跟上面说的，好像都是反着的。说是在搞量化交易，实际只是在门外踱步；甚至，可能连门都还没找到。

1127 - 喧嚣与寂静

发表于 2019-11-27 | 分类于 每天写一点

当初，人们第一次尝试在大西洋底铺设电报电缆时，英国与美国人民无比热情。在海岸边停靠的铺设电缆的大船边，有无数小船夹船欢送，岸上锣鼓喧天。

不出意外，第一次失败了。
不出意外，接连几次都失败了。

慢慢的，人们的热情被耗尽。组织这件事的公司，甚至准备把剩下的电缆卖了，平摊损失，大家散伙算了。好在，其中一个领导者意志坚定，终于促成最后一个铺缆尝试。

不出意外，这次成功了。

两艘大船分别从大洋两岸出发，在大洋中间某处相遇。把两段电缆连接后，沉入海底，然后返航。在这本应激动人心的历史性一刻，船员及工作人员一点都不兴奋。长时间的工作和多次失败，已经让他们提不起兴趣。历史性的一刻，便在这种寻常又不寻常的寂静中度过。

不知是偶然还是巧合，历史上很多喧嚣，好像都只是铺垫。真正关键的瞬间，好像反而比较平淡。可以说是水到渠成，也可以说是否极泰来。

我也有些类似的经历。比如，经常很长时间的折腾，终于搞定某一关键性的一步。而那时，可能已经是深夜，周围很安静，无人可说。自己内心的欣喜，被强行压平，没入黑暗。

1126 - 贩卖焦虑，还是情怀

发表于 2019-11-26 | 分类于 每天写一点

如果不买，你会失去什么，这是贩卖焦虑；
如果买了，你会得到什么，这是贩卖情怀。

有用的东西，大概可以算到情怀里。

于是，能卖的东西，大概就是这两类。

不过，既然是卖，就不要有太强的道德判断、莫名的优越感。

卖不出去，说什么都晚了；
卖出去了，说什么都是对。

1125 - 不要想着抄底摸顶

发表于 2019-11-25 | 分类于 每天写一点

还是这句话，今天看到一个不一样的角度：给对手留余地。

你要买在最低处，谁卖给你？
你要卖在最高处，谁来买走？

把对手赶尽杀绝，这游戏就玩不下去了。况且，你也没这个本事。凡事留余地，日后好相见。
