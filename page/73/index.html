0614 - 思考最近的交易

发表于 2022-06-14 | 分类于 每天写一点

只要复盘、只要思考，就有收获：

  • 考虑只做一个方向，思路更清晰。
  • 考虑只做螺纹。螺纹和热卷很像，强弱切换，选不好，干脆不选。
  • 定一个极值，突破一定做，避免错过机会。此时，一般离均线有段距离，可用均线止损。
  • 在此之前，如果有更好的形态，比如形成一个次高然后突破，则可以在极值之前提前进。
    ‌
    简单说就是：
  • 只做螺纹单方向；
  • 拐点信号或极值；
  • 轻仓试错盈加仓。

0612 - 交易与时间

发表于 2022-06-12 | 分类于 每天写一点

对于交易新手，在练习时，一方面要低频，降低本金和情绪消耗的速度。

另一方面，任何领域，都需要大量的刻意练习，才能有所精进。

在低频的前提下，如何才能完成大量交易练习？答案只能是时间。

交易，极难速成。一定要在时间中，经历情绪起伏、时悲时喜、希望失望、资金涨落，才能修炼升级蜕化。

给交易以时间，给时间以意义。

0611 - 自我思想重建

发表于 2022-06-11 | 分类于 每天写一点

以下为全文转：

所有的方法都是简单易学的，唯有思维的转变是最难的，打破一个人的思维是很痛的…

凡是以为学别人几个绝招就能成功的人，都是精准的韭菜苗子。因为，没有永远正确的方法，市场环境一变，过去成功的方法，就变成了今天失败的原因，这就是很多高手保持复盘总结的原因…

比如十几年前，别人做淘宝可以成功，你今天再做淘宝就必然失败，因为淘宝日活只有两三干万，已经不再是快速增长的行业…任何市场都一通百通…

所有的结果都是来自于思维的转变，这样你才能做对选择，选对方向…

方向上你要坚定不移，但是在打法上你要千变万化…今天趋势线没什么用就换黄金分割[捂脸]

正如邓爷爷说的：坚持社会主义方向100年不变，但是战术要不断变化，白猫黑猫，抓住老鼠就是好猫…

你不能死守着过去的经验不变呀，就像诺基亚一样，最后不就完蛋了吗？

人就是一个打破自我思想的过程，只有彻底颠覆固有的观念，你才能更上一层楼…

如果你脑袋里的东西是对的，为什么你口袋里没有得到你想要的？

如果你一直以为你的想法是对的，你目前的生活状态骗不了人…

所有高手都必备的思维，就是批判性思维…

什么是批判性思维？

很多人都听过批判性思维，但大部分人对批判性思维的理解都是错的…

他们认为批判性思维就是不要轻易接受别人的意见，找到对方观点的漏洞，这导致很多人把自己变成了一个杠精。事实上这是错误的理解…

批判性思维的核心不是批判别人，而是批判自己，反思自己…

为什么我们要批判自己呢？因为从我们出生开始，我们的大脑就出现了认知的盲区…

事实上，你大脑中的想法并不是你自己的，而是从你出生开始，就由你周围的人给你植入的。这就导致了一层人只能被种植同一阶层人的思想，重复这一阶层人的命运…

如果你被种植了底层穷人的思想，那你不批判自己，你不改变的话，你不就永远待在社会底层了嘛

打破阶层固化的核心，是你要先打破你阶层固化的大脑，把你大脑中那些底层阶级的思想剔除，再重新植入顶层阶级的思想…

而这恰恰是最难的，因为当多数人看到和自己过去观念不同的思想时，第一想法就是不接受，甚至骂街…

只有否定自己大脑的思想，才有机会打破阶层思想固化的顽疾。这个过程，其实就是“解放思想”的过程…

当你不断拿自己的思想进行反思、批判的时候，才代表你有真正的进步…

打破自己的思想再重建，需要有一颗谦卑而开放的心。所有的高手都是自我觉察，自我反省，时时认错，随时推翻自己，才能加速成长…

0610 - 交易是自己的事

发表于 2022-06-10 | 分类于 每天写一点

我越来越觉得，交易还是自己的事；吹牛可以，交流极难，基本是自说自话，尤其当自己差不多有成型的交易系统时。

一个人的信心，来自于自己的交易手法、和手法曾经带来的盈利。

要从交流中提高自己，首先是否定自己的交易手法，至少相信是有明显可改进的地方。而这相当于否定自己的信心来源，这个，不容易。能随时保持空怀心态，不容易。

以及，为什么要不断改进交易系统呢？差不多就行，更多的是执行。
