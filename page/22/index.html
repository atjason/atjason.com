0424 - 代偿

发表于 2021-04-24 | 分类于 每天写一点

今天又接触到了这个概念：代偿。先看看一个定义：

代偿，生理学上的意义是指人体的一种自我调节机能。当某一器官的功能或结构发生病变时，由原器官的健全部分或其它器官来代替，补偿它的功能。一些例子：

  • 如果一个人的背部脊椎向右倾斜，脖子的脊椎就会向左倾斜，以保持平衡。
  • 如果一个人全盲，他的听力通常会很突出，以使他更好的适应周围的环境。
  • 膝关节中的软骨及半月板磨损时，骨头上会长出骨刺，以增加关节稳定性。

这些代偿，通常是盲目的、生理性。有些时间，短期看有好处，却有长期的、更严重的问题，比如关节中的骨刺。

除了生理性的，人的能力、性格等方面，也会有代偿现象。比如，处于弱势的人，在受到欺负时，可能会有阿 Q 精神来自我安慰，直到心理上的自我保护。

我们要注意的，就是要意识到代偿的存在和其盲目性，做有意识的筛选。比如，用听力来弥补视力，就有好的代偿。

0423 - 多头、空头、滑头

发表于 2021-04-23 | 分类于 每天写一点

看到一个观点，觉得很有意思：

在这市场里没有所谓的多头和空头。机构为什么要做多，其目的是为了以后的做空，之 后为什么要去做空，也是为了将来的做多。在这一多一空之间完成利润的转移。所以说，我们在这市场里要做滑头。

0422 - 做长期正确的事

发表于 2021-04-22 | 分类于 每天写一点

又是句正确的废话。怎么理解呢？举个反例吧。

比如，交易中，时常想赌一把。

  • 明明在下跌，却想抄个底。
  • 明明持仓是亏损的，却想抗到回本。

这样的事，哪怕短期、单次能获利，长期来看是没有意义的。能重复做、反复盈利的交易，才是值得做的交易。做其他事，也是类似道理。

0421 - 专注一种交易方法

发表于 2021-04-21 | 分类于 每天写一点

不同的方法，对于同一种的行情的处理方式，是不同的。进而，其盈亏结果也是不同的。

拿炒单和波段来举例。比如，在 4328 做多，价格上涨至 4336 后回落至 4330。

  • 如果是炒单，应该已经平仓，能赚几跳的钱。
  • 而做波段，为了抓可能的波段，在建仓初期，止损是放在保本甚至是亏损位置的。对于这样的走势，不仅不赚钱、甚至可能亏钱。

这时，做波段的人，容易羡慕炒单的：还是落袋为安好。

这样的思路是不对的，既看到已经发生的，还要想想没有发生的情况。比如，价格如果继续反弹，炒单不一定能找到机会重新开仓、或者很难，进而不容易抓住完整的波段。而做波段，则可以抓住这波上涨的大部分，盈利更多。

另外，毕竟波段消耗的精力小。如果单一品种没什么机会，可以同时关注多个品种；这点，是炒单无法做到的。

因此，每种交易方法都有其优缺点。不要当自己的方法遇到命门时，就羡慕别的方法。而是找到自己真正认同的、适合自己的方法；好好了解它，尤其是它的命门；然后，好好利用它，找到适合它的行情，反复使用它来赚钱。
