0504 - 保守治疗

发表于 2021-05-04 | 分类于 每天写一点

人到中年，越来越频繁听到这个词：保守治疗。不是对自己，就是对父母。

尤其是老人，心理是很纠结的。一方面，老年人其实挺害怕给自己的子女添麻烦。另一方面，也很想把自己的病彻底治好。他们很喜欢听医生说你：这个问题不太严重，可以先手术，先开点药吃吃吧。这样两边的都照顾到了，心里就舒服很多。

其实，所谓保守，就是放任。医疗在这个时候，就是安慰剂。

0503 - 经常做点确定性的事

发表于 2021-05-03 | 分类于 每天写一点

交易，有太多不确定性的点。什么时候买、什么价位卖、买卖多少、什么品种等等。

总在不确定性中混，人会像浮萍一样，没有根的感觉。

做点确定性的事，人的掌控感、安全感，会好很多。

可以是交易中的事，比如复盘。也可以是交易之外的事，比如软件开发。软件是极其确定的事，该是什么就是什么，不模糊、不撒谎。

0502 - 节奏感

发表于 2021-05-02 | 分类于 每天写一点

五一放假，好像我没什么感觉。因为不上班、不通勤、没有朝九晚五，对于假期，就没什么感觉。

不过，这样也有一些问题。比如，天然缺失了节奏感；没有被逼的忙碌，也没有了假期巨大落差下的放松。

驾驭自由，也是不小的挑战。

0501 - 软件的粒度

发表于 2021-05-01 | 分类于 每天写一点

最近买了一款软件，关于它的模式，让我有点思考。

这款软件的功能相对比较多，定价也比较高。而我只需要其中很少的功能，高的定价就有点难以接受。而这个领域好的软件并不多；以我有限的了解，这是唯一一款满足我需求的，其它同类的差距很大。在这样的情况下，我还是花钱买了。虽然，买得并不舒服。

对于这种情况，如果能推出按功能付费，那我付的钱会少些，也更乐意付。但其实我知道，从开发的角度，这不现实。实现更细粒度的收费，会增加软件开发和维护的复杂的度。和换取的收益相比，得不偿失。

对于目前的现状，之所以能成立，一方面是这个领域竞争不激烈，售价较高；另一方面，是软件还没有成熟到可以无限分割的程度。像水电这种基础性的服务，是可以非常细分的。

0430 - 新闻还是洗脑

发表于 2021-04-30 | 分类于 每天写一点

微信上，我有关注「腾讯科技」公众号。印象中，最近很长一段时间，经常报道特斯拉相关的新闻。尤其要注意的是：几乎全部是负面消息。

如果只有这一个信息源，很容易得出结论：特斯拉真是家傲慢的企业；特斯拉这车肯定有问题，谁买谁傻。

问题是，事实真的是这样吗？反过来说，如果公司这么差、问题这么多，怎么还不关门倒闭？

很明显，这家媒体在报道这家公司时，是不公正的。

类似的，前阵子很热的 BBC 报道武汉的阴间有法，也是不公正的。

问题是，我们老百姓，怎么能辨别公正不公正呢？除非是自己内业的事，一般的都是别人家的事，我们并不专业、并不了解，很容易被媒体带着走。在这样的情况下，看新闻 = 被洗脑。

我觉得，很关键的一点：媒体在不向金主谄媚的情况下，能很好的生存，其报道的客观性就相对好些。比如，一家靠向会员进行收费阅读的媒体，就比发公关稿、发广告的媒体，相对更靠谱些。毕竟，前者是站着挣钱。
