0603 - 很难买到满意的东西

发表于 2022-06-03 | 分类于 每天写一点

最近买了几次东西，都不是很满意。

买个投影仪，要么清晰度不够，要么音响效果差。

买个路由器，要么颜色不好，要么个头太大，要么穿墙不行。

大部分时候，加价可以买到满意的。而有时候，有钱也买不到想要的，因为可能就没这样的东西。

这个过程，需要折腾。所谓折腾，就是一步步明确自己的需求、已有的解决方案及其优缺点，然后用恰当的方式实现。

0602 - 选择路由器

发表于 2022-06-02 | 分类于 每天写一点

家里的华为 WS5200 路由器换到门口，结果卧室的信号很差，只有 1MB。趁着 618 买了另外两款华为路由器：AX3 Pro 和 AX6，想改善一下。

先说 AX3 Pro，实测卧室速度提高到 4MB。不过，比较奇怪的是，300MB 的宽带，在路由器边上，只能测到 60MB，怀疑碰到了次品。

再看 AX6，实测卧室速度提高到 60MB，最高出现过 160MB，改善非常明显。卫生间这个角落，信号也不错。整体很满意。

当然，二者价格分别是 349 和 549，几乎不是一个档次的。

不过，AX6 有两点有点纠结：黑色，看着比较突兀；个头比较大，比 AX3 大一圈。纠结要不要等下白色。

另外，最近对路由器又有了新需求：有 USB 接口，进而可以开放出 Samba 或 DLNA 服务。

0601 - 保险、交易与大数定律

发表于 2022-06-01 | 分类于 每天写一点

而对一位 60 岁的人，要判断其 1 年后的存活概率，是很难的，因为个体差异巨大。

而要判断 1 万甚至更多 60 岁的人，要判断其 1 年后的存活概率，就相对准确多了，这就是大数定律。

保险业的核心之一，就是利用大数据、大数定律，精算出概率，制定有利于自己、又能被大众接受的保险条款。

同样的，对于交易，某一次的开仓，其结果是盈是亏，是很难判断的，因为走势的不确定性。而如果长期执行一套正期望预期的交易系统，其结果一定是盈利的。

这其有几个关键点：理论上的正期望、长期、一致性执行。其实，第一点很容易，网上随便就能找到经典的交易策略，都是有正期望的。关键在于，自己能否接受、能否驾驭、能否始终如一的长期执行。

0531 - 波动率

发表于 2022-05-31 | 分类于 每天写一点

在波动率低的时候，才开始等信号交易，本质上是在赌波动由低到高、由收敛到发散的过程，长期看是有概率优势的。

发散之后再发散的概率，自然低于收敛之后的发散。而所谓的行情、趋势，不就是发散。

如果是单品种，可以在波动率低时，仓位重些；否则轻些。
如果是多品种，可以只在波动率低时，才考虑交易。

如果判定波动率变低？K 线开始接近较大均线，就是一个简洁的标准。

0530 - 交易中的低频与情绪

发表于 2022-05-30 | 分类于 每天写一点

如果交易次数多，总会遇到见鬼的行情；明明好的信号，行情就是反向一根针所止损，那能怎么办？

关键一点在于，这样的事情会影响情绪。这样的情况出现几次，会影响情绪的稳定，会怀疑交易系统是否需要调整，甚至怀疑是否真的需要止损。

试问，当情绪出现这样那样的问题，当遇到一个信号不好、但事后看能走出大行情的信号，你还敢做吗？如果错过这样的信号，交易的整体预期由盈利转亏损，还能抗得住吗？

所以，低频很重要的一点：能保护情绪。我们都是主观的、有情绪的，不要假想自己有很强的情绪控制能力，不存在的，不要挑战人性。我们能做的，就是好好的呵护情绪，避免轻易的被打乱。

低频、交易次数少，意味着即便出现见鬼行情，也有足够的时间安抚情绪、恢复平静，不容易暴走。
