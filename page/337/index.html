0216 - 财不外露，才要露

发表于 2019-02-16 | 分类于 每天写一点

运营宣传很重要。

除了绝对隐私，其他的东西，基本越多人知道越好。甚至，有时为了效果，可以牺牲隐私，比如明星的各种八卦。

多了些露出，就多一些合作的机会。虽然并不是每个机会都需要把握，但总归是有比没有好。

现在自己的露出方式还太少，要改进；当然，更重要的是，有要才。

0215 - 商业嗅觉

发表于 2019-02-15 | 分类于 每天写一点

最近什么火？什么最能赚快钱？接下来什么可能会火？

能回答这些问题，就是商业嗅觉很敏感了。

这其中，一个是对流量的敏感，什么东西突然出镜率变高了，讨论的人多了。不是等量上来后的后知后觉，而是能感觉到趋势的微妙变化。

再有一个，就是对人性的洞查，尤其是贪嗔痴。太阳底下无新事，很多东西其实是已有东西的翻版。比如，之前线上娃娃机很火，而最近火的是线上玩游戏免费得口红，其实本质上路子差不多，只是多了些变种，引入一些新的玩法。

我在这方面就比较迟钝了，常常是看到什么火了，只有一拍大腿的份。感觉这本事有点玄学，不是按什么套路就能精进的。

0213 - 可交互的视频教学：Scrimba

发表于 2019-02-13 | 分类于 每天写一点

今天发现一个好玩的网站：Scrimba

比如，你可以打开这个介绍视频：https://scrimba.com/c/cKp6LhL

在视频播放过程中，你可以随时编辑正在演示的代码；一旦编辑，视频即暂停，进入 ‘playground mode’；等编辑结果，还可以恢复视频播放。

这在实际学习中还是挺有用的，比如碰到一个相对难的概念，自己可以通过实际动手的方式，帮助理解。

好奇这个网站是如何实现的，大体应该是记录了屏幕操作、文件变化、声音，然后用时间轴同步。

0212 - 事业，指什么？

发表于 2019-02-12 | 分类于 每天写一点

我想，人们所说的事业，多半是指能持续赚钱的东西。这东西多半是指公司，而关键的还是持续赚钱的能力。

可，有钱之后做什么、为了什么？更多的钱？用繁忙的工作来掩盖内心的空虚？如果迟迟赚不到钱，又该用什么来支撑疲惫迷茫的心？

赚钱，最好是内心满足后的副产品。
