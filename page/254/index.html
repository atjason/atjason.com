1221 - 情绪的影响

发表于 2019-12-21 | 分类于 每天写一点

淘宝上买东西，最后卖家说不能当天发货，瞬间感觉不爽，晚收货一天，感觉会错过很多东西。但实际上，接下来几天因为别的事，可能并不会用到这个东西。这种怅然若失的感觉，完全没必要，只是幻相。

交易时，比如说做多，有了浮盈，于是加仓。由于加仓拉高了仓位成本，一个价格回调，便会浮盈变浮亏，甚至触发止损出局。更可气的是，接下来价格立马又涨上去了。不仅煮熟的鸭子飞了，还顺手给了一巴掌，如何不气？

心态还是要放平，浮盈不是盈。就像是一位美女在你身边坐了一会，你能说美女是你的吗？过了一会，美女起身离开，你能说失去了这么美女吗（抱歉有点不尊重女性的意味，能理解我意思便好，女权见谅）？你这样邪恶的想了，不该给一巴掌吗？

这么一想，是不是就平衡了？更关键的是，这事已经发生，是过去的事，已经成了幻相。过去便要放下，不能因之影响更重要的当下。

1220 - 身体的信号

发表于 2019-12-20 | 分类于 每天写一点

记得有个实验是这么说的，如果你在两个选项中纠结，就抬起一只胳膊并保持，同时全心全意想着第一个选择。这时，请另一个人用力按下你的胳膊。然后，换另一只胳膊和另一个选择。按下哪只胳膊所需要的力量越大，说明你其实更中意这个选择。

做交易时，会需要做大量的选择，比如是买还是卖、买多少、什么价位平仓等等。当然，大部分选择还是要靠分析、靠判断。但，经验、直觉这些模糊又显得有些神奇的因素，有时也能帮助做出快速准确的决定；身体的信号便是其中之一。如果一个决定让你感到很紧张、甚至坐立不安，很可能，这就不是一个好决定。

1219 - 郑重推荐：云聪的金融交易技术分析

发表于 2019-12-19 | 分类于 每天写一点

郑重推荐：云聪的金融交易技术分析

目前是仅在 YouTube 上发布的系列视频：
https://www.youtube.com/channel/UCg9zI64pEB_3xwJPur6isiQ

已经有 150 个左右的视频，最近几天突击看，已经扫到 105 期，计划本周全部看完。

对我来说，最大的帮助是，云聪老师 全面系统的介绍了不同交易方法，这对于初学者，是很有价值的。如果说交易是本很厚的书，老师相当于提供了一个目录，让我知道这个行业有哪些门类、有哪些要学、我可能对哪个方法更感兴趣、更适合我等等。有了这样地图式的介绍，就更容易走出交易的迷宫。

接下来就是 交易心态的讲解。交易，最好是有名师带。如果只是自己看书、学习、练习，太累太苦。尤其是如果遇到连续的、大的亏损，对心理的打击是很大的，需要有外力了抚慰。而现实中，绝大多数人并没有这个条件，这又很难通过看书等方式学习。云聪老师非常坦诚透彻地介绍了自己多年交易的心得、交易理念，尤其是踩过的坑、亏损的经历，是非常罕见、很有用的东西。

当然，老师不仅提供了交易的目录，也很详细地介绍了目录中的各项内容。基本上，仔细听完视频，总结出几套可盈利的交易系统，是毫无问题的。也正是看完这么多视频，我也更有信心走交易这条路。

其他还有很多可点赞的点，比如老师的真诚，化繁为简的功力，多个交易品种的经验，等等。话不多说，但凡你炒股做期货，必看。

1218 - 减肥与交易

发表于 2019-12-18 | 分类于 每天写一点

减肥，道理很简单：少吃多运动。难在执行。

交易，道理很简单：赚大亏小；机会来了就去试，盈利了就坚持持有，亏损则坚决止损。难在执行。

还有个共性：反人性。

吃是人的天性，多吃、吃甜食、吃完休息这些尤其是天性。少吃多运动，则是反人性。

赚点小钱就跑、亏得再多也要死抗，这几乎也是天性，却是交易大敌。
