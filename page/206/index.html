0227 - 多给孩子一些空间

发表于 2021-02-27 | 分类于 每天写一点

今天，娃在玩一种玩具时，想要把另一种玩具拿出来一起玩。

我想当然的说：你先把这个玩具收起来，然后再玩另外一种。目的自然是希望娃能养成收拾玩具的习惯；同时有个好的玩的区域，不至于到处都是散落的玩具。

然后他就说，我想把两种玩具一起玩，拼搭出不同的效果。显然，他这个理由更合理，我也没有拒绝的理由。于是，我给他搬出来了。

我们都希望孩子有想象力，那就不要给他太多局限，给他们一些空间。

0226 - 在系统层面思考

发表于 2021-02-26 | 分类于 每天写一点

最近，交易方面，一直在做一些具体的事，包括制定交易计划、交易、不能止损等。

虽然，这些都很重要，但我好像陷进去了。尤其，当遇到问题，比如不能止损、亏损抗单时，就更乱、更迷茫。

一方面，我不断在花时间研究具体的环节，比如止损。但另一方面，我开始意识到，我需要在更高的系统层面进行思考。

交易是个系统工程。那么，交易要成功，就需要完善的交易系统。时不时的，比如每周，复盘这个交易系统；根据自己这一周新学到的东西、在交易实践中感悟到的东西，去丰富和改进这个系统。只能这样，才能不迷失在细节中，才不至于迷失方向。

具体的操作层面，我采用大纲和思维导图的方式，把交易涉及的方方面面，从大到小、从粗到细进行分类，把自己能想到的点都塞到这个文档中。然后，不断去调整、优化各个节点。

尤其，开始时，这个系统会越来越复杂，涵盖的内容越来越多。就像读书一样，书越读越薄。但到后来，我希望这个系统越来越精简，去掉不必要的，仅留下最适合我的；就像是书越读越薄。

0225 - 交易中的止损

发表于 2021-02-25 | 分类于 每天写一点

本周的交易，不及时止损、亏损抗单、亏损加仓，造成了比较严重的亏损。我还是没有能力止损，还需要继续修练。

网上学习了一些关于止损的思想，整理了一部分：

趋势与时机

  • 趋势，是大的时间和空间范围内，价格的走势。
  • 时机，是小的时间和空间范围内，价格的趋势。
    • 本质是一回事，只是时间和空间的范围不同。
  • 止损，通常是时机抓的不对，是小周期的预判错误。
    • 不要因为止损、尤其是多次止损，而轻易怀疑对大周期也即趋势的判断。

开仓与止损

  • 止损是系统交易策略中的一环，单独讨论无意义。
  • 开仓策略包含：高效率的试错、被止损后再次入场。
    • 开仓包含试错；
    • 止损是一个对自身错误的纠正。
  • 比如，被止损之后，如果行情继续走好，能够再次杀进去试错、而不错过行情。
    • 具体的，突破开仓被止损；下次突破，依然要开仓。
  • 在容易设止损的位置进场，将止损设在行情不容易到达的位置。
  • 每次开仓应该是独立的，不应该受之前交易的影响，不论之前是止损还是止盈。

关于卖出

  • 卖出可分为止损和止盈，其实是一回事，只是时间有所不同。
    • 止损：当买入逻辑已经不成交时，自然应该卖出。
    • 止盈：当持仓理由已经不成立时，自然应该卖出。
  • 所谓理由不成立，对于趋势交易者，就是趋势反转、真实的趋势和自己的预期不一致。

无效止损 & 有效止损

  • 比如，日线上涨，策略做多。
  • 在 15 分钟小周期内，根据形态进场。
  • 15 分钟小周期形态跌破，止损出场。
  • 这时，日线依然是上涨形态。之前的止损，可能就是无效止损。

连续止损

  • 连续止损，必然发生，无论概率多低。
  • 重点是：
    • 连续止损，是否在交易策略之内。
    • 连续止损带来的损失，是否在承受的范围之内。
    • 或者说，你能承受最多多少次的连续止损。

0224 - 微信读书中的「热门图书 + 热门标注」

发表于 2021-02-24 | 分类于 每天写一点

前几天在 Klib 中增加了导出微信读书笔记的功能：链接

做的过程中，就想整这么个东西：把微信读书中热门图书的热门标注，集中整理出来，一次看个够。

热门标注还是挺好玩的；像是众多读者一起，把书中精华的部分拎出来，三分钟看完一本书。在决定是否精读一本书时，除了看评分，还可以看看这些热门标注，看是否对自己的胃口。

比如，我看到 这段话，就笑了：

“轻车熟路，自己慢慢生吧，”上官吕氏把一卷白布、一把剪刀放在炕上，蹙着眉头，不耐烦地说，“你公公和来弟她爹在西厢房里给黑驴接生，它是初生头养，我得去照应着。”

0223 - 什么是吃苦

发表于 2021-02-23 | 分类于 每天写一点

大多数人对吃苦的含义，理解得太肤浅了。

穷，根本不是吃苦。
吃苦，不是受穷的能力。

吃苦的本质，是长时间为了一件事聚焦的能力；以及，为了长时间聚焦的做一件事的过程中，所放弃的娱乐生活、所放弃的无效社交、所放弃的的无意义的消费生活，以及在这个过程中所忍受的孤独和不被理解。

吃苦，本质是一种自控能力、自制能力和坚持能力，以及深度思考。

很大程度上，靠自己成功的富人，往往比穷人能吃苦；否则，他就不能靠自己白手起家。你会发现，他富了以后，做事情还是比你勤奋、还是比你能忍受孤独、还是比你更有理想。

这才是吃苦。
