1104 - 你鸟股市，股市不鸟你

发表于 2020-11-04 | 分类于 每天写一点

今天美国总统大选，几乎所有群、公众号、朋友圈、新闻都在讨论。跟综艺选秀一样，很是热闹。当然，总统宝座花落谁家确实会对很多人的生活产生很大的影响；只是，我们作为国际友人，绝大多数只是像看秀一样在看选举。

果然，不孚众望，总统选举一天反转了好几次，貌似截止到股市收盘前，最后有可能现皇被翻船。与之相对的是，几乎全球股指都大涨，尤其是本土的美国，涨了好几个点。

这就像很多只手在打脸。现皇经常自诩对股市带来巨大帮助，是他 Make 股市 Great Again。结果，现实却是如此不给面子，一有他要回家的消息，股市就涨给你看，啧啧。

股市，作为如此清高自在的存在，不论你鸟不鸟它，它都不鸟你。

1103 - 每天做点喜欢的事

发表于 2020-11-03 | 分类于 每天写一点

我之前说过，赚钱以不受钱所限，释放时间，然后做自己喜欢的事。

反而，这成了不做事的借口。因为目前在忙着赚钱，等赚了钱再做。

何必呢？为什么不能现在就做呢？这实在是经不起发问。

况且，只为钱而累，很容易为钱所累。遇到困难，很容易泄气，人财两空。

为自己喜欢的事做个计划，拆分出一些细碎的、可做的事，然后有把刷抖音、朋友圈、看总统选举的时间，用来做这个事，慢慢朝目标迈进。

1102 - 某类人扔垃圾

发表于 2020-11-02 | 分类于 每天写一点

走在路上，走在前面的建筑工人模样的大叔，顺手把刚吃完的外卖餐盒扔在路边，浑然不觉的继续往前走。

你在心里暗想：这人素质真差。进而，很自然的推想：农民工的素质真差。

再比如，下面两类人，哪种更可能是图书馆工作人员：

  • 女生
  • 文质彬彬、性格恬静的女生

估计大部分会觉得第二类人更合适。

这些都是我们的直觉思考，可，真的是这样吗？

如果你冷静想想逻辑、算算比例，肯定能发现自然不是这样。可，为什么我们还会有那么「自然」的错误推论？

经常看到的解释，远古人类生活在险恶的自然环境中，看到一头老虎，仅凭自己有限的经验，逻辑上自然不能推导出「所有的老虎都会吃人」这样的结论。可，没人敢赌这样的结论，谁知道这头老虎刚刚吃饱，还是外出给崽子找食物。谁也不会傻到把这样老虎想成老虎群中的素食者。这事情理论上存在，现实中却不能相信它的存在。

所以，繁衍至今的族群，不断强化了仅凭个体便放诸群体的思维。只是，发展至今，号称每粒沙子的数据都要统计的大数据时代，每头老虎是不是素食者的数据自然也有。在这样的数据面前，前面的推论就显得很幼稚了。就算不考虑大数据，在个性化日益突出的时代，也不能随意的一叶落而知天下秋。不下断言，让我们的判断成本大大增加，也带来了越来越多的可能性。
