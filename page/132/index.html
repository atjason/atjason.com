0428 - 时间的批发与零售

发表于 2021-04-28 | 分类于 每天写一点

看到一个观点，从时间的角度上看：

  • 上班，是你把时间批发卖给公司；
  • 自由职业，是你在零售自己的时间。

和传统意义上的批发零售类似：

  • 批发的单价低，但胜在量大、稳定。
  • 零售则相反，单价可能更高，但客源不稳定。

我们正在处于时代的变迁中。批发时间，变得越来越困难、越来越不划算。

比如，人到中年，不得到分出很多时间精力到家庭和生活上，女性还要生娃带娃养娃。公司就受不了了，招聘广告明确指出：35 岁以上，不要（虽然，这条规则可能是 35 岁以上的管理层制定的，讽刺不？）当然，想说却不能说的是：快生娃和刚生娃的，不要。

不能批发了，只能想办法零售自己的时间。

相应的职业形式会越来越普遍，比如外包、骑手、快递小哥。

居安思危。即使你目前是在批发销售自己的时间，也想时不时想一想，我能不能零售？

0427 - 自己的娃

发表于 2021-04-27 | 分类于 每天写一点

给自己的娃擦屁股，不觉得臭；换作别人的娃，就受不了了。

目前试过的交易策略，都是在别人的基础上，加上自己的理解，然后进行试错性的交易。一旦遇到困难、瓶颈，很容易产生怀疑。比如，会觉得不适合自己。然后，就坚持不下去、换策略。

每种策略，一定有自己的命门，也可能有不适合自己的地方。如果自己不是理解的很透彻；甚至，如果没有信仰般的信心，在遇到连续的困难时，很难坚持。

现在，我想换个思路。不是在别人的基础上加工；而是自己试着想出一套策略，然后结合别人的策略，对照出自己策略的优缺点。然后，在实际交易中验证。

就算是给自己的娃擦屁股；我想，这总能坚持的久些吧。

0426 - 峰谷电与电动车

发表于 2021-04-26 | 分类于 每天写一点

电力行业有个问题：发电与用电时间的不匹配。

比如，水力发电是 24 小时运转的。而人们用电，通常是在白天。于是，到了晚上，发的电就白白浪费掉；或者是降低电价，让大家多用电；或者是把电存储起来，比如抽水，然后白天再用来发电。

电动车增加了一个选项。试想，如果全国有巨量的电动车，也就有了巨量了锂电池。到了晚上，这些平时不用的电，全部存储到电动车的电池里，岂不是两全齐美的解决办法。

风水太阳等绿色发电 + 特高压输电 + 电动车用电，这三者，可以构成绿色能源大循环。

0424 - 代偿

发表于 2021-04-24 | 分类于 每天写一点

今天又接触到了这个概念：代偿。先看看一个定义：

代偿，生理学上的意义是指人体的一种自我调节机能。当某一器官的功能或结构发生病变时，由原器官的健全部分或其它器官来代替，补偿它的功能。一些例子：

  • 如果一个人的背部脊椎向右倾斜，脖子的脊椎就会向左倾斜，以保持平衡。
  • 如果一个人全盲，他的听力通常会很突出，以使他更好的适应周围的环境。
  • 膝关节中的软骨及半月板磨损时，骨头上会长出骨刺，以增加关节稳定性。

这些代偿，通常是盲目的、生理性。有些时间，短期看有好处，却有长期的、更严重的问题，比如关节中的骨刺。

除了生理性的，人的能力、性格等方面，也会有代偿现象。比如，处于弱势的人，在受到欺负时，可能会有阿 Q 精神来自我安慰，直到心理上的自我保护。

我们要注意的，就是要意识到代偿的存在和其盲目性，做有意识的筛选。比如，用听力来弥补视力，就有好的代偿。
