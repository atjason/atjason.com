0402 - 知识和能力不同

发表于 2022-04-02 | 分类于 每天写一点

关于交易，我看起来积累了很多知识，知道了很多东西。但这些知识是静态、杂乱、不成体系的。

实际交易，每次面临的都是新的、不同的情况。要在短时间内，综合所有已知知识，做出最好的决策和行动，这是能力。

能力和知识不同。知识是静态的，能力是动态的，是模式的，有着情况与应对方法对应关系的。

就像医生，平时积累了很多疾病的症状。这些症状本身是杂乱的、重叠的。当面对一位新患者时，医生需要在短时间内，综合调度所有已知信息，做出最符合病症的判断。了解病症知识只是基础，这种综合判断能力，才是更关键的。

对交易也是如此。复盘、背图这些都是基础，关键在于一次又一次的实盘交易中，不断建立行情与应对策略的对应关系。行情与模式化应对的匹配图谱，才是更关键的。

人们习惯于认为，知道必要的知识，就有了相关的能力。这是错误的。就像是，刚毕业的学生，知道很多知识，但离上手工作，还差很多的练习和积累经验。又或者，知道了很多道理，却依然过不好这一生，也是一样的道理。

0401 - 放空后思考

发表于 2022-04-01 | 分类于 每天写一点

独立内求

  • 独立交易，有自己的判断和节奏。

    • 不受别人影响，也不在心理和行动上依赖别人。

    • 有别人可依靠，前期可少走弯路。可别人替代不了自己。关键的突破，还要靠自己去经历和顿悟。

  • 外求的东西已经够多，接下来开始内求。不会不懂的东西，自己思考。

练习迭代

  • 基础是记录，然后是思考，最后是落实到下一步行动中。

  • 刻意练习。每次交易，都要明确要练习哪些环节。交易结束后，要总结回顾练习结果。

稳步推进

  • 账户资金、交易品种和手数，慢慢增加。

  • 交易系统慢慢改进及复杂化。

    • 先用最少最核心的东西，把系统跑起来。

    • 遇到问题，再思考解决办法。并且，只在非常有必要的前提下，将解决方法并入方案。始终保持方案的精简可执行。

0331 - 把对冲当作止损

发表于 2022-03-31 | 分类于 每天写一点

把对冲当作止损。

只有在原有的仓位重新出现进场点的时候再解锁，也就是真正的止损。或者在原有仓位彻底转向时，两边同时平仓，放弃仓位止损。

0330 - 把剑当斧头用

发表于 2022-03-30 | 分类于 每天写一点

写毛笔字、粉笔字，受物理限制，只能慢慢写、一个个写，相对容易把字写好。

用圆珠笔写，笔尖顺滑字又小，可以写得很快。可是，这样写出来的字更容易乱。

没有能力，或者说能力固化之前，还是慢点好，把剑当斧头用。

0329 - 不要白白犯错

发表于 2022-03-29 | 分类于 每天写一点

当交易犯错，遭受亏损的时候，交易者非常可能逃避现实，拒绝认错和反思，将市场和交易完全抛之脑后，不做任何思考。

其实犯错、失利的时候正是全力思考市场和交易的时候，正是彻底反思过错的时候。

自己真金白银换来的教训，不吸收成经验，真是可惜了。
