0428 - 回报在别处

发表于 2019-04-28 | 分类于 每天写一点

有个观点：做离钱近的事。我大体也是认同这个观点的。只是，凡事有两面。这样的思想，在帮我们抓住一些机会时，也会损失其他机会。

通常，离钱近的机会，就像比较浅的煤矿一样。你看到了，别人也会看到。大家都要抢，于是红海、竞争激烈、利润稀薄、失败风险大。而赢钱远的机会，可能刚好相反。

还有个方面，世界这么复杂且多变，谁又能看清楚未来。现在觉得不赚钱的东西，一段时间之后会怎样，谁知道呢？事情是演化的。马云最开始想切入电商领域的，是电子黄页。虽说这也是粘边，但和后来的淘宝差异还是很大。如果当初没有切入这个领域、又或者是进入的晚了，后面也就难说了。

再有个方面，回报可能会以我们意想不到的方式出现。比如，做一个项目，没成功，却锻炼了技术、建立了信誉、积累用户、甚至结识了大 V。这些收获，都会在将来以某种神奇的方式，极大地帮到你。

当然，并不是说一定要做目前看起来赚不到钱的事。只是，别把这个因素看得太重，别只因这个因素就放弃整个项目。

谋定而后动，那当然好，所有的资源都用在刀刃上。可，如果没那能力，还是 干了再说 吧。

0427 - 学这个有用吗

发表于 2019-04-27 | 分类于 每天写一点

开始学一个东西之前，很自然会问：学这个有什么用？很有可能你无法回答这个问题，或者目前的答案是否定的，你很「明智」地放弃学习，转头刷抖音。

然后，接下来的某一天，你遇到一个问题，开始后悔：要是当初学了那个东西该多好呀。又或者说，因为你根本不知道有这个东西，可以专杀眼下的问题，连后悔的机会都没有。

很多时候，学就是了，别问有什么用。
有用，是学习的结果，而不是原因。

本文启发自：李笑来《学习是门手艺》

0426 - 高价格的原因之一：减少技术支持成本

发表于 2019-04-26 | 分类于 每天写一点

最近的 SEO Grows 项目中使用到了代理 Scraper API（不用代理，会被 Google 反爬虫屏蔽，你懂的）；今天在跟他们聊时，问起是否可能提供价格更低的套餐（目前起步价是 $29 每月），他们的回答是：

We can’t afford to do a lot of support for plans that cheap.

确实，技术支持是相当大的成本。你感觉对方可能只是回答了一个邮件而已，可这背后可能是一个技术支持岗位、对现在工作的中断等等。不是说用户反馈不重要，而是很多用户问的其实是非常基础的、重复性的问题，甚至是文档里已经有答案，他们只是不愿意花时间看，只希望通过客服来替他们解决问题。且，很多问题并不是产品本身的问题，而是用户缺乏必要的相关、或基础知识。直白的说，是用户自己的问题。

尤其，说的难听点、可能不正确的话，越有钱的人素质越高（反过来我就不说了）。越是花了很多钱的人，可能越不觉得自己是大爷，沟通起来也更客气、更理性。那些免费用产品、还觉得是给你面子的大爷，真是伺候不起，那就通过价格歧视，躲着点。

分享一个监控 Google 关键词搜索排名的必备工具：seogrows.com

发表于 2019-04-26 | 分类于 IndieDev

Seo Grows 核心功能

一句话：监控 Google 关键词搜索排名，记录排名历史。

有什么用呢？

主要是为网站 SEO 服务的，是关键词优化的量化助手。

事实上，SEOer 的日常工作，就包含在 Goolge 等搜索引擎查询不同关键词、并记录排名，比如记录在 Excel 中。如果关键词很多，这是很花时间的重复性劳动。seogrows.com 可以完全自动化这项工作，几乎是 SEO 必备工具。

进而，在优化关键词期间，seogrows.com 所记录的精准排名历史，可以作为量化参考。当关键词异常变动时，及时邮件提醒（待实现）

另外，对外贸独立站等类型的商业，网站是主要客源，关键词的排名更加重要且敏感，几乎直接等同于成交量，需要实时监控。

基本用法

基本上，只要一次性地添加域名和关键词，就可以随意摆弄搜索结果了。

添加域名和关键词

简单介绍下各项：

  • Title: 相当于域名的别名，界面显示更友好。
  • Domain: 就是域名，注意无需 http，当然程序会自动过滤。
  • Path: 比如，你想监控子目录 http://example.com/blog，就在这里输入 /blog
  • Search Engine: 目前实现了 Google(谷歌)
  • Country: 不同国家的搜索结果差异很大，选择你的目标市场，重点优化。
  • Keyword: 以 ，区分

注意：刚添加的关键词需要一点时间才能查询完毕，这和服务器繁忙程度有关；通常需要几分钟，请耐心等待。之后，服务器便会每天自动查询收录和排名情况。

摆弄搜索结果

先来看看 过滤器：

  • Device: 桌面端和移动端的搜索结果差异很大，还是要注意你的目标用户的习惯。
  • Date Range: 查询的日期范围。注：添加关键词之前的数据，自然是没有的。
  • Country: 在这里可以切换创建域名时设置的国家列表。
  • Position: 查看排名落在某区间的关键词。
  • Keyword: 筛选关键词。

另外，所有的列都可以点击排序。比如，你可以将所有关键词按排名排序。

再来看看 排名历史：

  • 点击 Position 列左侧的图标，则可显示该关键词的排名历史。
  • y 坐标是小的、好的排名在上方，和搜索结果视觉一致。

还有 SERP(Search Engine Results Page):

  • 点击 URL 列左侧的 SERP 按钮，则可显示该关键词的 SERP
  • 比如，你的关键词排名第 2，你自然想知道谁他丫的敢排第 1；
  • SERP 会完整显示 Google 的前 100 条结果。其实 100+ 之后的意义不大。

分享域名及关键词

如果你想将查询结果分享给同样做 SEO 的同事或老板，可以点击域名右侧的分享图标，然后点击 Start Share，将链接发给对方即可；不需要登录便可打开分享链接。

之后

其实还是很多想做的功能，比如：

  • 设置关键词排名大幅变化时，邮件提醒
  • 和竞争对手的关键词排名放在一起比较
  • 关键词搜索量趋势
  • 导出查询结果
  • …

之后慢慢加上。

有没有百度版的？
有，这是 百度版关键词搜索排名监控，由于微信登录、排名历史等功能还没做，就没单独拿出来介绍；未来也会考虑做百度竞价排名监控。感兴趣的朋友可以打开试试。

一些细节：

  • Google 登录需要特殊上网
  • 微信内嵌浏览器不支持 Google 登录，可使用手机系统浏览器
  • 不登录，也可以查看首页的 Demo
  • 手机端优化不充分。建议使用方式：电脑端创建、手机端查看
