0313 - 抓住机会

发表于 2020-03-13 | 分类于 每天写一点

今天早上比特币大跌，本来可以盯盘并抓住机会，结果却去了超市买东西，其实并不是特别必要的事。结果，不仅没有做空或抄底，还因为机器人不适应这种情况，略亏了些。

危机危机，危中有机。感觉自己对机会的敏感度差了些，不太适合做需要人进行实时判断的操作。程序化对这个缺点是个弥补，可以无感情地快速执行预定的策略。

0311 - 有钱与审美

发表于 2020-03-11 | 分类于 每天写一点

在老家，人们有钱之后很重要的事是：建房子。

也就是在自家宅基地建房子。没有的建，有的拆了重建。总之，是要建大的、好看的房子。后者尤其重要，要的不就是面子嘛。

可，这事还是有很多里子在的。房子建成什么样子，里面如何装修。大到布局，小到配色，既关系到口袋里有多少钱，更由审美决定。

钱这东西，可能在短时间内暴富。而审美这东西，却快不了，需要在平时有意识的培养。

0310 - 自建工具

发表于 2020-03-10 | 分类于 每天写一点

这几天，在给自己搭建交易工具，包括 Order Book、订单流、K 线图、下单界面等。

因为全部是自己实现（当然是借助第三方组件），可以很灵活地配置成自己想要的样子。尤其，可以做到极其精简，只显示必要的信息；且可以用自己能看懂的方式继续简化，而不用像大众产品一样，在效率和易用性间平衡。

当然，即使这些都做好了，对赚钱与否，也不起主要作用。不过，帮助还是有的。尤其，我个人很喜欢这种一边砍柴、一边磨刀的感觉。

对了，又买了一台服务器…现在已经有 5 台了…

0309 - 不劳而获

发表于 2020-03-09 | 分类于 每天写一点

在老家，有种操作叫：拾东西。

比如，有人家种了一大片胡萝卜。等他们收获完时，地里其实还会有些遗漏的。这时，主人家其实已经放弃一遍遍去地里找这些遗漏的胡萝卜，因为投入产出比太低，不划算。

可有人多的是时间，比如在家闲着的老头老太。他们会带上小工具和袋子，去地里进行地摊式的搜索，找到这些遗漏的胡萝卜。不花一分钱，捡到就是赚到。更神奇的是，不管前面的人找的有多仔细，后面的人总还是能找到遗漏。

这种接近不劳而获的感觉，很爽。甚至可以说，会让人上瘾。有时明明不差这点钱去买，或者已经拾到足够自己吃的了，还是继续拾，多出来的分给亲戚邻居。

我也觉得挺好玩，今天还带着娃去地里，体验了原版的拔萝卜。可笑的是，他力气小，会你排雷一样，把胡萝卜周围的土都用手划开，然后再拔。其实，在我看来，那胡萝卜小的，我两根手指头就可以直接拔出来…
