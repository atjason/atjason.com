0415 - 别人能放心把事情交给你做

发表于 2019-04-15 | 分类于 每天写一点

有没有这种感觉，有时你让别人做件事，其实你心里就感觉他可能会出问题，需要自己去处理异常、盯结果。除非必要，你实在不想托付此人做某事。

明显，这种感觉很差；
明显，你并不希望别人会对你有这种感觉。

一定要别人对你有这种印象：交给你做的事，你一定会做好；如果你做不好，会提前、至少及时通知对方。不管大事小事，都要如此。

这其实并不完全是做事能力的问题。一件小事没做好，也能极大地伤害别人对你的印象。

title: ‘0414 - 数据驱动型产品’
categories: 每天写一点

date: 2019-04-14 22:00:00

对一款软件，如果从用户操作角度去看，可能会感觉状态很多、线索很多、七零八碎。换个角度，如果以数据为中心去看待外围的部分，事情可能就会明朗很多。

比如，对于 Vue，可以把 store 作为数据中心、发动机，以此来关联和驱动状态变更，如连接网络请求、本地缓存、界面呈现、响应用户操作等。思想上有了中心，就不容易乱，更可能一步步去实现和丰富整体。

还有一个好处，就是前后端对接的时，大家说的是相同的话：都以数据为中心，进行交流就会顺畅很多。

当然，这个思路并不是万能的；目前感觉，对中小型的工具型产品，比较适合。

且，这个方法自然也有问题，就是不可避免会把一些业务从 View/Control 层下沉到 Data 层，或者是纠缠不清，或者是把这个数据中心搞得很复杂，进而带来维护、测试复杂等问题。

发表于 2019-04-13

title: ‘0413 - 996 是必经阶段’
categories: 每天写一点

date: 2019-04-13 22:00:00

昨天，马云也发表了关于 996 的观点；本来被视觉中国抢走的话题，好像又要复燃起来。恩，我也来插一句：

996 是必经阶段；是一个东西从小到大、由弱到强过程中，无法规避的环节。

太阳底下无新事，就像资本主义当年的原始积累、黑奴及血泪劳工史。和其相比，现在的 996 已经文明太多了。

当然，你可能会说，个体是无力的、没什么选择的，毕竟换工作的成本不是每个人都能承受、且大部分公司都要加班。既如此，我们更应该团结起来，去对抗这种大环境、抵制这种不良现象。但，既然你提出大环境，那就得从大环境上思考，这个问题是否真的可以彻底解决。

从个人上升到稍微大点的群体：国家。国家和国家之间，也是弱肉强食的逻辑。只是，在现代环境下，这种逻辑变得隐蔽了。典型的表现，就是战争少了，更多的是使用金融工具。

比如当年的亚洲金融危机，先是民间资本索罗斯洗劫一空，又是国际货币基金组织这种官方机构趁火打劫，给急需帮助的国家及政府，发放条件极其苛刻的贷款。接受贷款的国家，基本是饮鸩止渴（比如指定接收的贷款必须用来购买西方的东西，相当于我借给你钱买我的东西，你还要还我高额利息）

又比如现在的拉美国家，甚至出错了经济倒退。美帝等国家减免其债务，并不是出于善良，而是现在把这些国家搞崩了，下次去剥削谁呢？且，说是债务，更多的是债务利息，本金可能早就还得差不多了。

说了这么多，只是想提醒，在国家层面，我们还是弱国，处于明着暗着被剥削的地位。强者已经卡位，弱者需要比强者当年更多的努力，才能和现在的强者站在同一起跑线上；而可惜的是，那时候强者已经跑出去很远了。这种情况下，又哪来的资本谈舒适。

说得有点沉重，说的好的。说 996 是必经阶段，并不是说永远都会如此，毕竟事情是在变化中的，比如，说不定现在就是大的转变开始的时候，而 996.ICU 正好当了旗手。虽然我觉得为时尚早，谁知道呢？

还可以引出个话题：效率的提升，会不会让人们变得轻松？

还是从历史找答案。工业革命之前，劳动人民想必是累的。有了机器，能代替人干活，这下工人应该轻松了吧？我想答案你也知道，机器、尤其是流水线的出现后，工人连上厕所的时间都被严格限制，自然没有变得更轻松。

再看看现代。比如，没有电话、没有微信的年代，至少下班后员工可以好好休息。现在呢？微信这样的工具把效率提升了很多，按道理应该缩短工作时长了吧？可事实上，活更多了，老板随时能抓住，隐性加班更多了。

我是感觉，人这个物种，估计只会越来越累，不会有退休的一天，除非是被别的物种取代了。就像是年纪大了退休，还不是要给孩子们带孩子…

0412 - 好产品，就在那里

发表于 2019-04-12 | 分类于 每天写一点

好的产品，就像本就在那里，只是被发现、而不是被创造，且无法被更好的人造物取代。

就像好的雕塑，不是被刻出来，而是本来就藏在石料里，只是被褪去石头外衣、露出本来面目而已。而，你又无法用石头粉捏出个雕塑，不是那个感觉。

我们不是在创造产品，而是沿着用户需求和体验这条纹理，慢慢褪去石头外衣，找到本就藏在里面的产品。

至少，要有对好产品保持敬畏的心态，而不是以造物主自居。

等你来发现。

0411 - 能力与意识

发表于 2019-04-11 | 分类于 每天写一点

有没有这种感觉：
有的人你想让他做点事，他很快就能理解你到底想要什么、做成什么样子，可做事情的速度可能不如意；
有的人刚好反过来，做事情挺快，但总感觉做的不是你想要的样子。

要是这两种东西同时放在自己身上，就有点难受了：
有时，明明已经想清楚了事情，可就是做的很慢，迟迟拿不到结果；
有时，状态很好、做事很快，但不知道最应该做什么、做到什么程度为止。

以自己目前独立开发、尚没有稳定团队的情况，还是得能力与意识兼备，不能有明显的短板。不过，感觉自己做事的速度太慢，远远跟不上想的速度。得考虑搭班子、建团队了。
