0927 - 谁会迷信

发表于 2021-09-27 | 分类于 每天写一点

有两类人会迷信。

一类，是自己能力实在很弱，想改变又无能为力，所以求助于神明。比如愚昧。

另一类，是在做很多不确定性的事。虽然自己的能力圈很大，但圈越大，接触到的未知也越多。这种情况下，不确定性增加，运气成份就显得比较大。于是，遇事烧个香，至少图个心安。比如老板。

相反，整天呆在自己能力圈内，做着 1+1=2 这种确定性的事，实在没理由去麻烦神明，自然是无神论者。比如员工。

0926 - 供养

发表于 2021-09-26 | 分类于 每天写一点

供养这种模式，常见又不常见。

比如，活佛僧侣，就是被信徒供养着。施主给吃的，佛祖给心安；各取所需。

比如，一群读者养活一个公众号，一些用户减少一位独立开发者，也是供养。

比如，家边上新开了个咖啡店，环境不错，很适合办公。虽然，我不喜欢喝咖啡、他们家做的也不好喝，但我应该会经常去光顾、去消费。因为，对我来说，这是个不错的办公场所，我不希望它倒闭，所以想供养它，出一份力。

供养，是在商业逻辑之下，附带感情色彩的一种模式。

0925 - 现金不是资产

发表于 2021-09-25 | 分类于 每天写一点

什么是资产？能生钱的是资产。现金放着会随着通货膨胀贬值，故而不是资产。

国人普遍缺财商这东西，比情商智商都缺。

比如，有哪些可选的理财方式、各有什么特点、对应的投资策略是什么、应该如何组合搭配、家庭资产应该如何配置等等。这些基础的问题，能回答上来的，可能真不多。

一方面，需要加强财商的基础教育。
另一方面，需要更多的理财规划师；就算没有保险规划师多，也得有其几分之一吧。

0924 - 回杭州

发表于 2021-09-24 | 分类于 每天写一点

暑假带娃回老家，不想遇到疫情，一耽误拖到了现在才回杭州。

娃能有个农村生活，挺好的。可能是天性，我的娃能在土里玩一天；挖坑、堆土堆、打洞。我们去黄河边玩，吊床铺垫子什么的，他都没兴趣，一个人玩沙子从头玩到尾。

虽然少上了几个星期的学，但娃他妈在家教的也挺好的。写数字、画画、做手工、投篮，进步都很明显。

我能做的，就是多赚钱。负担起家庭开销，给家人提供更多这样的自由，选择的自由、时间的自由。

0923 - 微信读书的读就送机制

发表于 2021-09-23 | 分类于 每天写一点

之前，我是用的邮箱注册的微信读书，并购买了无限卡。可是，在墨水屏版上，微信读书只支持使用微信账户登录，不支持邮箱账户。于是，我的无限卡白瞎了。

切换到微信账户，发现微信读书有类似读就送的激励机制。而且是阶梯式的。基本上，只要每天读、每周共 10 小时，一周 7 天就能换 11 天的无限卡。只要坚持读，无限卡只会越来越多。

这个挺好嘛。像游戏打怪掉装备，能激励人刷读书时间。本周我就拿到了 11 天，还挺有成就感的。恩，今年不准备续费微信读书无限卡了。

不过，腾讯也不是白给的。免费送的无限卡，一些书是没法看的。只能付费、或者用书币换。算了，不能看的就先不看，或者用读书时长换书币再看。和激励的效果相比，这点代价是值得的。再说，还能在我的邮箱账户看嘛。
