0823 - 平衡

发表于 2020-08-23 | 分类于 每天写一点

接着昨天的说。

做开发的人，应该都有这种感觉：学新技术，学的时间长了，就没兴趣了、不想学了，至少没有刚接触时的新鲜劲。就像是一个杯子，装满水后，就没办法再装更多的水。

这时，如果用新技术做了个新产品，那就不一样了。随着产品研发的深入，对新技术的掌握也提出了更多的要求。发现了更多自己不会的点、更多需要解决的问题，就又重新有动力去学习了。

信息的接收和输出也是一样。每天刷手机，看各种新闻。看到最后，就想吐了，再好看的也看不进去，就算看抖音这种快节奏的东西也要快进。就是接收的东西太多了，已经装不下了。

这时，如果进行一些输出，就不一样了。比如，刷半天的新闻，就所看到的东西写篇文章，这就是种输出。写文章时，就需要反刍所接收到的信息，可以还需要多方查证，或者补充。输出之后，再去接收新的信息，就容易多了。

多输出，既是帮别人，也是帮自己。

0822 - 内求、外求

发表于 2020-08-22 | 分类于 每天写一点

交易，看起来是想将市场上的钱放自己口袋里，是外求。其实更多的是和自己内心争斗，是内求。交易的方法很容易学；甚至不太聪明的人，反而学得更好。可真到了交易实战中，理性这个看门人，往往压不住非理性在内院放火。搞定了内在，才有可能偶尔赚顺应市场的钱。

而如果一直是内求，则很可能变得狭隘、短视、自负、易怒、简单粗暴、以自我为中心、等等。总之就像内分泌一样，一旦失调，各种毛病。

作为平衡，就要对外输出。说是输出，其实还是为了换回自己内在无法生产的东西，就像国际贸易一样。比如，写东西给别人看、做出产品售卖，看似是为了赚钱或别的，但更重要的是收获其他的东西。比如，解决用户问题时带给自己的满足感；用户向自己表示感谢时，自己内心的喜悦；看到自己的产品频频出现在媒体上，自己内心的虚荣得到满足；等等。这些都是钱买不到，冥想也想不出来的。

内外兼修，自在平衡，方得始终。

0821 - 易怒、刻薄，涵养、修养

0820 - 诗、意境、传承

发表于 2020-08-20 | 分类于 每天写一点

为什么诗能流传千年？

比如，你挥汗如雨，终于爬上山峰，看着眼前开阔的场景，是不是很想说点什么？说啥呢？很自然，类似「会当凌绝顶，一览众山小」这样的诗句就蹦出来了。

比如，你在草原生活，眼看着远处的羊群若隐若现，会不会想起「风吹草低现牛头」？经过四季变迁，眼看着茂盛的草地变成枯黄，又变得绿油油，想不想说「春风吹又生」？

这样的场景，千年不变；这样的意境，千年来都需要被表达。拿什么来表达？诗起到了这个作用，也就跟着场景与意境，流传千年。

与之类似的，还有哲学。人类有困惑，所以有哲学。人类永远有困惑，所以永远有哲学。

0819 - 离婚率

发表于 2020-08-19 | 分类于 每天写一点

今天同时知道几个关系不错的人都离婚了，还是挺震惊的。

当然，并不是今天离婚，而是已经半年或一年多了。只是，毕竟这也不是什么好事，大家都不愿说出来，因此周围知道的并不多。

要说为什么，我想无非还是那些老套的剧情：

  • 比如，长期两地分居、聚少离多，慢慢感情也就淡了，经不起生活中不如意的冲击。
  • 比如，不太门当户对，尤其女方家庭经济条件好、社会地位高，甚至女方比男方收入还高时，就很容易产生矛盾。如果再扯上婆媳关系，那就更麻烦了。从男方的心理，你看不起我、或者日常一些不太尊重的言行，我也就忍了。可如果是对我爸妈也这样，那就不能忍了。
  • 比如，期望与现实的差距。比如女方对男方期望高，期望能承担养家糊口的重任，同时能关心自己，一起过好小日子。这样的期望或要求自然很正常，可如果遇到的男方是小富即安，或者有些懒散，甚至是没能力、没责任心、脾气还不好，那这差距可就大了。再看看自己闺蜜的老公，那简直要气死，不过了。

我是觉得，离婚真是件很麻烦的事。如果要重新建立家庭，之前的麻烦都要再来一遍，关键还少了新鲜感的调和。如果还要背上上一段婚姻遗留的问题，比如小孩，那就更痛苦了。想到这一点，我绝对佩服再婚且能过得很好的人，处理关系能力一流，情商绝对高。
