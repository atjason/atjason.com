0831 - 余量

发表于 2020-08-31 | 分类于 每天写一点

医院、学校等公共场所的洗手台盘上，一般都有类似七步洗手法的宣传图。真有人洗手洗七步吗？我觉得很少。那这个宣传图是笑话吗？可能还是有用的。比如，洗手的人看到这个宣传图，有可能会想：那我多洗会吧。洗不了七步可以洗三步，至少保证了去洗手。

从这个角度上讲，宣传的内容有余量、有冗余。这样，受众即使在执行时打了折扣，也还是有效果的。

再比如，疫情期间有很多自己量体温，然后上报的机制。无论这个机制有多少「本人保证如实上报」的签定画押环节，我绝不相信每次都是真实量体温然后上报，绝大多数都是 36.x 随便填填。而负责收报表、做统计的人，也绝对知道这里面有很大比例的弄虚作假，但也假装没看见。

那这个机制是笑话吗？可能还是有用的。比如，一个发烧的人，把体温填写成 36.x 以上报时，心里多少还是发虚的。再加上对未来可能惩罚的担心，还是可能如实上报的。这样一来，只要真正有问题的人如实上报了、目的达到了，那些本身就正常的人，写 36.3 还是 36.5，有什么区别吗？这就是余量。

确实，余量一定包含浪费，全民填写 36.x 也一定浪费了很多资源。但没办法，与能达到的效果、避免的损失相比，这点浪费完全可以接受，也必须承担。

0830 - 要不要去医院

发表于 2020-08-30 | 分类于 每天写一点

每次小孩子生病都很纠结：要不要去医院。尤其是对于感冒发烧这种比较常见的病。

不去吧，各种担心：

  • 万一不是普通感冒呢？
  • 万一引发肺炎、病情加重呢？
  • 万一贻误最佳治疗时机呢？

去吧，也是各种担心：

  • 万一在医院交叉感染怎么办？
  • 万一抗生素用的多，产生抗药性怎么办？
  • 医生几分钟就下诊断了，万一误诊了呢？
  • 孩子生病需要多休息，去医院太累太折腾了。

我觉得现在的医疗粒度还是太大了，生病基本只能去医院，没有粒度更小的办法，比如家庭医生。其实杭州这边理论上也有签约的家庭医生，不过说实话我从来没联系过，认都不认识，也不清楚对方能提供什么服务。

现在「轻奢」的概论很流行；对应过来，我觉得很需要「轻度医疗」，小病、医疗咨询等领域的轻量级解决方案。

0829 - 和善、坚定

发表于 2020-08-29 | 分类于 每天写一点

看教育孩子的书《正面管教》，其中有个核心观点：对待孩子要和善、坚定。

从反面来介绍吧。

和善的反面是严厉。具体的，就是频繁使用惩罚。尤其当孩子「不听话」时，就用呵斥、打骂、冷落等方式，强迫孩子「听话」。负作用是，孩子只是当下听话，其后反而会带来更强烈的反抗。直到打不过、不敢打时，这一招自动失效。

坚定的反而是娇纵。表面上，想给孩子一个美好自由的童年。孩子想要什么，就给什么；不想干什么，就不干什么。负作用是，家长会非常累，被宠溺的孩子，自然也会有一堆问题。

和善、坚定，就是既要试着建立孩子的自尊心、自信心、责任心、决断力等，也要在孩子做错事、无理取闹时，坚定自己的立场。看起来是很矛盾的，实践起来也确实很不容易，尤其是刚开始时。我也在学习，先了解这个正确的观点吧，慢慢照着练习。

0828 - 以简单应对复杂

发表于 2020-08-28 | 分类于 每天写一点

交易市场价格复杂多变，怎么应对呢？

其实也简单。

首先，不要想着能应对不同市场的不同情况，那太难了，超出了个人的能力。

然后，就是反其道而行，修练自己的一招鲜。只瞄准市场的一种情况，然后用一种方法去应对。

既然是一招鲜，还需要修练吗？是的，非常需要。

首先，市场复杂多变，出现自己能应对的那一种情况的概率可能并不高。这就意味着，在绝大多数时间，你都需要耐心等待。就像狮子藏在草丛中，等着猎物慢慢出现在视野中。练出这种耐心，极难。就像狙击手，重要的不只是枪法准，还要能在一个地方趴上一天一夜，只等一个开枪机会。

上面说的是等待、是瞄准，接下来就是机会出现的那一瞬间，能不能捕捉到、敢不敢开枪。机会不会是完美的，总有些风险。这就需要快速判断输赢的概率；一旦概率合适，勇敢的开枪。命中则已；不命中，要立马舍弃一切，换个地方继续潜伏等待。

还有，就是要平衡自己的多重身份。自己不只是交易者、狙击者，还是现实生活中的各种角色。在交易中的喜怒哀乐，不应过度影响生活，尤其是坏的影响。

0827 - 迭代速度

发表于 2020-08-27 | 分类于 每天写一点

买房投资，周期长。其经验教训的增长，是按年计。

交易很快，按天计，甚至按分、按秒计。

迭代速度快，可快速积累经验教训，快速进化。昨天交易方法不对，明天就改；明天改了见效，后天就坚持。

其实很像传统行业和软件业。比如房子盖得不好，拆了重建很麻烦。软件要是做的不好，改进了再发个新版即可。

毕竟我是软件行业出身的，还是喜欢节奏比较快的。买个股票放着一年不动这种价值投资，我是做不来。
