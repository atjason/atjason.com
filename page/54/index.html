1108 - 开放才有大格局

发表于 2021-11-08 | 分类于 每天写一点

现在满大街跑的是新能源车，都是锂电动的。

那新能源就只有锂这条路吗？明显不是。还有一条明显的技术路线：氢能源。

事实上，日本在这方面非常领先，世界上已经有很多氢能源车在路上跑了。

为什么目前看起来，氢能源输了呢？其中有一个很重要的原因：日本太领先了。关键是，只是自己跑得快、跑得远，手里握着绝大多数的专利。这种情况下，谁来陪你玩？陪你玩，结果钱全是你赚走了，那我图什么。

于是，大家被迫在其他赛道找机会，于是就找到了锂。正好在方面，专利垄断没那么严重，理论上大家都有机会。更多的人才、资源参与进来，使得锂这条赛道，迭代得非常快，也导致了目前的局面。

开放，才能把蛋糕做大，自己也才能真正的获取更大的利益。利他，才是真利己。

1107 - 行业知识树

发表于 2021-11-07 | 分类于 每天写一点

最近看了很多行业研报，对风光锂等行业有了树级别的认识。

之前看交易相关的公众号，除去惊悚的标题和灌水的内容，其中有价值的点并不多。并且，这些点比较零散，无助于形成系统的认识。

研报则不同。一些深度研报，其功能之一，就是分析这个行业完整的产业链、分别有哪些环节、又主要有哪些企业参与。有了这些信息，就有了一颗知识树。

有了这个基础，再去个公众号，就很容易对号入座。一方面，会发现这些公众号的内容比较浅，通常也没介绍多少新的东西。另一方面，还是有点用的，主要是介绍行业动向；像是鼻子，替自己捕捉变化。

尝到了这种方法的甜头，下周准备再研究几个行业。

1106 - 太平洋屋脊步道

发表于 2021-11-06 | 分类于 每天写一点

看了《走出荒野》，对太平洋屋脊步道萌生了兴趣。

太平洋屋脊步道（英语：Pacific Crest Trail，简称：PCT，又名：太平洋屋脊径），是一条贯穿美国西部荒野、大致沿着内华达山脉及喀斯喀特山脉行走、全长4,286千米（2,663英里）的远足径。

从最低处近海平面，到最高处的13153英尺（4009米），经过25个国家森林7个国家公园。每年从4月开始，有大约300人从墨西哥边境出发，企图走完全程，其中大约有180人成功地在9月底或10月初到达加拿大边境。

作者是在 1995 年完成的徒步穿越，而现在通信、定位、装备都大大升级，穿越难度肯定相对小很多。

1105 - 奇怪的知识

发表于 2021-11-05 | 分类于 每天写一点

宇航员失重，是因为地球对他们没有吸引力了，对吧？

否。

他们所受的重力，是地表的90%，比在月球表面高的多。离心力和向心力打平，所以感觉失重。

拉格朗日点，是两个天体的引力平衡点。对于地日，大概在 150 万公里。也可以理解为，位于这个点的东西，会和地球一起，围绕太阳旋转。

地球同步轨道，是绕地球旋转的角速度，和地球自转速度相同，大概在 4 万公里。也可以理解为，位于这个点的卫星，始终在你正头顶不动。

地球每天都在吸收太阳的能量，对吧？

否。

要是这样，地球不就是一直被加热的锅，我们不就成青蛙了。

地球不是貔貅，吸收的能量又辐射到宇宙了。

太阳和月亮哪个重要？

当然是月亮啊。

太阳只在白天出来，而白天已经很亮了啊。

为什么先看到闪电、后听见打雷，因为光速快吗？否，因为眼睛在耳朵前面。

哈哈哈。

1104 - 反感标题党

发表于 2021-11-04 | 分类于 每天写一点

现在的公众号，已经不会正常的起标题了：没几个能从标题里知道，这篇文章主要是讲什么的。

我理解这样做是为什么，无非是增加点击率。本来可以在标题中直接说明，非不说，让你点进去看。对于经常看的老用户，感觉每次都被调戏一次。

不止个体户，连官媒也是这样。人民R报有一堆类似这种「伉俪情深！这一幕好暖」，直接说人名能 Si 啊；「厉害！消防员真是啥都能救！」，说救了啥能 Si 啊，不用！！！能 Si 啊。
