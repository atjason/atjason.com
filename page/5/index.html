0608 - 干什么、为什么

发表于 2020-06-08 | 分类于 每天写一点

小孩总是问类似这样的问题：爸爸，这里有个坑干什么？

妈妈就说：一般小孩子都是问为什么，你怎么总是问干什么？

我笑了，说你分别回答「干什么」和「为什么」这两个问题试试？

结果，好像是差不多的；或者说，相同的答案，都能无违和地对上这两个问题。

说的哲学一点：存在，就是它的理由。

0607 - 交易者是孤独的

发表于 2020-06-07 | 分类于 每天写一点

交易者是孤独的。

于是，去和人交流沟通。

于是，变得更孤独。

交易，像文学、哲学，而不是数学、物理学，并没有定法，也没有统一的标准。交流，多半是不再思想的碰撞。而因为没有事实上的标准，这样的碰撞很多时候是鸡鸭聊天。确实，存在互通有无的可能。只是，这需要交易本身有一定的功力，有经验教训，才有可能博采众长。不然，只会越来越乱。

在交易中得不到共鸣，只能变得更孤独。要么在孤独中坚持，要么在孤独中放弃。

加入过一些交易相关的群，先前我还时不时说几句。后来，我不愿说了，因为说了也没什么意思。别人不大可能从我说的内容中有所收获，我最多也只是宣泄下情感而已。如果是想从群友身上得到慰藉，不如多陪陪家人。

0606 - 世俗的成功

发表于 2020-06-06 | 分类于 每天写一点

我们太需要世俗的成功，让自己不狼狈，证明自己的正确，乃至赋予意义。

比如，愤世嫉俗，不想上班，不想过规划好的生活，不想做到了某个年纪该做的事，等等。

想在俗的社会里，做些不俗的事，难。世俗就像网，各个角度束缚你、拉扯你。更重要的，让你开始怀疑，自己是否真的错的，不该有那样离经叛道的想法。

标新立异，可以；但前提是已经有了世俗的成功。

比如，马云可以在年会上朋克放歌，够出格吧？可你觉得奇怪吗、难以接受吗？好像又不是。可，要是换做阿里的底层员工上台这么干，你又会做何感想？

0605 - 沟通与默契

发表于 2020-06-05 | 分类于 每天写一点

尤其和至亲至熟的人，很容易觉得对方会知道自己怎么想、想要什么，于是少了某些必要的沟通，因为觉得没必要。结果因为理解不一致，有了不好的结果。

以为有默契，结果很尴尬。

慢慢的，好像对默契失去了信心，觉得越来越没有默契了。

缺少沟通，像是赌。赌对了，那种貌似有默契的感觉，很美妙；可惜赌错了，很尬。算算概率吧。

给沟通加个保险，多一点核实确认，没坏处。

0604 - 不折腾，无发展

发表于 2020-06-04 | 分类于 每天写一点

回杭，知道了老小区改造，什么外立面、道路、水电光纤之类的。这就是折腾。

不折腾，不会错，也不会有发展。

折腾了，直接的可以带动人员就业、建材销售，间接的可以撬动一批，比如人员就业后的消费，生产建材的企业，等等。即使是左手倒右手，GDP 数字也会往上涨。

当然，当地老百姓的生活水平会有所提高。只是，这其中有多少是人们主观想要的、多少是被强加的、所做的是不是最迫切需要的，就很难说清楚了。
