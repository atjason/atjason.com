0412 - 理财与赌大小

发表于 2020-04-12 | 分类于 每天写一点

这两天接触到了币圈的一个 理财 产品。比如，投入资产 A，当币价高于某个值时，你会得到带有一定收益的 A 资产；反之，你会得到一定的币。

这不就是赌大小嘛。

如果我知道币价会涨，直接做多不就好了；反之就是做空，犯得着这么麻烦嘛。

有趣的点在于，这么一个简单的核心，被包装上理财的外衣，再加上华丽的年化收益数字，甚是诱人。是什么样的人编织出这一切，又是什么样的人发现并陷入其中，这些人最后的收益或亏损是什么，等等。这些事情，想想还是有意思的。

0409 - 手段光明

发表于 2020-04-09 | 分类于 每天写一点

哪怕是为了达到好的目的，也不应该不择手段。结果的正义，并不能粉饰过程的邪恶。

和差的人打交道，一定要小心，尤其要避免沾染上对方的习气，陷入对方的思维逻辑中，落入比下限的境地。一旦如此，你会慢慢变得和他一样 Low，变成你现在讨厌的样子。

有时，对付什么样的人，就要用什么样的办法，是对的；但又是很危险的。对付恶人用恶的方法，不是不可以，但一定要小心，因为恶的方法通常是捷径、见效快；一旦用了，很容易上瘾。

比如，有人打了你一拳，你用更狠的一拳还击，这自然是见效最快的，他不敢再欺负你了。这里的陷阱在于，下次呢？一旦你出过拳，之后就有可能主动出拳，出错误的拳。

写到这里，又突然想到交易中的一致性、纪律。偶尔破戒，确实无伤大雅。问题在于，它会破功，大大削弱下次守戒的定力。当个笨人，比偶尔耍小聪明，可能更好。

0408 - 不立不破

发表于 2020-04-08 | 分类于 每天写一点

不破不立；旧的不去、新的不来；基本我们习惯这么想。

而事实上，现在「旧的」，也是之前「新的」。你新交的男/女朋友，也是之前别人旧的。

现在的东西，你看不上，想要改变，可以；但改变之前，请拿出新的更好的方案。如果拿不出，那你只是想抱怨。可要知道，存在即合理；现存的东西，可能是稍早之前最好的解决方案。

立，也分两种；改良、改革。改良，在已有的基础上改进，甚至可能只是优化排列组合、辗转腾挪。也是有效果的；且更容易看到效果、风险更低。

改革，则是彻底抛弃旧秩序以重建。怎么说呢，想起了软件开发中的重构。重构的难度大，成功率很低。通常，如果只实现核心功能，是容易的，很容易高效实现、并保持简洁。但要考虑到各种历史包袱、意外情况，方案会不可避免地变得臃肿，也就会引出各种问题。重构后简洁的东西，加上越来越多的补丁，越来越像重构前的东西。有点像，我们总会变成自己讨厌的样子。

总的说来，我更像是保守主义改良派。
