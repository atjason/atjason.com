0507 - 地球的能量来自何处

发表于 2020-05-07 | 分类于 每天写一点

地球源源不断从太阳接收能量，为地球上的生物提供能量。

这话听起来没问题吧。

可，如果从地球总体而言，如果总是吸收能量，那不就像是不断吹气的气球，早晚要爆吗？一定还有什么地方在泄气对吧。

是的，地球无时不刻在向宇宙扩散能量，以中和吸收到的能量。

那么，问题变成，既然能量是守恒的、平衡的，是什么驱使在地球上一切的运转？

熵增。

太阳传递过来的是低熵能量，地球扩散出去的是高熵能量，这中间熵的差值，是地球上一切的源动力。

那太阳的低熵来自何处？形成太阳系的原始星云。如此反复，一直向前追溯，直到宇宙最初极低的熵值。

0506 - 哲学与终结

发表于 2020-05-06 | 分类于 每天写一点

这两天看了《时间的秩序》，是一位物理学家关于时间这一理论物理命题的科普读物。

每一部分，时间的崩塌，解释了我们通常理解的时间，并不是真的时间。比如，不可细说的当下。

第二部分，解释了假设没有时间，我们又该如何认识世界。其实很简单，就像我们的祖先在认识时间之前那样认识世界：世界是由事件与关联构成的。

第三部分，解释了时间的来源。说得很抽象，我看得也很抽象。简单的说，最后上升到了哲学意识。我未理解，自然无法在些转述。

想起青泽所著的《澄明之境》，比如人为市场立法。市场是混沌的、多态的。具体坍缩是什么样，取决于观察者、取决于你的视角。你觉得该涨该跌，这一套东西，便成了市场的法。

哲学是个筐，什么都能装。所有解释不清、模凌两可的东西，都可以用哲学来解释，或者说不解释。越深入的思考，越接近本质的理解，越靠近我们自身，回流到思想的起源。

0504 - 《坏血》

发表于 2020-05-04 | 分类于 每天写一点

今天看完了这本书，感觉挺一般。

从一开始，我脑子里一直有这个问题：在都是人精的现代，这样的惊天骗局，是怎么一步步形成的？

看完书，我也没找到满意的答案。

其次的疑问，她是怎么得到巨量融资、做到天量估值。尤其，有大量钱的人，都是人精中的人精。看完书，我感觉最主要的，是大饼与吃大饼的人。

首先，她画的大饼，是很多人需要的。有钱人，在我们看来，可能衣食无忧，没什么可烦的。事实上，他们的烦恼也不小，即如何保持现在的财富、地位，以及谋求更多的财富、更高的地位。要实现这样的目标，就需要新的增长点，尤其是像 iPhone 那样划时代的产品；一般的小项目，入不了他们的眼。

反过来讲，他们特别特别害怕错过这样的机会。比如，当机构 A 想要投资该公司，在对公司的问题提出质疑时，他们特别担心，对公司不提出质疑的机构 B，得到了这次黄金投资机会；而自己，则被挤出未来的诱人的增加机会。于是，他们放弃的质疑，把钱打了过去。机构的判断力，就这样慢慢丧失了。而该公司，也越来越被纵容。

书中还有一些有意思的点。比如，投资该公司的爸爸中，有新闻大亨默多克。爆光这一骗局的，也正是旗下《华尔街日报》的记者。期间，她试图让默多克毙掉这篇报道；毕竟，这会影响他真金白银的投资。可，后者拒绝了。

让我比较佩服的一点，她聚拢的都是顶级资源，包括前国务卿、军方高层、默多克、甲骨文创始人等等。有如此能耐，如果不是骗局，真的能成大事。

0505 - 不可细说

发表于 2020-05-04 | 分类于 每天写一点

很多东西，看起来很明确，可真往细了看，一片模糊。

比如，从飞机上看，大陆和大海之间，有明显的海岸线。可真站在沙滩上，你能画出海岸线在哪吗？

比如，我们总说，活在当下，意思很明确对吧？那么请问，这个当下是什么，几点几分？哪个时区的几点几分？前后持续持久？超出一秒，还算当下吗？

比如，我说手、胳膊，你可以很清晰地在脑海中想到是什么对吧？那么请问，手和胳膊的分界点在哪？在哪条线的一边是手、另一边是胳膊？

也许你会说，你这都是闲的，没事找事型的。好，来点实际的。

比如，交易中的低买高卖、赚取差价，简洁明了对吧？那么请问，多低是低、多高是高？现在是低还是高？

无限细分，事情不是变得更清晰，反而是更模糊。适当的抽象、简化，是我们认识世界，没有办法的办法。
