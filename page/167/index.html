0226 - 连接的多样性

发表于 2020-02-26 | 分类于 每天写一点

家里停电了，想去看看邻居家有没有停电，以确定是自家线路问题，还是整体停电了。

怎么做呢？

最直接的，就是去邻居家敲门问问。现在，有了更多的办法。比如，用手机看看附近的 WiFi 热点是不是全没了。如果是，那基本可以确定是整体停电了。

无形的 WiFi，也成了人们之间无形的连接。

0224 - 权力的滥用

发表于 2020-02-24 | 分类于 每天写一点

比如，当孩子小的时候，父母在体力上占用绝对辗压优势。只要愿意，几乎可以随时、无偿地打骂小孩，这就是权力。当权力不被约束时，更是权力。

就像墨菲法则一样，只要权力有被滥用的可能，就一定会被滥用。最有效的，就是增加约束的时效性。

比如，打了孩子，孩子就不和自己亲近，将来长大后更是如此。可毕竟长大后的事太过遥远，无法对现在产生影响。爷爷奶奶辈的，则可以不断重复将来可能的事，算是提高了时效性，用将来可能的影响，来影响现在，以达到心疼小辈的目的。

说的有点乱。对于自己，就是要提醒自己，小孩的任性、不配合，确实很容易激怒耐心丧失的父母。不过，打骂，更多时候不是教育，只是发泄自己心中的怒火，只是权力的滥用。打则打矣，可别拿当教育来当挡箭牌。

0223 - 知道在赚谁的钱

发表于 2020-02-23 | 分类于 每天写一点

在交易市场里混，想赚钱，就得知道赚的是谁的钱。

如果是交易所提供的羊毛，那是容易的；甚至可以说，不赚白不赚。

如果是要赚别人的钱，就要慎重。因为大家都是来赚钱的，都是聪明人。想从别人口中抢肉吃，自然不容易。

如果确定要这么做，就得研究这些人。他们来市场的目的是什么，套保还是套利；他们的策略是什么；他们有什么优势、劣势、特征；什么要避免、什么是可以利用的；等等。

知道是赚谁的钱，才能避免稀里糊涂亏钱。
