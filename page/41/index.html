0429 - 三和一哪个大

发表于 2020-04-29 | 分类于 每天写一点

天热开始用电风扇了。家里比较老的风扇，是有 1/2/3 三个档位那种的。问题是，1 档和 3 档，哪个大？

怎么说都有道理。比如，1 排第一，所以最大；3 在数字上最大，自然风也最大。

都有道理，就是都没道理，都不够直接。不如改成大中小、强中弱，亦或是用小风扇、大风扇的图标，辅以渐粗的曲线，来得直接易懂。

0428 - 没有净土

发表于 2020-04-28 | 分类于 每天写一点

这两天在看《战胜一切市场的人》，看到一个有意思的点：

当我将精力从如何在赌场中获胜转移到分析股市时，我曾天真地认为自己将从一个舞弊百出、问题频发的世界中抽身，进入一个受规章和法律束缚、投资竞争更加公平的世界。然而我看到的真相是，更大的赌注只会吸引更狡猾的骗子。

赌场本质上是私人俱乐部、并没有义务向所有人开放；进而，有权将任意一位赌客逐出赌场，无需理由。与之相比，资本市场「公平公开」得多；可，也只是相对而言。这时依然充满着欺诈，什么财务造假、金融诈骗，多的是。

并没有净土，只能在自己可以理解、部分掌握的范围内活动。

0427 - 邻里问题很麻烦

发表于 2020-04-27 | 分类于 每天写一点

碰到一个不好相处的邻居，很麻烦。

今天跟朋友聊，他也跟我吐槽了奇葩邻居。也是采光权之类的问题，基本是我占你便宜可以、你的正常权益我也想试试的那种。

碰到这样的，软硬都不好折腾。用明面的、甚至是法律这样的办法，周期很长、成本高，且最终执行会打多少折扣也不好说。用阴的吧，又变成了比下限的操作，咱不擅长啊。

更麻烦的，还不能一走了之，还得抬头不见低头见。这种高频次、小难受累积在一起，真是要了亲命了。

碰到这样的，我是没什么好办法。在走完正常途径后，更可能的是苦练内功，降低对单点的依赖，增加选择余地，就不至于过于难受。

0426 - 思考宏观问题

发表于 2020-04-26 | 分类于 每天写一点

通常，慢慢的，我们的思维、视野会越来越狭窄，聚集自己身边的事。有时，哪怕是一颗树、买什么手机这样的小事，都会耗费大量的心神。

只把目光放在手边的小事，会让我们逐渐丧失思考宏观问题的意识和能力。

而一旦站得更高、看得更远、高屋建瓴，一些问题就不再是问题。

很遗憾，一旦我们试图站起来，懶、现实的压力等等，会立刻将我们拉回来，压在地上摩擦。慢慢的，也就习惯了，也就这样了。
