0908 - 增长的边界

发表于 2023-09-08 | 分类于 每天写一点

如果在我们身体里，有个东西一直在增长，不顾其他的增长，我们一定很恐惧，因为它叫癌。

如果在我们生活里，有个东西一直在增长，不顾其他的增长，我们一定很开心，因为它叫钱、叫经济。

前期是现实，后者是理想；如果边界条件不变的话，就是幻想。

所谓的边界条件，比如掌控更高效清洁的能源、殖民火星以获取地球外的资源、改造身体以更高效地使用能源。

0907 - 欧洲付费厕所

发表于 2023-09-07 | 分类于 每天写一点

欧洲很多公共场合，上厕所要投币，甚至你去快餐厅吃饭，用他们的厕所也要投币。这对于习惯于免费公厕的我们，有点不可思议。

反过来说，所有事情都是有成本的。我们免费用，就得有人付费去打扫。只是，我们的付费，是政府行为，一般人看不见。可政府的钱，还是我们的钱，只是政府替我们花了。投币上厕所的现象，要分清楚，是政府不作为，还是民众没有投票让政府这么做。

我当然赞成免费公厕。但也要注意，像拉美一些国家，政府动摇国本提供免费服务，讨好选民以获选票，就走了另一个极端。

这中间，就需要一个度，一个政府和民众部的平衡，考验执政能力。

0906 - 努力与幸福

发表于 2023-09-06 | 分类于 每天写一点

是努力工作带来幸福，还是幸福促进高效工作。

很多人都在这个状态：背负生活压力，被迫努力工作，以缓解生活压力，却很难走出泥潭。

少数幸运的人：生活压力很小，按自己的兴趣工作，有不错的产出，不知不觉上了一个台阶。

也许是同样优秀、同样努力的人，只是初始条件不同。

可，生命只有一次，且是自己的。与其嗟叹命运不公，不如好好梳理自己手上的牌。没有大自在，总有小确幸。总有一些事、一些方法，让当下的自己更开心一些。然后，用这种开心，面对一切。
