1007 - 干时不想，想时不干

发表于 2018-10-07 | 分类于 每天写一点

如果已经开始做了，就不要再去想这件事值不值得做、应不应该这样做，干了再说；不然，秀才造反，三年不成。

如果当前正在思考，就不要花太多时间做具体的事情。这样会打断思路，可以先记下来，具体执行时再操作。

思考和执行，很多时候思考的方式并不相同，甚至是相反的，不要互相影响了。

1006 - 占便宜 PK 不给别人添麻烦

发表于 2018-10-06 | 分类于 每天写一点

我的 MacBook Pro 屏幕涂层脱落，可以在 4 年内免费更换屏幕。正巧之前在朋友圈里看到，MacBook Pro 可以在 3 年内免费更换一次电池（且当是真的吧），于是在去苹果授权店修电脑时，很自然地想：如果真的免费，就让苹果换电池吧。

其实，这只是贪便宜的心理。实际上，电脑的电池可能性能不如之前，可并没有给我带来什么困扰或麻烦，大可不必更换。而如果更换，则实实在在是给别人带来了麻烦。甚至，如果苹果不能 100% 回收电池的材料，这还会破坏环境。

不给别人添麻烦，是种美德；这样的人，会有好运气的。

1005 - 要有主心骨

发表于 2018-10-05 | 分类于 每天写一点

当我有一件对自己很重要、自己又很有兴趣的事情做时，整个人会很精神，效率会很高。且，除了工作，也能很敏感地察觉到生活的方方面面（虽然不一定有时间真的去照顾）

而如果这样的事做完了，手上没有很急迫的事做时，整个人会变得涣散。甚至，平常生活中自己注意到、但没时间做的事，在真的有时间时，也会遗忘、或者并不想去做。

对我而言，被一件事牵着，会让整个人变得好一些。

1004 - 好像我还是不适合在手机上看书

发表于 2018-10-04 | 分类于 每天写一点

最近微信读书搞活动，送了几天无限读书卡，我就试着在手机上看书。不想，2 星期都没看完一本书…

在手机上看书，缺点比较明显：

  • 手机屏幕上，一次看的内容少，需要频繁翻页，会影响思维的连续性。
  • 手机给人的心理暗示就是个零碎的东西，很难保持长时间的专注，很容易看着看着思维就跳出来了。
  • 缺少仪式感。比如翻开纸质书，气场就切换到读书的状态了，比较容易沉浸其中。而手机则不同，很难跟「玩手机」区分开来。

当然，上面说的听起来比较矫情，但读书有时候差的就是那点 feeling…

自然，手机上看书也是有好处的，最大的自然是可以随时读，不用携带额外的设备。读书社交这种东西，我是不感冒的。读书是私人的事，并不想跟别人掺和在一起。让别人知道我在读什么书、在哪做了标注，就像是给别人看我穿了什么衣服、内裤是什么颜色一样。

实际经验来看，还是纸质书看的最快、效果最好，比较适合好书、需要精读的书。一般的，目前还是用 Kindle PaperWhite 吧。如果苹果出了 iPad mini 全面屏版，可以考虑入个用来看书。
