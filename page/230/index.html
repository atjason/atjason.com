0912 - 接纳当下，赢得未来

发表于 2020-09-12 | 分类于 每天写一点

比如，在华为被美国刻意打压的当下，华为发布鸿蒙 2.0。自然，这个系统并不成熟。必然，引来一些讽刺挖苦，比如只是套壳的 Android、魔改的安卓。

对，这些说的都对，但有什么用呢？有本事，你来一个新的、纯粹的创新，又能让所有的生态位都来陪你玩。抱歉，批评你会、段子你强，可建设的能力你没有。

当下不完美，可这是我们唯一能拥有的。接纳当下，面向未来努力。

悲观主义者猜对当下，乐观主义者赢得未来。

0911 - 追求长期价值

发表于 2020-09-11 | 分类于 每天写一点

比如，口腹之欲，即便被满足，也只能持续很短时间。更别说，吃的喝的多了，还会身材走样、肥胖。

相反，健身、锻炼、早睡早起这些习惯，短期都是有点痛苦的，很可能让位于当时的贪图享乐。可长期看，其带来的回报更持久。

做交易也是。不用太过在意一池一地的得失，而是要建立能长期存在的交易模式，持续的盈利。无论从标的选择、交易策略等方面，均是如此。

比如，有人买彩票赚钱，有人炒空气币赚翻；看过，笑一笑，也就过去了。这些，都是不可重复、不能持久的，属于口腹之欲，切不可因此乱了追求长期价值的心态和阵脚。

0910 - 教师节快乐

发表于 2020-09-10 | 分类于 每天写一点

教师、老师，首先是种职业、一份工作。和所有的工作一样，在工作中体验自己的价值，并取得相应的报酬。

而老师，又不是普通的工作。孩子，是一个家庭最大的希望。而家长由于时间、能力等方面的限制，不能在教育孩子上亲力亲为、面面俱到，只能把这种希望转嫁到老师身上。

一份普通的工作，莫大的期望，这二者明显是不匹配、不平衡的。很明显，老师达到了这种期望，并不能取得和期望相匹配的回报，自然无法产生足够的动力。说的极端点，孩子在家长眼里是未来的龙凤，在老师眼里可能只是车床上的工件。

怎么办呢？家长也没什么好办法。所能想到的，就是在其他方面找补回来，比如送礼、请客吃饭游玩、各种明面上恭维老师、等等。具体的表现就很多了，尤其比如家长群里，家长的各种舔；老师随便做的点什么、哪怕什么都不做，也会得到家长的各种赞扬、花式恭维。

这显然是很畸形的。很直接的，一旦孩子毕业，家长很可能和老师再无往来。老师见多了，也麻木了；可你如果不恭维，那也是不行的。

怎么办呢？我不知道，我也会在群里恭维老师…

0909 - 先知、后知、不知

发表于 2020-09-09 | 分类于 每天写一点

《交易冠军》里讲了一个「三日定律」，说的是不要在连续上涨的第三天做多、不要在连续下跌的第三天做空。

为什么这么说呢？因为：

  • 第一天的市场变化，是提前知道消息的玩家发动的，是为「先知」
  • 第二天的市场变化，是后知后觉的操盘手发动的，是为「后知」
  • 第三天，连那些原本不知不觉的散户也都进场时，就危险了，是为「无知」

可能不一定准确，也可能不是严格的三天。不过，这种扩散理论还挺有意思的。

其实，很多其他领域也是类似的。比如时尚圈，一个新配色的流行，可能最开始是在巴黎时装周上展示的，几个月后开始出现在杂志、明星身上，再几个月出现在快时尚服装上，最后出现在淘宝上、地摊上。然后，这个配色就​变得土了。

0908 - 大量复盘

发表于 2020-09-08 | 分类于 每天写一点

交易的必经之路：大量复盘。

核心的一点，复盘加快进化速度。一年的 K 线，你用一个小时复盘，就相当于你用一个小时过完一年。这速度，相当于加速了 8760 倍。长此以往，你怎能不加速进化。

还有一点，相比于在实盘中积累经验，复盘时更纯粹、不被打扰。交易时，由于是真金白银在操作，心理压力大，一点风吹草动就可能影响判断。复盘则没有这种打扰，安安静静的看，安安静静的思考。复盘时总结出来的规律，在应用到实际交易时，会更有信心、更容易坚持。

当然，实战中也能积累经验。但更可能的，是在复盘中摸索出规律，在实战中应用，并根据实际情况进行调整、优化、丰富细节、增加对各种情况的应对。实战更多的是执行，而思考要在盘前进行。复盘，是思考的一种很好的操练方式。
