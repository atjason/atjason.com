0711 - 不能太舒适

发表于 2019-07-11 | 分类于 每天写一点

人有一种能力，不能在什么环境下，都能构建一个小小的茧；躲在里面，很安全，很舒适。

这种能力很重要；不然在恶劣的进化过程中，人不被搞死，也会痛苦死。

只是，这种能力，就像脂肪一样，有点过时，有点显得过于强大。

我们既不会被野兽吃掉，也不会吃不饱肚子，不用一直躲在舒适区中，而要不断走出去，在难受痛苦的过程中，增加能力，拓展边界。

0710 - 透明的力量

发表于 2019-07-10 | 分类于 每天写一点

比如做交易，策略是核心资源，把它捂的严严实实，最有利于价值最大化。

是这样吗？不一定。

比如，分享的过程，会迫使自己系统地梳理思路，然后用简洁的方式组织，以便将信息传递出去。这样有目标的倒逼过程，可以有效地发现之前的思维盲区，找到潜在的问题。有时，还能意外地找到长期郁结问题的解决之道。

比如，展示自己是交流的前提。而交流不仅仅是互通有无，很重要的一点是他励。别人随口的一句话，有可能给自己带来全新的思路。

比如，暴露会带来机会。钱在找赚钱的方法，赚钱的方法也在找钱。夜晚的灯塔，固然会有飞蛾的烦恼，却也会带来聚拢的船只。一旦成为联结的节点，价值也会几何放大。

0709 - 长期愉悦

发表于 2019-07-09 | 分类于 每天写一点

比如，大热天的来瓶冰可乐，爽！可，这样的爽持续不了多久。且，也不可能通过持续喝可乐得到持续的爽；不仅肚子装不下，这种爽的效用也是递减的。

这就是短期愉悦；其门槛很低，很容易得到，但持续时间也短。

与之相对的，是长期愉悦。比如，花时间精力掌握一种工具。之后每次遇到相同的问题，祭出这一工具，迅速解决问题，也会带来爽快感。这种爽是长期的，每解决一次问题，就会体验一次。效用还会累积，成为高效和自信。

慢慢的，把时间精力更多的花在能带来长期愉悦的事。

0708 - 指责埋怨，慎之慎之

指责埋怨，自己会有黑色成就感。至少是把负面情绪、失败的原因转移到对方身上，自己感觉就会轻松些，负罪感小些。

可，于事无补。

张口指责、就要埋怨之前，把话咽回去；尤其是对家人。

自己如果已经把事情做好，而来对方没把事做好之说？
