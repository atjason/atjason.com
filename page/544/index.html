0818 - 正确做事的感觉

发表于 2016-08-18 | 分类于 每天写一点

  • 拥抱新技术，不死守旧技术、尤其是本身有缺陷的技术
  • 适度优化，但不过度优化
  • 尽量减少维护成本，最好不需要维护
  • 保持精简，杜绝臃肿
  • 每写一个功能，就完善其单元测试
  • 不做让少数人爽、而让多数人不爽的事
  • …

这些都是正确做事的方法。保持这种感觉很好、很重要。事情做对了，只要做就可以了，其他都不用考虑。

0816 - 万事开头难，开头就不难

发表于 2016-08-16 | 分类于 每天写一点

据说，拖延症的一个原因是，因为怕事情太难、或做不好，而不愿意开始。

这时候，解决的办法是，立即开始。别管什么效率、别管什么方法、别管能不能做完，随手抓起一件具体的事，做就可以了。

做着做着，自己就会慢慢有感觉，知道接下来要做什么、知道先做什么、知道怎么做。

另外，当状态不好的时候，换个环境，看些打鸡血的书（要持续看，至少看半小时），很有效。

0814 - 少出错

发表于 2016-08-14 | 分类于 每天写一点

今天看了奥运会乒乓球男单决赛，很明显的感觉是，张继科输在失误上。当然，说是失误，也可能是能力上的问题。不过，也确实是有不太应该的失误。

做事、做产品也是一样，机会总是有的，关键是能不能抓住。错过一些机会也不是太大的的问题，但不能一直错过，尤其是到手的机会一定要转化成实际的东西。

努力做好眼前的事，也才有资格考虑更长远的事。
