0713 - 剥削

发表于 2020-07-13 | 分类于 每天写一点

有没有觉得，「剥削」这个词，变得有些陌生。

是的，不论是电视还是公众号，不论是中国还是美国，已经极少有这样的表达。取而代之的，更多的是竞争。

真是这样吗？剥削已经被现代文明彻底消灭了吗？自然不是。

用绝对实力、以历史积累的优势赢得所谓竞争，就可以说是剥削。

比如，早些看课本里常提到的，发达国家与发展中国家之间的剪刀差：中国出口无数双袜子，以换取一架飞机。这很难说是贸易，更像是剥削。

比如，贫富分化引发的各种问题。极少数富人拥有大量的财富（尤其是仅通过继承得到的财富），提供就业机会给穷人，后者拿相对于富人财富增值而言微不足道的工资。这，是不是剥削呢。

比如，阶级固化，就好像纯粹是一个人不够努力的结果。只要睁开眼睛看看世界，就知道巨大的机会不平等之下，努力能提供的加分项，实在太少太少。

剥削，不是没有了，只是被掌握话语权的人隐藏了。

0712 - 大量小仓位练习

发表于 2020-07-12 | 分类于 每天写一点

看交易书籍、学交易视频时，哇，感觉交易好容易。结果一上手，各种骚操作、一通亏。

怎么办呢？无他，大量小仓位练习。

练习，真实的交易，才能接触到交易中最真实、最细微的地方，才能体会理论应用到实际，会遇到怎样的变数和困难。

大量练习，才能遇到更多的情况；下次再遇到时，才会有熟悉感和自信；才能变得麻木，尤其是在止损时，像机器一样坚决遵守交易纪律。

小仓位，是为了避免爆仓，有继续练习下去的资本。大仓位一旦造成大亏损，很容易影响心态，甚至打击继续交易的勇气。

反过来，既想练习、又想赚大钱，是矛盾的。

0711 - 《谁是美国人》

发表于 2020-07-11 | 分类于 每天写一点

扫一下这本书，有个有意思的点，关于身份认同。美国有点 U 型的意味：

  • 在高层面，随着美国的全球扩张，美国人认为自己有全球责任，诸如世界警察。
  • 在低层面，又有种族、宗教、职业之分。

而在中间的国民身份，却好像没那么强。说的形象点，一个在跨国公司工作的拉美裔移民美国人，其「跨国」和「拉美裔」的属性，要明显强于「美国人」。

比较而言，在中国，你不太会问「你是什么人」这样的问题，感觉很奇怪对吧？因为中国的国民认同比较强，或者说是强于宗教、区域、职业。在美国却并不奇怪，可能得到的答案包括「我是牙医」「我是犹太人」等等。

一个具体的现象，9.11 之前，美国街头主动挂国旗的不多。9.11 之后，暴涨。而现在，再次大跌。说明在没有外界刺激时，国民身份认同，会急剧下跌。

有个原因，美国两党选举，会有意的分化群体，诸如黑人、同性恋者等等。然后针对性的讨好这些群体，以获得其好感与选票。这自然也会加剧群体分化，减弱共同认知感。

0710 - 不要因为忙而不学习

发表于 2020-07-10 | 分类于 每天写一点

如果你不学习的理由是太忙，那很可能你一定比例是在瞎忙。

说到我自己。如果我的理由是，最近的机会太好了，我要抓住机会多交易，趁这波多赚点。那很可能，我已经过度交易了。不仅抓不住机会，反而可能亏钱。

抽象来说，我们是在用旧知识解决新问题，也可以说是凭经验做事。但明显，旧知识或者说经验是会过时的、有缺陷的。只有不断打磨完善，才能更好的解决新问题。

另外就是心态。能静下心来学习，说明本身心态就不错。而学习的过程，会让心态更好。就像是，知道自己不知道，于是去学习；学习之后，能力圈扩大，接触到更多未知的东西，才发现有那么多自己不知道的东西。在这样的情况下，怎会不谦虚谨慎。

而谦虚谨慎，对交易来说太重要了。

0709 - 不要着急进场

发表于 2020-07-09 | 分类于 每天写一点

做交易总是会遇到这样的情况：听别人说、或者自己感觉发现了机会，机不可失，再不进去就来不及了。于是，立即开仓。然后被套，由小亏变大亏。

经验来看，上赶的不是买卖。一旦你着急，就落了下风。

一旦你也陷入了这种状态，离开交易界面，自言自语几遍：慢就是快；以后机会多的是。

更靠谱的状态是：就像猎人狩猎一样，事先找到猎物可能出现的区域，然后耐心等待。发现猎物后，不是立即开枪，而是先锁定。直到最佳射击机会出现，一击毙命。
