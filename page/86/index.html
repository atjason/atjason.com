0103 - 困难还在，我们不同

发表于 2023-01-03 | 分类于 每天写一点

今年遇到的困难，明年还会遇到。比如，供应商的不配合、竞品变得更强、用户变得更挑剔。

甚至，今年的困难，会继续拖到明年，使得我们并不能轻装前行。

但是，我们变得不一样了。一些困难我们见过了，再见时，心里就不慌了。甚至，一些困难我们解决过，再遇到时，心里就有底了。

我们怀揣着对未来美好的想象，就是做出更好的产品、卖更多的钱，我们一点点展开想象，用一件件事、一个个动作，逐渐描绘想象的样子，最终把事情做成、让想象成真。

0102 - 给明天一个容易的开头

发表于 2023-01-02 | 分类于 每天写一点

作家海明威有个习惯：在一天写作结束、要写最后一句话时，这句话不写完，而是只写前半句。第二天一开始写作时，面对的是这半句话。接这半句话很容易对吧。等这半句话写完，自己的写作灵感就被激发出来，就刷刷的开始写了。

这个小技巧我们也可以用。一天结束时，确定一件明天最想做的事。等第二天一开始，面对的就是自己最想做的事。等这件事做完，自己做事的冲动就被激发出来了，刷刷开始做接下来的事。

1227 - 自制电路测试锂电池放电

发表于 2022-12-27 | 分类于 每天写一点

现在确定锂电池供货商，从哪家采购呢？相同的标称容量，价格差一倍，不知道该选哪个。

如果相信样品能代表大货的真实水平，那也需要在不同样品中，确定用哪个，这就需要测试电池的真实电量。这就需要用到负载机，可是负载机要几千块。单纯为了这个目的而共这个钱，不划算。

于是，自制了一个小电路和配套固件，在电路供电过程中，每 30s 采集一次电池电压，并记录在芯片内置存储中；同时，使用小的 OLED 屏幕，显示已测试时长了当前电池电压。

等电池完全耗尽、电路无法工作时，再通过串口，读取芯片内置存储中记录的电压曲线，绘制成表格。

从图表中可以看出，大概在 12 时 22 分后，电池电压下降至 3.3V。之后，电池电压迅速下降，真正低于 2.7V 保护电压，电池停止供电。

1226 - iPic 默认上传图片至 Backblaze

发表于 2022-12-26 | 分类于 每天写一点

微博图床，之前上传的图片，现在开始出现403无法访问的情况。哎，免费的最贵，要开始找替代了。

为了让产品能持续，iPic 提交了新版，默认上传至 Backblaze。

目前还不确定每年会产生多少费用。仅域名每年约了90 元，Backblaze 的存储费用，就要看实际的使用情况。如果按 1TB 计算，每年约 10240.00512=$62，不算太吓人。

流量费暂不用太考虑，因为都走 Cloudflare。

还有个风险，就是一旦图片被用于黄赌毒，域名会有被封的风险，这样其他用户上传的图片，也变得无法访问了。
