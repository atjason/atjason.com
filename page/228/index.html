0122 - 应对无法掌控的事

发表于 2017-01-21 | 分类于 每天写一点

感觉，自己的焦虑多数来自于自己无法做的事，或者是自己无法掌控的事。

坏消息是，大多数事自己都做不了、或无法掌控；
好消息是，这样的事，做一件、少一件。

虽无法掌控，但毕竟还是可以影响的。进一步，说是事，但其实还是做事的人。这样，问题就演化为如何驱动一个人朝着自己期望的方向做事？

因势利导。

不同的人，能驱动他的东西是不一样的，比如：

  • 权利。比如事业单位体系中的小职员，最能驱动他的就是领导的一句话，然后就是纪律、督查。
  • 钱。这种没什么的好说的；如果觉得跟解决的问题相比，花的钱并不算多，那就用钱砸呗。
  • 成就感、虚荣心。一个褒义、一个贬义，但实际的意思差不多的。比如对方就是助人为乐、乐善好施的主。

说是能说很多，但其实我在这方面很不擅长，也并不想做这样的事，比如管理、驱动手做事。与其驱动一个人做事，不如找个不需要驱动的人合作。

0119 - 服务好存量用户

发表于 2017-01-19 | 分类于 每天写一点

获取新用户，难

这一现象有很多的原因，比如，分散用户时间的产品太多了，将产品触达用户面前太难了，能在第一时间打动用户不容易，等等。

尤其是，人口红利的消失。之前，哪怕你的产品没有任何变化，每年新的手机、Mac 电脑用户，都有可能成为你的新用户。而现在，这些新用户的增加停止了，分到自己头上的红利自然也消失了。

存量用户，值得珍惜

经过前面讲的层层困难，最终成为你的用户，已经是非常宝贵的。如果让这些已有用户流失，那实在是太可惜了。

要服务好，可以有很多方面；比如，

  • 改进产品本身的体验；
  • 增加与用户的直接沟通，让他们觉得自己使用的产品是有温度的、有故事的；
  • 增加用户的参与感。
    • 比如用户提了新需求，新版本满足这一新需求后，及时通知用户。
    • 注意，最好是一对一的沟通，这样更有效。虽然这样做成本很高，但得到的用户也更忠诚。

不能做朋友的用户，放弃

比如，单纯因为价格而给差评的用户，我想说：请走开。说的难听点，他不值得拥有我辛苦做出的、好的产品；说的实在点，他的消费能力还没有上来（恩，就是没钱）。毕竟，明码标价、童叟无欺，如果这都要骂，那请移步奢侈品牌，够他骂个够。

其实，也并非单纯因为他们消费能力差而放弃，而是他们缺乏对他人的包容、理解、支持、尊重。与其把时间花在这样的人身上，不如服务好那些值得服务的用户。

我相信，靠自己努力吃饭的手艺人，值得拥有体面的收入。

0118 - iKindle 要出来见人啦

发表于 2017-01-18 | 分类于 每天写一点

历时 18 天、105.6 小时，iKindle 终于要出来见人啦！

其实，本来只是想随意整整，没想到做着做着觉得还挺有意思，就越来越想把这个小工具做好了。

为什么我知道 105.6 小时这么「精确」的时间？因为我有记录时间的习惯啦，这样就可以对自己在各个项目上投入的时间更清楚。对于自由职业者，时间就是最宝贵的资源，当然要好好把握了。

其实，以 iKindle 目前的规模，13 个工作日的开发时间确实长了些。换个角度看，真正在设计、开发 iKindle 本身业务的时间，可能只有一半的时间。更多的时间是在学习，比如 UI 开发、SQLite 使用、等等。这次开年的小项目，也算是为未来可能的「中程序」练练手。

最后，还比较满意这次设计的 Logo，虽然又是简陋的文字版，谁让咱就这水平呢，也算是和 i 家族的小工具们相响应吧…

0117 - 明确目标、有效行动

发表于 2017-01-17 | 分类于 每天写一点

都说浓缩是精华，那这句一定是精华中的精华：

明确目标、有效行动

不管是协作、还是竞争，不管是与人相处、还是自我提高，都离不开这句话。如果展开来说，就感觉像是拿沙子把珍珠盖住，反而淹没了重点。

另，感觉状态有「大小天」的切换：昨天效率还行，干到凌晨 2 点；结果，今天愣是上午 11 点才起床，今天的时间立马少了、变成「小天」了。
