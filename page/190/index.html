0827 - 做与不做

发表于 2021-08-27 | 分类于 每天写一点

还是关于孩子的教育。

马上开学，一些学生面临是否要住校的选择。这时，家长就纠结了。

事实上，尤其一些处于青春期的孩子，家长说什么，孩子已经不怎么听了。再加上功课越来越难，家长已经辅导不了作业了。

在这样的情况下，家长对孩子的教育作用，已经越来越小。可即使是这样，家长还是想对孩子做点什么，也才安心。如果送到学校，一个星期回来一次，心里总是不放心。万一孩子养成坏习惯怎么办？哪怕真有坏习惯，自己事实上也管不了，但毕竟在自己身边，还是安心一些。

我现在感觉，教育这种事情，越早，效用越好；越晚，孩子定型了，越难改变。因此，有可能的话，还是在孩子小的时候，多花点时间精力。长期来看，这样性价比最高。

0825 - 交易中的一致性

发表于 2021-08-25 | 分类于 每天写一点

交易策略想通了，就要开始长期实盘实践。这里有一个重要的问题：要保持动作的一致性。

这个问题有个前提：市场的随机激励。

也就是说，相同的操作逻辑，有时结果是好、有时是坏。越是在这样的前提下，越要保持自己动作的一致。如果市场是动态的，自己的操作也是动态的，那完全是乱来，很难从操作中总结出什么规律。

另一点就是长期。如果只实践一两周，亏了就没信心了，就要换方法，这是很要命的。可是，反过来讲，在持续亏损的情况下，继续坚持的动力来自哪里呢？

一方面，是对策略的深刻理解和认同。从逻辑上相信，策略是正确的、概率占优的。

另一方面，就是大量复盘。通过复盘，感知策略的优缺点、胜率。如果结果是正确的，那就自然更有信心。

当然，无论如何，实盘中的盈利，是坚持下去最好的动力。

0823 - 家庭财产规划师

发表于 2021-08-23 | 分类于 每天写一点

最近国家层面在大力推进共同富裕。从结果上看，核心目标是增加中产阶级数量；从金字塔分布，向橄榄球分布转换。

从趋势上看，中产阶级会越来越多。这些刚刚迈入中产的人，会面临很多之前无产时没遇到、不会处理的问题。比如，财产如何保值增值。

这一领域，专业性很强，不是自己随便瞎操作就也可以，也不是几年前闭着眼睛买房这么简单。同时，随着国内金融资产证券化越来越深入，未来理财产品，会比保险产品一样，品类繁多，让人无从选择。

于是，就需要有专业人士进行指导，对家庭财产进行更合理的规划。这一需求如此之大，大到会催生出一个产业。说到这如果你感兴趣的话，不妨朝这个方向努力下。
