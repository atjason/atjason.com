1226 - 臣服市场

发表于 2020-12-26 | 分类于 每天写一点

开仓，然后双手合十，期待价格朝自己想要的方向运动，这不是交易，这是赌博。

市场永远是对的，不和 K 线争辩。

在交易中，不要把自己当成聪明人。不要想着，根据自己的经验和判断，当前的价格走势会是怎样的。还是笨一点好，只有当市场走势明朗时，再跟着喝口汤；而不是预测价格即将反转，然后从头吃到尾。

市场说啥就是啥。市场愿意赏口饭吃，那最好；如果市场不给，认赔止损。

1225 - 《血疫：埃博拉的故事》

发表于 2020-12-25 | 分类于 每天写一点

在新冠疫情的当下，看这本书，算是很应景的。

快速看完书，印象比较深的有几点。

一是被病毒攻击过的生命体（人、或灵长类的猴子等），身体内的血肉器官，会变成血水、液体；进而，会从七窍、肛门等地方喷出。比如肠子，当生命体倒下时，如果当时剥开肚子，发发现里面的肠子，就好像已经坏死了好多天一样，像一团浆糊。

是不是觉得很恶心反胃？是的，我当时也是这种感觉。

还有个印象深的，病毒是种「非生非死」的状态。在脱离生命体时，它们就像是「死」了一样，没有任何生命一样。而一旦找到了宿主，立马又有了「生」的迹象。你很难说，它们到底死了没有。

再有，病毒对人类而言，是如此的小，小到我们其实并不能真正的消灭它们。怎么才算消灭呢？你用什么方法才能消灭病毒曾经出现过的地方呢？很难很难。况且，为了研究等目的，我们其实会保留病毒样本。而，如何既能保证这些病毒不会永久死掉，又保证它们不会死灰复燃？

在面对新冠这样的疫情，想想人类还有各种推诿，国与国之间还有各种指责，说病毒是谁谁谁产生的的，真是幼稚可笑。就算是 A 国产生的，B 国指责 A 国有什么意义吗？除了短期政治上的可笑利益。

再说，「幸灾乐祸」也是很可笑的想法，你比对方多活几天，有意义吗？可能只是将恐惧延续几天而已，「人类共同体」了解一下。

1224 - 知行合一

发表于 2020-12-24 | 分类于 每天写一点

最近做交易，很强烈的有这种感觉：看起来都会了，一上手各种乱来。说了每单要止损，可还是会亏损抗单。说白了，还是做不到知行合一。

这种状态下看到这段话，感觉很认同：

阅读各种资料、书籍、文章，只是让你懂得更多，并不会让你提高技能。只有通过练习（practice）才能提高技能。如果你想变得更好，那就去动手做。

– Jason Fried（Basecamp 的创始人）

1223 - 区块链技术实体化

发表于 2020-12-23 | 分类于 每天写一点

不要怪我偷懒不写文章，看到这个我觉得太有意思了，就搬过来了。

小区大门敞开，容易让社会车辆乱入，影响业主们自己停车。小区的业主想出了妙招，在小区大门上串联了66把锁头，被网友赞为“最便宜的门禁系统”。

启用该功能后，小区不再像以前乱七八糟。居民表示，这种方式省钱省力，特别好用。
网友看后表示：这就是区块链技术实体化了。具体特点有：去中心化（不需要统一管理）、可追溯性（一人一锁谁没锁找谁）、不可篡改性。
