0719 - 回调验证心态

发表于 2021-07-19 | 分类于 每天写一点

交易中的回调，非常能检验交易的能力和心态。

首先，回调是不是在自己的计划、预期范围内。如果自己的计划完全没考虑到回调、对回调没有包容性，显然这样的计划是有问题的。

然后，如果计划中有回调的部分。可当回调真的出现时，自己又接受不了。心情开始变差，甚至开始胡乱交易，那计划还有什么意义。

当然，最好是小回调、小亏损。如果还做不到这一点、或者亏损时心态不好，那就先轻仓操作；至少不会大亏，不至于乱来。

0718 - 交易，简洁有力

发表于 2021-07-18 | 分类于 每天写一点

交易，两个极端更容易见效。

一种是简洁，考虑的因素不多，规则清晰明确，很容易执行。这种，适合个人使用。

一种是复杂，考虑尽可能多的因素，模拟现实金融环境，以期最真实的理解现实，做出最佳决策。这种，明显适合大机构使用。

大部分人，其实两边不靠。太简洁的东西，自己不信任，不相信这么简单的东西就能赚钱。于是，慢慢的加各种东西。遇到一种行情，规则一说不做，规则二说做。自己主观想做，就倾向于规则二。规则越多，其实和没有规则差不多。

明知做不到复杂，就往简单了做吧。

0717 - 谁在规划一座县城的发展

发表于 2021-07-17 | 分类于 每天写一点

我不知道答案，只是好奇。

一座城市，该往哪个方向发展，有限的资源应该如何在不同产业间平衡，哪里该修条路，建多少学校，等等。大大小小的问题，其决策是怎么做出来的呢？

其中很明显的一点，领导的决定权相当大。虽说是民主，但我们的官场我们懂，基本还是一号的一句话。

可是，官场又分两种，一种是轮换制的，也就是一把手，干几年就走。另一种是扎根在本地，会干很久、甚至是一辈子的。在古代，就是「官」和「吏」的区别。

再有，就是乡绅富豪。财政靠钱，钱靠税收，税收靠大企业，大企业主必定是乡绅富豪。权和钱，被紧紧的拉在一起。富豪是缺少大局观的，通常只管自己的一亩三分地；哪怕这地非常大，也只是局部思维。

看起来有点悲观，不论是一号还是富豪，没有人从本质上有意愿对大局进行长久的思虑。真正在大局改善中受益的大多数，其实并没有太大的发言权。

0716 - 趁着还不老

发表于 2021-07-16 | 分类于 每天写一点

回到老家，看到了很多老人。一年年的，明显感到他们老了。今年这位走不动了，明年那位眼睛看不清了。

说实话，给我带来深深的恐惧：等我变这么老了，活着和死去，有什么区别呢？

在城市里，周围都是朝气蓬勃的年轻人，大家都充满干劲，城市也一天一个样。

而到了农村，生活节奏一下子慢下来，夏天乘凉、冬天晒大阳的老人，多了很多，也让人看到很多生命阴暗的部分。

趁着自己还不老，好好活。

0715 - 穷人的教育方式

发表于 2021-07-15 | 分类于 每天写一点

父母这一辈，都是穷过来的。他们这一代，基本没有教育的困扰。换个角度说，方式简单粗暴，却非常有效：

不听话？晚上别吃饭。
不好好学习？退学回家种地。

现在人们的生活水平提高了，不再像以前那样受穷了。这自然是好事，却也带来了教育上的困境：他们从他们父母身上学到的教育方式，无法再应用到他们的子女身上。或者说，之前的方法都失效了。

不让我吃晚饭？太好了，出去买好吃的。
让我回家种地？对不起，家里已经没地了。
不好学习将来没前途？要前途干嘛，躺平不香吗。
不让我玩手机？那你别玩啊。
让我运动健身？那你咋不动呢。

于是，父母懵了。并没有人教他们，怎么用新的方法，来教育新的一代。没人教、自己又不擅长，自然带来了很多教育上的问题。

不穷了，却不知道穷之外的教育方式，就是这一代父母的教育困境。
