1031 - 新见 10w+ 诞生

发表于 2017-10-31 | 分类于 每天写一点

这几天有件事，在麦当劳中国公司改名「金拱门」后，「三胖」迅速买入卖出 jingongmen.com 域名，赚了 15000 元；同时，在自己的公众号「成长」中，出了篇 因为麦当劳改名金拱门，1个小时我赚了15000，迅速获得 15w+ 的点击，粉丝增长无数。

亲见 10w+ 诞生。

今天，亲见了三胖本人，聊了一天，收获良多。比如：

  • 花开别处：对于这次炒作的域名本身，三胖是亏的，因为本可以赚更多却出手太早。可是，之后的 10w+ 的文章，却为自己带来收益无数
  • 广积人脉：不知道在什么时候、什么地方就能帮到自己
  • 以和为贵：对于自己不认同的人，不会主动批评、树敌，敬而远之也就可以了
  • 加大杠杆：逐级放大自己的影响力
  • 边做边说：这年代，埋头苦干已经不行了，或者说已经不够了，自己还得会营销产品、营销自己
  • 累积背书：每个人在认知别的人、物时，都需要一个锚定。进而，每个人都需要不断增强自己的锚定
    • 比如，别人在把你介绍给他人时，说的前几句话，基本就是你的锚定

晚上，还见了位在 巴比特 做运营的朋友，其中他的一句话刺激到了我：感觉现在互联网对年轻人已经没什么机会了，所以我选择了币圈。

确实，互联网到目前已经充分发展，对于新入行的人，感觉已经像是传统行业了。要在互联网这个新兴的传统行业中打出一片天地，变得越来越难。同时，也更加凸现选择的重要，选择一个本身就在快速成长的行业，自己哪怕静止不动，也会坐地日行八百里。

自己收获的，才是收获。别人分享的，带给自己的最多是思路；实操还是得自己。接下来，准备：

  • 试水「微信读书笔记分享」，感受下蹭热门应用是什么感觉
  • 左手开发、右手运营，看看能把 Klib 的影响力和收入做到什么程度

今天在看接下来 Klib 准备做什么，发现有个好玩的点：管理微信读书笔记。

微信向来是很封闭的，目测并没有开放接口可以拿到此数据。大致搜了下，网上也有一点点做微信 API 逆向的，但明显会有大量工作，且很不稳定。

不过，倒是引出了另外一点需求：共享微信读书笔记。目前，微信读书里仅支持分享单条笔记，分享一本书的笔记？不可能。但有没有这个需求呢？大致问了下，不强，但还是有。准备先折腾下这个事情，看看效果再说。

1029 - 小城生活

发表于 2017-10-29 | 分类于 每天写一点

今天去了趟嘉兴，去朋友家蹭饭，也顺便感受了把小城生活。

挺爽的。

当然，嘉兴也并不算无名小城，只是相对于一线城市而言。感觉还是挺舒服的，比如：

  • 房价物价相对不高，生活压力骤降
  • 基础设施不差，比如高铁直达
  • 人车密度相对低，不会觉得很拥挤
  • 生活节奏慢些，既不会因此失去什么，却赢利了悠闲惬意

对我而言，并不太喜欢非常快的生活，也不太愿意为了城市的福利而牺牲很多，其实还挺想真的试试小城生活。

不过，既然目前还无法离开杭州这座相对更快、资源更集中的城市，那就充分利用这里的优势吧。如果总是被城市的负面包围、无法享受其便利，那就太亏了。
