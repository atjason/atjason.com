Shell 字符串

发表于 2016-08-13 | 分类于 Shell

字符串是shell编程中最常用最有用的数据类型。字符串可以用单引号，也可以用双引号，也可以不用引号；区别如下。

单引号

单引号字符串的限制：

  • 单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的；
  • 单引号字串中不能出现单引号（对单引号使用转义符后也不行）。

双引号

双引号的优点：

  • 双引号里可以有变量
  • 双引号里可以出现转义字符
